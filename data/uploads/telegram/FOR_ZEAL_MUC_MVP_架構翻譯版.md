## 🏗️ 系統架構（12 核心模組）

```
[使用者前台 Portal]
   ├─ 帳戶登入/註冊
   ├─ 訂閱管理（查看續期狀態、倒數提醒）
   ├─ 設備綁定（QR Code 配對）
   ├─ 自助結帳（選擇方案 → 確認付款）
   ├─ 點數額度（2026-03-25 新增）
   └─ 通知中心

[後台 Admin]
   ├─ 序號管理（建立 / 啟用 / 停用 / 作廢）
   ├─ 帳號管理（客戶資料 / 權限 / 啟用狀態）
   ├─ 設備管理（客戶設備清單 / 強制停用）
   ├─ 付款紀錄（手動確認收款 / 對帳）
   ├─ 訂閱分析（進度條 / 收入統計）
   ├─ 告警中心（異常監控 / 風控）
   └─ 稽核紀錄（所有操作都有日誌）

[客戶設備 Agent]
   ├─ 配對申請（輸入配對碼）
   ├─ 授權查詢（5 分鐘輪詢一次）
   ├─ 接單前檢查（確認還有額度）
   └─ 離線保護（超過 15 分鐘停止接新工作）
```

## ⚠️ 注意事項

### 已內建的安全基線
- 序號不能直接當授權憑證（需配對碼）
- Device Token 高熵亂數 + 只存 hash
- Entitlement 簽章驗證（Agent 本地驗證）
- 15 分鐘離線自動停止接新工作
- 所有操作完整稽核

### 明確不含 MVP 範疇
- 自動金流串接（第一階段只支援人工對帳）
- 自動退款流程
- 推薦分潤
- 國際稅制
