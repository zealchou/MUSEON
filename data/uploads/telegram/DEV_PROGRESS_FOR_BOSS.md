## 當前卡點 & 風險

### 🚧 Agent 正式對接測試（進行中）
- **情況**：Feng 完成了 SDK，現在等著跟「客戶的 Agent 開發者」聯調
- **風險**：如果對接有問題，可能要修改 API
- **建議**：趕緊聯繫客戶開發團隊，安排聯調時間

### ⬜ 通知紀錄查看（尚未完成）
- **情況**：Admin 看不到過去寄過哪些通知郵件
- **影響**：營運時查帳有點麻煩
- **工作量**：小（就加一頁面）
