Design Philosophy

Core Proposition: Warmth with Precision

MUSEON is neither a cold tech tool nor a frivolous SaaS dashboard. It is a precision instrument by the fireplace — warm enough that you want to draw near, precise enough that you trust its judgment.

References: Apple HIG (clarity / whitespace / hierarchy), Stripe (B2B info density), Notion (warm neutrals), Linear (dark premium), Figma (component systems)

Imagery Direction

Adopt

Warm-toned light and shadow (amber / terracotta / deep indigo), abstract neural lattice patterns, dark gradients with noise texture, fine-line architecture diagrams.

Forbidden

  • Blue-purple tech gradients
  • 3D robot renders
  • Business handshake stock photos
  • Meaningless dashboard screenshot collages

Pre-Flight Checklist

  • Background: Parchment #F7F5F0, never pure white
  • Headlines: Cormorant Garamond (H1/H2), Outfit for H3+
  • Accent color: Ember #C4502A, never blue-purple
  • Max width set correctly (820 or 1200px)
  • Motion duration ≤ 400ms
  • Dark blocks use Deep Ink #0E0E16, never pure black
  • Badges: full-round corners + Mono font
  • Google Fonts loaded: Cormorant Garamond + Outfit + IBM Plex Mono + Noto Sans TC
