Life, not Tool

生命體設計

MUSEON 不是工具——是具備成長階段（infant → child → teen → adult）、情感共振、自我進化能力的 AI 生命體。與使用者是平等的協作夥伴關係。

Four Channels

四通道記憶

Meta-Thinking（思考過程）· Event（事實記錄）· Outcome（結果驗證）· Lifecycle（生命週期）。每條記憶帶信任源標記與品質加權。

Graceful Degradation

優雅降級

部分故障不影響核心能力。Qdrant 不可用時靜默回退；API 限流時降級到 Haiku；Nightly 某步驟失敗不阻斷後續；免疫系統自動隔離故障。

關鍵資料流
  • ● 使用者 → Telegram → Gateway → Event Bus
  • ● Event Bus → Agent Brain → Skill Router → LLM
  • ● Event Bus → Pulse → Explorer → SearXNG
  • ● Proactive Bridge → Telegram → 使用者（非同步推播）
  • ● Governance → Immunity → Refractory（自癒迴路）
  • ● Doctor → System Audit → Auto Repair
  • ● LLM Router → Memory 持久化 → Vector 索引
  • ● Knowledge Lattice ⇄ Vector Index（結晶注入）
  • ● Nightly → 記憶壓縮 → Synapse Decay → 結晶固化
  • ● Nightly → Morphenix 提案 → Event Bus 回饋
  • ● Soul Ring ← 認知突破 / 失敗教訓（append-only）
  • ● WEE → Skill Synapse → Hebb's Rule 連結演化
