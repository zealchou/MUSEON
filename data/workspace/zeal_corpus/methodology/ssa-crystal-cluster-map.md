## CH1 顧問式銷售總覽（3 顆結晶）

**核心問題**：顧問式銷售到底跟傳統推銷有什麼不同？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C1-01 | 顧問式銷售不是推銷，是一起找答案 | 你不是來賣東西的，你是來幫他釐清「他到底要什麼」的人 |
| C1-02 | 交易是結果，同理是前提 | 沒有同理，就沒有信任；沒有信任，就沒有成交 |
| C1-03 | 好的銷售像醫生：提問比講話重要 | 你問的問題決定他願意告訴你多少 |

**本章能量線**：地（穩住自己的定位）→ 風（打開對話空間）
**SSA-I 對位摘要**：確立「同隊關係」的基本姿態——你是來跟他一起解題的，不是來說服他的。三軸中的「先穩後亮」從這裡開始：先穩住「我是誰」，才能亮出「我能幫你什麼」。

## CH2 八方位能量 × 顧客心理（5 顆結晶）

**核心問題**：八種能量到底是什麼？怎麼對應顧客的心理狀態？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C2-01 | 八方位能量其實是「顧客想被對待的方式」 | 能量不是你的招式，是他的需要 |
| C2-02 | 每種能量對應一種顧客當下的心理需求 | 風=要空間、水=要理解、山=要清楚、火=要方向、天=要格局、地=要安全、澤=要被認同、雷=要行動力 |
| C2-03 | 能量用錯，比沒講話還可怕 | 對方要安全感時你用雷，等於在他傷口上灑鹽 |
| C2-04 | 真正的銷售不是用一種能量，而是「切換」 | 高手是能感受到「現在他需要什麼」然後即時切換的人 |
| C2-05 | 顧問式銷售的本質：能量對位 | 能量對了，話不用多；能量錯了，講再多都是噪音 |

**本章能量線**：八方位全景——風、水、山、火、天、地、澤、雷
**SSA-I 對位摘要**：「場景×對位」的理論基礎。每一種能量都不是固定的人格標籤，而是顧客在「此刻」的心理需求。能量對位 = 你有辦法感知他現在需要什麼，然後給出對的東西。

## CH3 建立關係（5 顆結晶）

**核心問題**：怎麼在最短時間內讓顧客放下戒心？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C3-01 | 關係不是聊天，是讓顧客願意放下戒心 | 關係的目的不是變熟，是讓他願意說真話 |
| C3-02 | 開場能量決定後面能走多深：前 90 秒法則 | 前 90 秒用錯能量，後面 90 分鐘都在補救 |
| C3-03 | 真正的連結不是「共同興趣」，而是「共同問題」 | 你跟他有一樣的煩惱，比你跟他喜歡同一支球隊更有力量 |
| C3-04 | 同理不是安慰，是「站在你這邊」 | 同理不是「我懂你」，是「我跟你一起面對」 |
| C3-05 | 關係的終點不是熟，而是「讓他願意說出真正需求」 | 關係建立的品質指標：他願意跟你講他真正在意的事 |

**本章能量線**：風（打開空間）→ 水（深度同理）→ 地（穩住安全感）
**SSA-I 對位摘要**：「同隊關係」的實戰操作。前 90 秒是「先穩後亮」的縮影——先用風打開空間，再用水建立連結，最後用地穩住安全感。關係建立不是目的，是「讓他願意說出真正需求」的手段。

## CH4 發掘需求（5 顆結晶）

**核心問題**：怎麼挖到顧客真正的需求，而不是他嘴上說的那個？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C4-01 | 需求不是問出來的，是「被感受到」的 | 你讓他覺得被理解，他自己就會把需求說出來 |
| C4-02 | 問問題的順序比問題本身更重要 | 順序錯了，好問題也變成審問 |
| C4-03 | 真正的需求藏在「他不想講」的那句話 | 他閃躲的地方，往往就是他最在意的地方 |
| C4-04 | 安全感是打開需求的唯一鑰匙 | 沒有安全感，他只會告訴你表面的答案 |
| C4-05 | 需求整理完，成交才開始 | 需求沒整理清楚就推方案＝幫他亂買東西 |

**本章能量線**：水（深入感受）→ 山（釐清脈絡）→ 地（穩住安全感）
**SSA-I 對位摘要**：「場景×對位」進入深水區。需求發掘是「先穩後亮」的典型應用——先穩住他的安全感（地），才能用水去深入感受，最後用山來整理需求。核心技術是「問問題的順序」，而不是問題本身。

## CH5 痛點深挖（5 顆結晶）

**核心問題**：怎麼找到顧客真正的痛點，而且讓他自己意識到？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C5-01 | 痛點不是「他說什麼痛」，而是他反覆做卻改不掉的事 | 痛點藏在行為模式裡，不在嘴巴上 |
| C5-02 | 痛點的三層結構：表面行為→情緒連結→底層信念 | 表面在講價格，情緒在講安全感，底層在講「我值不值得」 |
| C5-03 | 提問是揭露，不是審問 | 好的提問讓他自己看見，而不是被你指出來 |
| C5-04 | 重新命名痛點＝幫他換一雙眼睛 | 當你幫他重新定義問題，解法就不一樣了 |
| C5-05 | 痛點用對了，是「他求你幫忙」的瞬間 | 痛點深挖的終點：他主動問你「那怎麼辦？」 |

**本章能量線**：水（深入同理）→ 山（結構化整理）→ 火（點亮方向感）
**SSA-I 對位摘要**：痛點三層結構是「場景×對位」的核心功法。表面行為用山（清楚），情緒連結用水（理解），底層信念用火（方向）。「重新命名痛點」是「先穩後亮」的高階操作——穩住他之後，幫他換一雙眼睛看問題。

## CH6 方案設計（6 顆結晶）

**核心問題**：怎麼設計一個讓顧客「看到自己會變好」的方案？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C6-01 | 方案不是產品介紹，是替顧客畫出一條路 | 你賣的不是產品，是他從現在走到未來的那條路 |
| C6-02 | 三段式方案設計：現在→橋樑→未來 | 讓他看見「我在哪、要去哪、怎麼去」 |
| C6-03 | 用「減法」設計方案：少＝強 | 方案越少越有力，選項太多等於沒有方向 |
| C6-04 | 時間線讓方案變得真實 | 沒有時間線的方案是空想，有時間線的方案是計畫 |
| C6-05 | 情緒上的認同感比邏輯上的合理性更重要 | 他不是因為合理才買，是因為「感覺這就是我要的」才買 |
| C6-06 | 方案是一面鏡子：讓他「看見自己會變好」 | 好的方案讓顧客看到的不是你的產品，是未來的自己 |

**本章能量線**：山（結構清楚）→ 火（願景方向）→ 天（格局提升）
**SSA-I 對位摘要**：方案設計是「場景×對位」的產出階段。三段式（現在→橋樑→未來）對應「先穩後亮」——先穩住「你現在在哪」（山），再亮出「你可以走到哪」（火→天）。減法設計是顧問式銷售的核心美學：少即是強。

## CH7 第一次成交（7 顆結晶）

**核心問題**：怎麼讓成交自然發生，而不是「逼他買」？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C7-01 | 成交不是關門，是讓顧客安心地「開始」 | 成交是一段關係的起點，不是終點 |
| C7-02 | 三段式自然成交法 | 確認需求→連結方案→邀請決定——讓他覺得這是自然的下一步 |
| C7-03 | 用「小承諾」開始，大承諾就會來 | 先讓他答應一件小事，大的承諾就不遠了 |
| C7-04 | 成交語言：讓顧客覺得「這本來就是我要的」 | 成交語言不是逼，是讓他確認自己的選擇 |
| C7-05 | 沉默是成交最強的武器 | 該說的說完，就閉嘴——讓他自己決定 |
| C7-06 | 如果不成交，就讓他記住你 | 這次不買沒關係，但他下次想到這個需求時第一個想到你 |
| C7-07 | 成交是「共同決定」，不是你的勝利 | 好的成交是「我們一起決定了一件事」 |

**本章能量線**：火（方向明確）→ 天（格局提升）→ 雷（推動行動）→ 地（穩住承諾）
**SSA-I 對位摘要**：成交是「先穩後亮」的收尾。先穩住他的安全感（確認需求），再亮出方向（連結方案），最後用雷推動行動。但雷之後要回到地——穩住承諾，讓他不後悔。沉默（地的能量）是最高階的成交技術。

## CH8 客戶經營（8 顆結晶）

**核心問題**：成交之後怎麼經營，讓顧客從「買了」變成「信任你」？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C8-01 | 30 天顧客全景：從「對位」到「信任」的旅程 | 成交只是起點，30 天內的經營決定他會不會留下來 |
| C8-02 | 前 72 小時決定顧客會不會後悔 | 買完之後的 72 小時是信任最脆弱的時刻 |
| C8-03 | 高頻互動 ≠ 騷擾：節奏感是關鍵 | 太頻繁是騷擾，太少是冷落——關鍵是節奏對 |
| C8-04 | 建立「被期待」的互動模式 | 讓顧客期待你的出現，而不是躲你 |
| C8-05 | 微調是日常，不是危機處理 | 每次互動都在微調，等到他不滿才處理就太晚了 |
| C8-06 | 強化成功經驗：讓顧客累積「我做得到」的證據 | 讓他看到自己的進步，他就不會想離開 |
| C8-07 | 30 天後的關係深化：從顧客 → 合作夥伴 | 30 天是分水嶺：之後他不再是「顧客」，是「一起走的人」 |
| C8-08 | 顧客生活圈擴散：當他開始「說給別人聽」的那一刻 | 最好的行銷不是你說，是他替你說 |

**本章能量線**：地（穩住承諾）→ 風（保持對話空間）→ 水（持續同理）→ 火（強化方向感）→ 澤（認同與讚賞）
**SSA-I 對位摘要**：客戶經營是「同隊關係」的長線延伸。30 天全景展現了「先穩後亮」的節奏感：前 72 小時穩住（地），然後持續用風和水保持連結，用火強化方向，最後用澤讓他感受到被認同。「被期待」是最高境界——他主動想要你出現。

## CH9 顧客升級 × 續約 × 再成交（7 顆結晶）

**核心問題**：面對各種成交障礙（價格、比較、拖延、多人決策），怎麼用對位化解？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C9-01 | 價格焦慮對位：顧客不是怕貴，是怕買錯 | 處理價格問題的關鍵不是降價，是降低他的恐懼 |
| C9-02 | 競品比較對位：顧客比較的不是產品，而是安心感 | 他問你跟別人差在哪，其實在問「跟你買比較安全嗎？」 |
| C9-03 | 進退場判斷：什麼時候推、什麼時候停、什麼時候先穩 | 不是每次都要推進，有時候停下來比往前走更有力量 |
| C9-04 | 多人決策對位：顧客不是一個人，而是一個「小系統」 | 面對多人決策，要找到「真正做決定的那個人的擔心」 |
| C9-05 | 需求模糊對位：顧客說不清楚，不是不要，是「心裡亂」 | 他說「我再想想」，你要做的不是等，是幫他整理 |
| C9-06 | 顧客說「我再想想」：拖延不是拒絕，是速度不對位 | 他不是不要，是你的節奏跟他的節奏不一樣 |
| C9-07 | 升級心法：顧客不是要更多，而是要「更適合現在的我」 | 升級不是賣更貴的東西，是幫他找到更合適的下一步 |

**本章能量線**：地→山→澤→火（穩→清楚→對位→前進）
**SSA-I 對位摘要**：CH9 是「場景×對位」的實戰百科。每一種障礙都有對應的能量組合：價格焦慮用地（穩）+山（清楚），競品比較用水（同理）+地（安全感），拖延用山（整理）+風（空間），多人決策用山（結構化）+澤（認同）。前進線：地→山→澤→火。

## CH10 顧問式關係的長線經營（5 顆結晶）

**核心問題**：怎麼讓一次的顧客變成一輩子的關係？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C10-01 | 顧客不是買一次，而是跟你「走一段路」 | 一次交易是偶然，一段路程是信任 |
| C10-02 | 顧客會記住「第二次幫到他」的人 | 第一次是專業，第二次才是信任 |
| C10-03 | 「情緒帳戶」：經營不是聊天，而是存安全感 | 每次互動都在存款或提款，餘額見底他就走了 |
| C10-04 | 長線信任靠「穩定」，不是靠熱情 | 三年如一日的穩定，比三天的熱情更有力量 |
| C10-05 | 成為顧客「第一個想到的人」的條件 | 不是最便宜、不是最厲害，是「最讓我放心」的那個人 |

**本章能量線**：風→水→山→火→地（長線循環）
**SSA-I 對位摘要**：長線經營是「同隊關係」的終極形態。能量線是循環的——風（保持開放）→水（持續理解）→山（穩定清晰）→火（持續帶方向）→地（穩住信任）→ 再回到風。「情緒帳戶」是「先穩後亮」的日常化：每次互動都在穩住（存款），偶爾才亮一次（提供價值）。

## CH11 業務高手的內在功法（5 顆結晶）

**核心問題**：技術都學會了，但為什麼有些人就是做不到？差在哪裡？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C11-01 | 穩定比技巧重要：高手先練心，才練方法 | 心不穩，技術再好也會走樣 |
| C11-02 | 面對拒絕的韌性：拒絕不是否定，是資訊 | 他拒絕的是這次的方案，不是你這個人 |
| C11-03 | 反脆弱心態：越賣越穩、越做越輕鬆 | 每一次失敗都在幫你校準，不是打擊你 |
| C11-04 | 專注功法：把力氣放在三件最重要的事 | 高手不是什麼都做，是只做最關鍵的三件事 |
| C11-05 | 成長節奏：不是狂，而是長；不是爆，而是穩 | 真正的成長像種樹，不像放煙火 |

**本章能量線**：內功線——心穩（地）→心韌（山）→心強（火）→心準（天）→心續（水）
**SSA-I 對位摘要**：CH11 是對自己的「先穩後亮」。先穩住自己的心（地），練出韌性（山），長出力量（火），找到格局（天），最後保持流動（水）。這是「同隊關係」的內在版本——你要先跟自己同隊，才能跟顧客同隊。

## CH12 整合 × 實戰劇情模式（6 顆結晶）

**核心問題**：所有東西怎麼串在一起？實戰中怎麼用？

| 編號 | 名稱 | 核心句 |
|------|------|--------|
| C12-01 | 顧問式銷售五幕劇情：建立→探索→願景→行動→承諾 | 每一次銷售都是一齣五幕劇——知道自己在第幾幕，就知道該做什麼 |
| C12-02 | 八能量 × 五幕劇情的自然切換 | 建立（風×水）→探索（水→山）→願景（火）→行動（山→火→天）→承諾（天×雷） |
| C12-03 | 錯誤 vs 成功：同一句話為什麼差那麼多？ | 能量對了，普通的話也有力量；能量錯了，漂亮的話也是噪音 |
| C12-04 | 七大場景應對：顧問式三句法（穩→明→走） | 任何場景都可以用三句話：第一句穩住情緒、第二句釐清問題、第三句帶出方向 |
| C12-05 | 顧全文案模板（三句萬用法） | 穩（我理解你的感受）→明（讓我們一起看看）→走（你覺得這樣如何？） |
| C12-06 | 顧客心理地圖：一句話判斷他在哪一幕 | 他說什麼話，就知道他在哪一幕，然後給對的能量 |

**本章能量線**：成交線——風→水→山→火→天→雷→地（完整七段）
**SSA-I 對位摘要**：CH12 是全書的終極整合。五幕劇情是「場景×對位」的完整實踐，每一幕都有對應的能量組合。三句法（穩→明→走）是「先穩後亮」的極簡版本。顧客心理地圖讓你一句話就能判斷該用什麼能量——這是「同隊關係」的最高形態：你比他更懂他在哪裡。
