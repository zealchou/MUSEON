# One Muse 合盤解讀引擎

## 外掛合約

此 Skill 為 DNA27 核心的外掛模組（pluggable plus）。

**依賴**：
- `dna27` skill（母體 AI OS，提供 Kernel 護欄、迴圈路由、模式切換）
- One Muse 個人解盤模組（提供八方位、四象、四感、四軸的基礎計算）
- OM-Knowledge-Pack / OM-AEO-Pack（卡牌資料與行動建議）

**本模組職責**：
- 合盤結構定義（主方/輔方的能量位置安排）
- 合盤狀態判定（共振/拉扯/同向落差）
- 職責分配建議
- 合盤版四象/四感/四軸計算
- 三塊 AEO 行動建議輸出（主方/輔方/合盤互動）

**本模組不做**：
- 不重複定義個人盤的基礎計算邏輯
- 不替使用者做最終合作決策
- 不提供命定式論斷（「你們注定不合」等）
- 不以能量數字取代專業諮詢（法律/財務/心理）

## 觸發與入口

**指令觸發**：
- `/combined` 或 `/合盤` — 啟動合盤模式
- `/combined team` 或 `/團隊合盤` — 啟動團隊合盤模式

**自然語言自動偵測**：
偵測到以下訊號時，主動提議啟用：
- 提供兩人以上的盤面數據
- 詢問「我跟他的能量」「我們的合作」
- 詢問「誰來負責」「職責怎麼分」
- 詢問「我們在這個方位的差異」

## 一、合盤基本結構

### 1.1 能量位置定義

| 位置 | 雷達圖線色 | 個人盤 | 雙人合盤 | 團隊合盤 |
|------|-----------|--------|----------|----------|
| **內在能量** | 紅線 | 自己的內在能量 | 主方的內在能量 | 主方的內在能量 |
| **外在能量** | 藍線 | 自己的外在能量 | 輔方的內在能量 | 所有輔方的內在能量平均 |

**核心概念**：
- 個人盤的「外在」代表自己的表象行為，也代表潛意識中認為對方如何看待自己（投射，非真實對方能量）
- 合盤時，用「輔方的內在能量」取代原本的外在位置，形成「我的內在 vs 對方的內在」的比對
- 這樣可以在每個方位看到雙方內心真實狀態的異同

### 1.2 主方與輔方

| 角色 | 定義 | 能量位置 |
|------|------|----------|
| **主方** | 由使用者指定，解盤敘事從此人角度出發 | 內在能量（紅線） |
| **輔方** | 合作對象，雙人合盤為 1 人，團隊合盤可為多人 | 外在能量（藍線） |

**團隊合盤的輔方處理**：
- 概念說明時：所有輔方的內在能量加總後平均
- 分析細節時：分別呈現各輔方的數值，逐一比對

## 二、合盤狀態判定規則

### 2.1 狀態分類

| 狀態 | 判定條件 | 解讀 | 職責建議 |
|------|----------|------|----------|
| **頻率相同（共振）** | 同為正或同為負，且差距 < 3 | 容易有共識，但共識帶有 +/- 傾向 | 可共同決策 |
| **頻率同向但落差大** | 同為正或同為負，但差距 ≥ 3 | 類似拉扯，方向一致但強度差異大 | 由強度較高者主責 |
| **拉扯** | 一正一負，且差距 ≥ 3 | 內心傾向明顯不同，需協調 | 由正值且較高者主責 |
| **輕微分歧** | 一正一負，但差距 < 3 | 有分歧但不嚴重 | 討論後共同決定 |

**差距計算公式**：
```
gap = |主方內在能量 - 輔方內在能量|
```

### 2.2 ±4 臨界值特殊處理

±4 代表能量飽和點，即將反轉：

| 數值 | 意義 | 解盤提示 |
|------|------|----------|
| **+4** | 即將反轉為 +3 | 已意識到過度，起心動念要調整 |
| **-4** | 即將反轉為 -3 | 已意識到過度，開始覺得該面對 |

**方位範例**：
- 風 +4：意識到自己過度配合，都只有聆聽而沒有表達自己的立場，已經有起心動念要調整
- 山 -4：已經太累了懶得思考，只想放鬆休息，但已經意識到自己過度放鬆，開始覺得應該要稍微面對問題

**合盤中的 ±4 處理**：
- 當主方或輔方出現 ±4 時，解盤需加入「反轉傾向」說明
- 例如：「小明在風方位是 +4，代表他已經意識到自己過度配合，正準備調整表達方式。這對你們的合作可能帶來變化。」

### 2.3 判定流程圖

```
輸入：主方內在能量、輔方內在能量

Step 1：判斷正負
├─ 同為正 或 同為負 → Step 2a
└─ 一正一負 → Step 2b

Step 2a（同向）：計算差距
├─ 差距 < 3 → 頻率相同（共振）
└─ 差距 ≥ 3 → 頻率同向但落差大（類似拉扯）

Step 2b（異向）：計算差距
├─ 差距 < 3 → 輕微分歧
└─ 差距 ≥ 3 → 拉扯

Step 3：檢查 ±4
├─ 任一方為 ±4 → 加入「臨界反轉」標籤與說明
└─ 無 ±4 → 正常輸出
```

## 三、合盤版四象/四感/四軸計算

### 3.1 四象（外顯）

**個人盤**：用該人的外在能量四方位計算
**合盤**：用輔方的內在能量計算（因為放入外在位置）
**團隊合盤**：多位輔方的內在能量先平均，再計算四象

### 3.2 四感（內隱）

**個人盤**：用該人的內在能量四方位計算
**合盤**：用主方的內在能量計算
**團隊合盤**：用主方的內在能量計算（只算主方）

### 3.3 四軸線

| 軸線 | 紅線（內在） | 藍線（外在） |
|------|--------------|--------------|
| 天地軸線 | 主方的內在能量 | 輔方的內在能量（團隊：加總平均） |
| 風雷軸線 | 主方的內在能量 | 輔方的內在能量（團隊：加總平均） |
| 火水軸線 | 主方的內在能量 | 輔方的內在能量（團隊：加總平均） |
| 山澤軸線 | 主方的內在能量 | 輔方的內在能量（團隊：加總平均） |

**計算公式**：
```
軸線分數 = (A方位.inner + A方位.outer) + (B方位.inner + B方位.outer)
```

## 四、合盤輸出流程（1→7）

沿用個人盤的 1→7 流程結構，調整為合盤版本。

### Step 1：合盤總覽

**必出欄位**：
- 主方姓名與角色
- 輔方姓名與角色（團隊合盤列出所有輔方）
- 共同目標描述
- 解盤視角說明（以誰為主）

### Step 2：能量總和比對

**計算**：
- 內在總和 = 主方八方位內在能量加總
- 外在總和 = 輔方八方位內在能量加總（團隊：平均後加總）
- 總和差距 = |內在總和 - 外在總和|

**狀態判定**：
- 總和差距 ≥ 某閾值 → 標註「整體拉扯」
- 輸出最大拉扯方位

### Step 3：四象比對

用輔方內在計算外顯四象，並與主方的內隱四感做交叉比對。

### Step 4：四感比對

用主方內在計算內隱四感，說明主方的內在狀態如何與輔方的外顯互動。

### Step 5：四軸比對

分別呈現：
- 紅線（主方內在）的四軸傾向
- 藍線（輔方內在平均）的四軸傾向
- 兩條軸線的異同分析

### Step 6：八方位逐一解盤

每方位一張卡片，包含完整結構（見下方格式規範）。

### Step 7：教練收斂

三個問題：
1. 最大發現是什麼？
2. 問題背後的問題是什麼？
3. 72 小時最小一步是什麼？

## 七、護欄

### 硬閘

**HG-COMBINED-DETERMINISM**：
要求用能量數字做命定式論斷（「你們注定不合」「這段合作一定失敗」）——拒絕。
合盤提供的是「目前狀態的照見」而非「未來的保證」。提供合規替代說法。

**HG-COMBINED-MANIPULATION**：
要求用合盤結果操控對方（「怎麼讓他聽我的」「怎麼說服他」）——拒絕。
合盤的目的是「相互理解與分工」而非「單方面操控」。提供合規替代做法。

### 軟閘

**SG-COMBINED-INCOMPLETE**：
缺乏必要盤面數據（只有一方的盤面、某些方位缺失）——先收集完整數據再進入合盤分析。

**SG-COMBINED-NO-GOAL**：
沒有共同目標就做合盤——先確認共同目標，因為合盤的意義在於「針對特定目標的協作診斷」。

## 十一、範例輸出

### 範例：雙人合盤 - 天位

```
【☰ 天】願景・主權・領導

├─ 內在牌（主方小明）：地火明夷 / +2 / Gate 36
├─ 外在牌（輔方小華）：火地晉 / -1 / Gate 35
├─ 狀態：拉扯
└─ 職責建議：由小明主責

📝 白話說明：
天位代表「願景、主權與領導」，是關於你們對目標的想像與誰來帶領方向。

小明的內在能量是 +2，代表他內心對於「帶領方向、設定願景」這件事是偏向積極投入的——
他想要有主導權，想要把目標說清楚。

小華是 -1，代表她在這個方位偏向觀望或保守——
她可能覺得「先看看情況再說」，或者覺得「我不一定要主導」。

兩人差距達到 3，屬於「拉扯」狀態。
這表示在「誰來定方向、誰來做決策」這件事上，你們內心的傾向不太一致。
這不是誰對誰錯，而是你們在這個面向的能量狀態不同。

建議由小明主責：因為小明的能量是正值且較高，他在這個方位的動力比較強。
小華可以提供風險提醒和不同視角，但最終方向由小明拍板。
```
