## 最高風險段落識別

### 🔴 風險 1：「三層各自抽成，不疊加」（v2.0 L90-100）

**原文**：
> 三層為**獨立等級，各自抽成**，不疊加。同一客戶的月費分潤，三層各自依自己的級距抽取。
> 
> **同一客戶最多被三層各抽一次**，上限為三層分潤加總。

**問題**：乍看之下像「多層次獎金」；需澄清這是**垂直員工薪酬**，不是**橫向代理網路**。

**修改建議**：
```
改為：「MUSEON 採用三層垂直員工薪酬結構。同一客戶的月費分潤，由負責該客戶的員工、其直屬主管、及其所屬經銷商各自根據自身級距和服務價值計算報酬。三層報酬基於各自員工身份獨立計算，不存在『上線獲利』的機制。」
```

---

### 🟡 風險 2：「招募並培訓業務人員」（v2.0 L32, v3.0 L104）

**原文**：
> 經銷商：完成培訓 + 能獨立輔導客戶，**可招募並培訓業務人員**

**問題**：「招募」二字容易被誤解為傳直銷式的「拉人」。

**修改建議**：
```
改為：「經銷商：完成培訓 + 能獨立輔導客戶，有權 **聘用並培訓業務人員**」

或更明確：「經銷商：經公司認證，可 **組建直屬業務團隊**（所有業務人員簽訂 MUSEON 員工合約）」
```

---

### 🟡 風險 3：「旗下業務簽的客戶」（v2.0 L97, v3.0 L116）

**原文**：
> 業務主管 | 旗下業務簽的客戶 | 依主管自己級距抽

**問題**：「旗下」容易被解讀為「下線」；應改為**直報**或**直屬**。

**修改建議**：
```
改為：「業務主管｜直屬業務員簽署的客戶｜依主管自己級距抽」

或：「業務主管｜其團隊直報員工簽署的客戶｜依主管自己級距計算佣金」
```

## 沒有傳直銷風險的機制

✅ **月費分潤不是「多層獎金」**  
- 每層獨立計算，不累積疊加
- 客戶付月費一次，被三層各分配一次（不是客戶被重複收費）
- 類似傳統公司的「主管分紅」，不是傳銷

✅ **引薦獎金是一次性，不是持續佣金**  
- $3,000 為一次性獎勵
- 不計入積分
- 客戶本身有獎勵（免月租），形成雙向激勵

✅ **升降級完全基於績效**  
- v2.0：連續 3 個月案件數
- v3.0：月度積分（新案 + 活躍客戶養護）
- 沒有「拉人數」作為升級門檻

✅ **防反套利機制健全**  
- 同統編 90 天內只算 1 次新案
- 引薦需主管確認
- 追回客服不得為原負責業務

## 終局判定

**傳直銷風險等級：🟡 中等（需文案調整，不需制度改變）**

**原因**：
1. 制度本身是合法的垂直員工薪酬
2. 但用詞「招募、旗下、分潤」可能被敵意解讀
3. 「三層各抽」的敘述不夠清晰，容易被誤解

**建議行動**：
1. 將 v2.0/v3.0 文件中的 4 個高風險詞彙替換（見上表）
2. 在商業計畫和銷售文案中，強調「員工制」而非「代理制」
3. 新增一份「與傳銷的區別」的白皮書（如有對外溝通需求）
