第 04 章：五大中小企業 Skill 組合實戰

核心洞察：MUSEON 的威力不在單個 Skill，而在「如何組合 Skill 應對特定困境」。以下五個場景都是真實發生、超過 10 人公司的實際困境——展示 MUSEON 如何在 18 小時內交付別人需要 3 週才能準備的成果。

1️⃣

客戶拜訪情報戰

⏱️ 18 小時 | 核心技能：ANIMA + Ares + Shadow

開頭困境

董事長 2024/09/15 十點半要拜訪福建某集團老總，談 500 萬融資。現在是前一天晚上 6 點。

一般 Agent 的做法

「我幫你 Google 搜尋這個人」→ 回傳 5 篇新聞 → 完。

MUSEON 的做法

客戶訊息
├─ anima-individual
│ ├─ 建檔：對手過往 10 年人物檔案
│ ├─ 七層鏡像：權力欲 / 利益傾向 / 人脈結構 / 關鍵決策信號
│ └─ 互動溫度：過去接觸過嗎？關係溫度幾度
├─ ares（戰神系統）
│ ├─ 多視角戰略分析：老總視角 vs 集團視角 vs 金主視角
│ ├─ 八大槓桿尋路
│ │ • 身份槓桿（他在集團的實權？只是代理人？）
│ │ • 利益槓桿（他要的是什麼？做大做強 vs 套現撤出）
│ │ • 人脈槓桿（他的決策權人？他背後的金主是誰）
│ │ • 時間槓桿（為什麼現在要融資？融資時間表是什麼）
│ └─ 陰謀辨識：可能有隱藏方 / 同業兩邊押注 / 政商關係風險
└─ shadow（人際博弈引擎）
├─ 對抗型態識別：他是快狠準 / 還是溫水煮青蛙
├─ 情感博弈模式：會不會用時間壓力脅迫 / 用親近感試探底線
├─ 防禦訊號：可能的騙局套路、測試話術反擊點
└─ 情緒穩定性：談判中可能的人格變化

具體結果

晚上 7 點開始 → 早上 8 點交付

產出：

  • A4 情報卡：對手 8 層人物模型 + 可能陷阱 + 反制話術
  • 談判時間線：預測他可能的出牌順序 + 每個階段最有可能的施壓點
  • 三條出牌路徑：
    1. 他真的想融資 → 你如何確認不是虛晃
    2. 他想打探競爭對手 → 如何識別並反制
    3. 他想試價格底線 → 價格談判邊界線在哪

💡 關鍵結果

董事長進去前已經知道「他可能在第 15 分鐘拋出同業融資邀請來測價」，進去直接回應「我知道你的玩法，我的價格是 X」。

  • 談判時間：從預計 2 小時 → 45 分鐘
  • 融資金額：額外加碼 20%
2️⃣

品牌定位到文案執行

⏱️ 10 天 | 核心技能：brand-discovery + brand-builder + text-alchemy

開頭困境

創辦人說：「我這個產品明明很好用，為什麼沒人買？廣告費燒了 200 萬，轉換率卻只有 0.3%。」

一般 Agent 的做法

寫個「產品特性分析」+ 「建議改進文案」→ 完。

MUSEON 的做法

品牌定位層
├─ brand-discovery（漸進式訪談）
│ ├─ 一次一問抽出客戶真實需求
│ ├─ 識別「你自己以為的需求」vs「客戶真實的卡點」的落差
│ └─ 找出品牌定位的真正支撐點
├─ brand-builder（奧美級策略）
│ ├─ Jung 12 原型匹配
│ ├─ 目標族群重新分層
│ ├─ 競爭定位矩陣
│ └─ 品牌訊息建築：策略層 → 承諾層 → 證據層 → 情感層
└─ text-alchemy + storytelling-engine
├─ 廣告文案重寫：從「功能堆砌」→「故事 + 共鳴 + CTA」
├─ 社群內容範本：5 個月的月曆內容
├─ 銷售信函
└─ 落地執行檢查單

具體結果

第 1-2 天（brand-discovery）：透過 12 個問題，挖出真實痛點：不是「使用性」差，而是客戶對「自己配不配用」有隱形自我懷疑。

第 3-4 天（brand-builder）：重新定位，從「高端產品」→「陪伴型工具」。新訊息架構：「不是讓你變成完美的人，是陪你成為更好的自己」。

第 5-10 天（text-alchemy）：改寫全套文案 + 10 篇真實用戶故事內容。

📊 轉換率翻轉

  • 轉換率：0.3% → 1.8%（6 倍）
  • 客獲成本：¥6,000/人 → ¥1,200/人
  • 客戶終身價值：提升 240%

關鍵洞察：改文案的人每週 20 篇，改定位的人一年只有一次機會——所以威力完全不同。

3️⃣

融資路演籌備

⏱️ 5 天 | 核心技能：market-core + ssa-consultant + investment-masters

開頭困境

團隊要去參加募資說明會，有 10 分鐘向 50 位潛在投資人 pitch。金額目標 ¥2,000 萬。現在連 slide 都沒有。

一般 Agent 的做法

「我幫你寫 10 頁 PPT」→ 完。

MUSEON 的做法

投資方情報 & 精準訊息設計
├─ market-core
│ ├─ 行業總經背景
│ ├─ 競品籌碼圖
│ ├─ 融資週期判斷
│ └─ 政策風險提示
├─ ssa-consultant（顧問銷售）
│ ├─ 投資人分類與訊息分層
│ │ • 對 PE 老闆：市場規模 + 增速
│ │ • 對股權投資者：團隊執行力 + 重複購買率
│ │ • 對戰略投資者：技術壁壘 + 生態協同
│ ├─ MECE 拆解法
│ └─ 異議處理話術
├─ consultant-communication
│ ├─ 麥肯錫金字塔原則
│ ├─ 故事框架
│ └─ 說話節奏
└─ investment-masters（投資軍師）
├─ 護城河評估
├─ 安全邊際檢查
├─ 反向思考
└─ 大師會診

具體結果

第 1 天：market-core 發現「現在融資環境比你預期冷 20%」，調整融資目標 ¥2,500 萬 → ¥1,800 萬。

第 2-3 天：設計 3 套不同 pitch 文稿（對應 PE / VC / 戰略投資者）。investment-masters 用「安全邊際」框架找出你的護城河。

第 4-5 天：12 頁 slide + 說話稿 + 異議問答卡。產出「融資 60 秒簡版」。

🎯 融資成功

  • 現場報名跟進：12 位投資人
  • 成交金額：¥1,750 萬
  • 價格估值：沒被低估（因為第 2 天調整了目標）

關鍵：有 3 位投資人後來說「最打動我的不是 slide，是你對『行業衰退時為什麼還要融資』的那句話」——那句話來自 market-core 的洞察。

第 2 天（dharma + ares）：根本原因是「公司沒有協作決策機制」。改變激勵機制，而不是叫他們「要互相理解」。

  • 8 週後，銷售主動說「我不催工程了，因為我們一起看市場數據決定 timing」

關鍵洞察：改機制比改人性有效 1000 倍。

公司開發了 SaaS 產品，費用模型還沒定。老闆說「對標產品好像都是 ¥299/月」，但不知道自己該定多少。不確定是主推「企業版」還是「個人版」。

一般 Agent 的做法

MUSEON 的做法

├─ market-core

├─ business-12（商模十二力）
│ ├─ 收入流設計

└─ storytelling-engine + text-alchemy

├─ 個人版訊息
├─ 銷售信函
└─ 廣告文案 6 個月月曆

具體結果

第 1-2 天：market-core 發現競品 ¥299 是抄別人的，沒驗證過。真實情況：企業用戶願意付 ¥1,200/月。

第 3 天：business-12 建議「不要二選一，雙軌制」——企業版 ¥1,200 + 個人版 ¥99。資源配置 7:3（企業優先）。

第 4-5 天：risk-matrix 檢查：如果對手降到 ¥699，你會不會掉客戶？結論：只要交付速度領先 2 個月，不會。定價可以更激進。

第 6-7 天：storytelling 設計故事 + text-alchemy 產出文案。

💰 定價策略成功

  • 個人版：¥99/月
  • 3 個月成績：
    • 企業版：8 家客戶，MRR ¥9,600
    • 個人版：3,200 個免費用戶，轉化率 3.2%，MRR ¥10,000
    • 總 MRR：¥19,600 / 年化 ¥235,200

核心差異：一般 Agent 說「對標 ¥299」，MUSEON 說「知道企業用戶願意付 6 倍，所以你虧大了」。
