# 戰略三稜鏡——深度知識庫

> 來源：MUSEON 原生 Skill（master-strategy/shadow/roundtable）+ 國際戰略 DSE（2026-04-04）
> 融合孫子兵法、Clausewitz、Boyd OODA、Wardley、博弈論、Kahneman、Klein、Voss、Greene、DQ Framework

## 核心命題
人類世界有陽謀與陰謀——陽謀是看得見的策略佈局，陰謀屬於人心。
master-strategy 處理陽謀，Shadow 處理陰謀，Roundtable 讓兩者在衝突中碰撞。
三者兼備才是完整的人類世界認知。

## 稜面一：陽謀（可見策略）

### 局勢判斷
- Clausewitz 三概念：戰爭迷霧（資訊永遠不完整）、摩擦力（計畫與現實的落差）、重心（敵方力量核心）
- OODA Loop：勝負不在單次決策品質，在循環速度。能進入對手決策循環內部就能主導
- Wardley Mapping：一切沿演化軸右移（Genesis→Commodity），差異化來源正在右移時必須往左找新元件
- 孫子五事七計：道天地將法的系統性盤點

### 路徑設計
- 奇正轉換：正兵接敵、奇兵致勝，奇正可互換
- Nash Equilibrium：理性行為者的穩態，但抵達前的過程充滿破壞
- 信號理論：成本越高的信號越可信——低品質方無法偽造的才是有效信號
- 備選方案 ≥ 3 個，否則不是策略是賭博

## 稜面二：陰謀（隱形動態）

### 動機辨識
- Shadow 防禦層 27 型態：辨識對方正在使用的對抗招式（主權反轉、覺察摧毀、詭辯轉移...）
- Shadow 洞察層 27 模式：理解招式如何運作（戒斷依賴、親密轟炸、推拉震盪...）
- Dark Psychology 防禦：Gaslighting/DARVO/Love Bombing/Manufactured Urgency 的識別與反制
- 核心原則：把對方的話當面值是最危險的假設

### 影響力動態
- Voss 戰術同理心：Labeling（標籤情緒）+ Mirroring（鏡像）+ Calibrated Questions（校準問題）
- Greene 權力法則：不要光芒蓋過主人、隱藏意圖、少說話
- Influence Without Authority：識別對方的交換貨幣（認可/資源/資訊/關係）
- 鬼谷子捭闔：開合控制、需求探測

## 稜面三：鏡照（決策品質）

### 認知校準
- Kahneman System 1/2：95% 日常決策由直覺做出，有系統性偏誤
- Klein RPD：有效反饋環境中的專業直覺可信，低反饋環境不可信
- Pre-mortem：假設已失敗倒推原因，最有效的反確認偏誤工具
- 認知去偏四技術：外部基率、對立假說、10/10/10 規則、紅隊

### 多角驗證
- Roundtable 常設三角：陽謀者 + 陰謀者 + 鏡照者
- Red Team：指定人找計畫漏洞，獎勵發現問題
- 螺旋仲裁：裁決不是終點，是下一輪追問的起點

### 決策紀律
- DQ 六邊形：框架/選項/資訊/價值觀/推理/承諾，最低分決定總分
- EVOI：花多少資源蒐集資訊才值得？大多數組織過度蒐集低價值資訊
- Bezos Type 1/2：不可逆決策慎重，可逆決策快速
- Clausewitz 摩擦力：不為摩擦預留緩衝的計畫不是計畫

## 三面稜鏡的協作邏輯
局勢判斷(陽) → 動機辨識(陰) → 認知校準(鏡) → 路徑設計(陽) → 影響力操作(陰) → 多角驗證(鏡) → 決策紀律(鏡) → 行動

## 關鍵量化指標
- OODA 循環速度
- 備選方案數 ≥ 3
- Pre-mortem 執行率
- DQ 六邊形最低分
- 校準信心偏差 < 10%
- 反對意見紀錄數 ≥ 1
- 決策可逆性評估完成率
