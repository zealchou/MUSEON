# 轉化漏斗三角星——深度知識庫

> 來源：SSA M-13 三重漏斗 + SSA-I 學員手冊 + 國際轉化框架 DSE（2026-04-04）

## 核心命題
三個漏斗只是在不同地方、用不同方式，證明同一個改變是真的。
漏斗是手段，Transformation 才是靈魂。
營收是價值流的指標，不是價值的替代品。

## 設計原則：PDEIF 逆熵流
產品漏斗的設計邏輯是**逆熵流程設計（PDEIF）**——不是從入口往上堆，而是從終點逆推：
1. **起點**：品牌價值主張（Transformation / Key Shift）
2. **第二步**：核心產品（能完整交付 Transformation 的最大包）= 重力中心
3. **第三步**：逆推產品階梯——每一階都是下一階的自然前置條件
4. **第四步**：設計三條漏斗——各自用不同方式證明同一個 Transformation 是真的

從入口往上堆 = 產品之間缺乏內在邏輯，客戶不知道為什麼要升級。
從價值主張逆推 = 每一步都在回答「如何更接近完整的 Transformation」。

## SSA 根基（來自學員手冊）
- SSA-I.01：商業的本質是價值交換（功能/情感/象徵三層）
- SSA-I.02：價值由感覺決定（感覺先於邏輯）
- SSA-I.03：價值是消費者決定的（品牌要「對齊」而非「說服」）
- SSA-I.04：品牌三角對位（老闆信念 × 顧客認知 × 產品表現，三角重疊才有品牌）
- SSA-I.05：品牌一致性的工程（理念/語言/視覺/行為四層閉環）
- SSA 五層架構：哲學(L1) → 語氣(L2) → 結構(L3) → 能量(L4) → 控制(L5/PCT)

## 01. 轉變設計（Transformation Design）
- Before → After → Key Shift
- Key Shift 是後面所有產品、行銷與社群的唯一核心
- 沒有清晰的 Transformation = 三條漏斗各說各話
- StoryBrand：客戶是英雄，品牌是嚮導
- 價值交換三層次：功能價值→情感價值→象徵價值

## 02. 產品階梯（Product Ladder A→D）
- A（入口）→ B（主產品）→ C（深度）→ D（系統/授權）
- Value Ladder（Brunson）：免費→低價→中價→高價
- Tripwire：「從不買到第一次買」的心理障礙 > 「從小額到大額」
- 核心產品是「重力中心」，其他階梯圍繞它轉
- Land & Expand：現有客戶擴張的成功率 60-70%（vs 新客 5-20%）

## 03. 產品漏斗
- 客戶從入口開始 → 自然走到主產品 → 完成核心 Transformation
- AIDA → AARRR → Flywheel 的演進：購買不是終點而是飛輪啟動點
- Freemium 自助轉換基準 3-5%，優秀 >8%
- 試用期 ≤7 天轉化率（40.4%）優於 >61 天（30.6%）——緊迫感是槓桿

## 04. 行銷漏斗
- 客戶最不相信什麼（Belief Gap）→ 用什麼證明（Proof Method）→ 讓他踏出下一步
- 92% 消費者信任個人推薦，30% 信任廣告
- 加入 testimonial 轉換率提升 5-25%
- 5 則以上評論 → 購買機率 ×270（vs 零評論）
- 品牌三角對位：信念×認知×表現三角重疊，價值才會被「看見」

## 05. 社群漏斗
- 進入社群 → 參與互動 → 升級購買
- 1%/9%/90% 法則：1% Champion / 9% Contributor / 90% Lurker
- 社群成員 LTV 比非社群高 25-40%
- 200 個高活躍成員 > 20,000 個沉默成員
- CLG 社群驅動交易 72% 在 90 天內成交（vs 銷售 42%）

## 06. 決策流
- 最猶豫的點 → 關鍵安心點 → 未購買最常卡住的原因
- Fogg B=MAP：多數瓶頸在「降低難度」不在「提升動機」
- 83% 流失在「加入購物車前」不是結帳時
- 每增加一個表單欄位轉換率降 11%
- 最強轉換工具不是 CTA 顏色，是「移除選項」

## 07. 身份螺旋（Triple Spiral）
- 使用者 → 參與者 → 示範者 → 協作者 → 系統角色
- 讓價值自然擴散與複製的關鍵
- Champion 推薦到付費轉化率 15-25%（vs 冷廣告 1-3%）
- 推薦客戶 LTV 高出 16-25%

## 08. 增長飛輪
- Growth Loop vs Linear Funnel：漏斗給線性回報，迴圈給複利回報
- K-factor > 1 = 指數增長
- Flywheel（HubSpot）：Attract → Engage → Delight → 回到 Attract
- 留存率決定增長天花板——留存差的產品再多拉新也填不滿
- 推薦獎勵必須與產品核心價值對齊

## 09. 漏斗整合
- 三條漏斗是否講同一個 Transformation 故事
- Binet & Field 60/40：品牌 60% + 效果 40%
- 多觸點歸因取代最後點擊
- 品牌一致性 → 營收 +23-33%（Lucidpress）

## 與其他星座的交叉
- 商模十二力.轉換力(BF12) → 決策流+產品漏斗
- 商模十二力.社群力(BF09) → 社群漏斗
- 商模十二力.產品力(BF01) → 產品階梯
- 商模十二力.感召力(BF06) → 增長飛輪
- 品牌七芒星.品牌敘事 → 行銷漏斗的 Proof Method
- 品牌七芒星.品牌共鳴 → 身份螺旋的最終狀態
- 能量八芒星.天(目標) → 轉變設計的方向感
- 能量八芒星.風(適應) → 行銷漏斗的敘事能力

## 追蹤邏輯
與商模十二力類似（越高越好），但多一個整合維度：
三條漏斗不是獨立評分再加總，而是看「協同一致性」。
三條漏斗都在 0.8 但各說各話 < 三條漏斗都在 0.6 但講同一個故事。
