# 市場七芒星——深度知識庫

> 來源：投資方法論 DSE（2026-04-04）
> 融合 Ray Dalio 經濟機器論、Howard Marks 週期掌握、Warren Buffett / Charlie Munger 價值投資、
> Philip Fisher 成長投資、George Soros 反身性理論、Nassim Taleb 反脆弱、
> Kahneman 行為金融學、Kelly Criterion 倉位管理、Fama-French 因子模型

## 核心命題
市場不是可被預測的機器，而是有週期韻律的有機體
贏家不是預測最準的人，而是在對的週期位置做出對的倉位決策
七維度缺一不可：錯讀一個就足以讓正確的六個白費

## 七維度感知路徑
總經判讀 → 流動性感知 → 估值判斷 → 趨勢辨識 → 情緒判讀 → 風險意識 → 週期定位
（由宏觀到微觀，最終在週期定位收口形成行動信號）

## 01. 總經判讀（macro_literacy）
（Ray Dalio 經濟機器論、PESTEL、CFA 資產配置框架）
- 三力引擎：生產力增長（長期斜率）+ 短期債務週期（5-8年）+ 長期債務週期（75-100年）
- PESTEL 六面向：政治/經濟/社會/技術/環境/法律，任何一面向劇變都是重新定價信號
- 領先指標（先行）：PMI 新訂單、信用利差、殖利率曲線
- 同步指標：GDP、工業產出、就業人數
- 落後指標：CPI 通膨、利率、失業率（確認但不預測）
- 反直覺：GDP 創高時往往是做空的時機，衰退見底時往往是建倉的時機

## 02. 流動性感知（liquidity_awareness）
（Howard Marks、央行政策分析）
- 「信用週期是最有力也最不穩定的力量」——Howard Marks
- 三大觀察窗：央行資產負債表擴縮、基準利率走向、信用利差（Investment Grade 與 High Yield 的息差）
- 流動性的時間尺度與總經不同：總經以季度計，流動性可以日內劇變
- 量化寬鬆（QE）注入流動性 → 資產普漲；量化緊縮（QT）抽走流動性 → 相關性趨向 1（所有資產同跌）
- 反直覺：流動性充沛時風險資產上漲不代表基本面改善，流動性消失才是真正的壓力測試

## 03. 估值判斷（valuation_judgment）
（Warren Buffett、Charlie Munger、Eugene Fama）
- 安全邊際：內在價值（合理估算）與市場價格的差距，差距越大安全性越高
- 三種估值工具：DCF（折現現金流）/ 本益比（P/E，看獲利倍數）/ 股價淨值比（P/B，看資產倍數）
- Munger 多元心智模型：用物理學、生物學、心理學等跨域框架交叉驗證估值假設
- Fama-French 價值因子：低 P/B 股票長期跑贏市場——但需要忍受長期低迷
- 價值陷阱的識別：便宜不等於便宜，護城河消失或業務萎縮的公司估值只會越來越「便宜」
- 反直覺：最貴的資產往往是「好公司」，最便宜的才是真正有安全邊際的機會

## 04. 趨勢辨識（trend_recognition）
（Philip Fisher 閒聊法、Carhart/Fama-French 動量因子）
- Fisher 閒聊法（Scuttlebutt）：訪問競爭對手、供應商、客戶，拼出產業真實圖景
- 動量因子：過去 12 個月表現好的股票傾向繼續跑贏（Carhart 1997），但反轉在高點附近突然發生
- 結構性趨勢（10 年以上）vs 週期性趨勢（3-5 年）vs 短期波動（<1 年）需要分層辨識
- 趨勢判斷的三個問題：誰在買？為什麼買？這個理由能持續多久？
- 反直覺：最清晰的趨勢往往已被充分定價，模糊的早期趨勢才是超額報酬的來源

## 05. 情緒判讀（sentiment_reading）
（George Soros 反身性理論、Daniel Kahneman 行為金融學、Howard Marks 心理鐘擺）
- 反身性理論雙函數：認知功能（市場嘗試感知現實）+ 操控功能（市場感知反過來改變現實）
- 泡沫的本質：正向反身性——上漲→樂觀敘事→更多買入→繼續上漲，直到脫離基本面太遠崩斷
- Kahneman 三大偏誤：過度自信（高估自己的判斷）/ 損失趨避（虧損痛苦是獲利快樂的 2.5 倍）/ 錨定效應（被買入成本綁架）
- Howard Marks 心理鐘擺：恐懼 ↔ 貪婪，市場永遠在兩端過擺，很少停在中間
- 情緒指標：VIX（恐懼指數）、put/call ratio、散戶持倉比例、封面故事指標
- 反直覺：市場情緒最悲觀時往往是最好的買點，媒體頭版吹捧時往往是賣點

## 06. 風險意識（risk_awareness）
（Nassim Taleb 反脆弱、Kelly Criterion）
- 反脆弱三態：脆弱（不確定性傷害）/ 穩健（不確定性中性）/ 反脆弱（不確定性獲益）
- 槓鈴策略：90% 配置極安全資產（現金/短債）+ 10% 配置高風險高報酬標的，中間地帶不碰
- 黑天鵝核心觀點：尾部風險（極端事件）無法被歷史數據量化，用標準差衡量風險是自欺欺人
- Kelly Criterion 倉位管理：最優倉位 = 優勢（勝率×賠率）/ 賠率，過度槓桿必然最終歸零
- 風險不等於波動率：波動是可見的，真正的風險是永久性損失
- 反直覺：最危險的時刻是市場感覺最安全的時候（低 VIX、高信心），此時槓桿最高、保護最少

## 07. 週期定位（cycle_positioning）
（Howard Marks、Ray Dalio、Nikolai Kondratiev）
- Marks 的多個互驅週期：經濟週期 / 信用週期 / 心理週期 / 風險態度週期——四輪同步時信號最強
- Dalio 短期債務週期（5-8年）：信用擴張→繁榮→泡沫→去槓桿→衰退→新一輪信用擴張
- Dalio 長期債務週期（75-100年）：長期債務積累到極限觸發系統性去槓桿（如 1929、2008）
- Kondratiev 長波（40-60年）：技術革命驅動的超長週期，現處於資訊/AI 波浪的中後段
- 週期定位的實作：不是預測頂底，而是判斷「我們大概在哪個象限」然後調整組合偏好
- 反直覺：週期不是時鐘，不能用時間預測；但可以用估值 + 情緒 + 信用條件綜合定位

## 維度交互與組合信號
- 總經轉差 + 流動性收緊 + 估值偏高 = 三紅燈，減倉信號
- 情緒極度悲觀 + 估值便宜 + 趨勢初轉 = 三綠燈，建倉機會
- 任何單一維度孤立判斷都可能誤導，七維度的共振才是高信心信號

- 荒謬.外部環境盲點 → 總經判讀與週期定位的起點
- 風險矩陣 Skill → 風險意識維度的量化工具
- 投資軍師團 Skill（investment-masters）→ 七芒星七個維度的對標大師各有 Skill 對應
- DARWIN 策略模擬 → 總經 + 流動性數據的實際來源
