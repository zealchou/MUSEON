# 六種情緒弧線詳解與範例（Emotion Arcs）

> **來源**：Kurt Vonnegut 情緒弧線理論 + UVM 大數據驗證研究
> **用途**：storytelling-engine 深度參考——情緒節奏設計
> **建立日期**：2026-03-23

---

## 核心主張

所有故事都可以用「情緒曲線」來描述——幸運到不幸的走勢圖。UVM 的研究用大數據分析驗證，發現絕大多數成功的故事都落在六種基本弧線中。

---

## 六種基本弧線

### 1. 白手起家（Rags to Riches）

**走勢**：持續上升
**情緒曲線**：
```
幸運 ↑    _____________
         /
        /
       /
      /
─────+──────────────────→
不幸 ↓
```

**特質**：從低點穩定上升到高點，沒有重大逆轉
**經典範例**：灰姑娘（前半段）、大部分勵志傳記
**適合用於**：勵志故事、成長敘事、「我如何從零開始」
**五幕對應**：特別適合第二幕（探索）到第四幕（行動）的漸進累積

**SSA 應用**：客戶的轉變故事——從「不知所措」到「漸入佳境」

---

### 2. 悲劇（Tragedy / Riches to Rags）

**走勢**：持續下降
**情緒曲線**：
```
幸運 ↑  \
         \
          \
           \
            \_____________
─────+──────────────────→
不幸 ↓
```

**特質**：從高點持續滑落，結尾在低點
**經典範例**：安娜卡列尼娜、馬克白
**適合用於**：警世故事、風險提醒、「如果不改變會怎樣」
**五幕對應**：特別適合 StoryBrand 元素 6（避免失敗）

**SSA 應用**：描述「不行動的代價」——讓客戶看見不改變的下場

---

### 3. 掉洞爬起（Man in a Hole）

**走勢**：下降 → 上升
**情緒曲線**：
```
幸運 ↑ \                 /
        \               /
         \             /
          \___________/
─────+──────────────────→
不幸 ↓
```

**特質**：角色陷入困境，然後努力爬出來。最普遍的故事結構
**經典範例**：幾乎所有喜劇、大部分好萊塢電影
**適合用於**：問題解決敘事、客戶案例、「遇到問題 → 找到方法 → 解決」
**五幕對應**：完美對應五幕劇情——建立（正常）→ 探索（掉進去）→ 願景（看到出路）→ 行動（爬出來）→ 承諾（新常態）

**SSA 應用**：最常用的客戶成功故事結構——他遇到什麼問題、我們怎麼一起解決、現在變成什麼樣

---

### 4. 伊卡洛斯（Icarus）

**走勢**：上升 → 下降
**情緒曲線**：
```
幸運 ↑        ___
             /   \
            /     \
           /       \
          /         \___
─────+──────────────────→
不幸 ↓
```

**特質**：飛得太高，然後墜落。傲慢的代價
**經典範例**：華爾街之狼、社群網戰
**適合用於**：反面案例、過度自信的警示、市場泡沫敘事
**五幕對應**：特別適合第三幕（願景）到第四幕（行動）的逆轉

**SSA 應用**：描述「看起來很好但有風險」的情境，提醒客戶謹慎

---

### 5. 灰姑娘（Cinderella）

**走勢**：上升 → 下降 → 上升
**情緒曲線**：
```
幸運 ↑        ___                 ___
             /   \               /
            /     \             /
           /       \           /
──────+────         \_________/  ──→
不幸 ↓
```

**特質**：得到好運 → 失去一切 → 最終逆轉。最受歡迎的弧線
**經典範例**：灰姑娘完整版、新約聖經
**適合用於**：品牌起源故事、創辦人故事、逆境重生
**五幕對應**：最完整的情緒體驗——五幕都有充分的素材

**SSA 應用**：最適合長篇品牌敘事——創辦人的初心（上升）→ 遭遇挫敗（下降）→ 帶著智慧重新出發（再上升）

---

### 6. 俄狄浦斯（Oedipus）

**走勢**：下降 → 上升 → 下降
**情緒曲線**：
```
幸運 ↑
                  ___
                 /   \
──────+─\       /     \________  ──→
         \     /
          \___/
不幸 ↓
```

**特質**：陷入困境 → 找到希望 → 希望破滅。複雜且令人心碎
**經典範例**：星際大戰前傳三部曲、安納金的墮落
**適合用於**：複雜角色描述、深度人物刻畫、黑暗轉折
**五幕對應**：特別挑戰第四幕（行動）的設計——行動不一定帶來好結果

**SSA 應用**：較少用於銷售，但適合描述「曾經以為找到答案但失敗」→ 引出「這一次不同」的轉折

## 弧線組合策略

研究顯示，最受歡迎的故事往往是基本弧線的組合：

| 組合 | 說明 | 效果 |
|------|------|------|
| 兩個「掉洞爬起」 | 第一次爬起後又掉進更深的洞 | 雙重逆轉，張力加倍 |
| 灰姑娘 + 悲劇結尾 | 逆轉重生後又失去 | 心碎但深刻 |
| 白手起家 + 伊卡洛斯 | 持續上升後驟然墜落 | 最大反差 |
| 掉洞爬起 + 灰姑娘 | 解決問題後又面臨更大挑戰 | 英雄旅程的經典節奏 |

---

## 選擇弧線的決策樹

```
你的故事想讓聽眾感受什麼？
│
├─ 希望與激勵 → 白手起家 或 灰姑娘
│
├─ 警惕與反思 → 悲劇 或 伊卡洛斯
│
├─ 共鳴與轉變 → 掉洞爬起（最通用）
│
└─ 複雜與深度 → 俄狄浦斯 或 弧線組合
```

## 與 SSA 能量的對應

| 弧線段落 | 對應能量 | 說明 |
|---------|---------|------|
| 上升段 | 火 × 天 | 方向感 + 格局感 |
| 下降段 | 水 × 地 | 承接痛苦 + 穩住安全感 |
| 最低點 | 地 × 水 | 陪伴 + 同理 |
| 逆轉點 | 火 | 照亮新方向 |
| 最高點 | 天 × 澤 | 格局 + 認同 |
| 結尾（行動）| 雷 × 地 | 推進 + 穩定 |
