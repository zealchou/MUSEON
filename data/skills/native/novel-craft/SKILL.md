# 小說工藝引擎 Novel Craft Engine

> **好的文字不是寫出來的，是削出來的。**

## 外掛合約

此 Skill 為 DNA27 核心的外掛模組（pluggable plus），同時是 text-alchemy 生態系的文字工藝專家，以及 c15 的深度技法庫。

**依賴**：
- `dna27` skill（母體 AI OS）
- `text-alchemy` skill（語言層路由，可被其調用）

**本模組職責**：
- 提供文字工藝診斷與設計
- 整合六大寫作大師的核心技法
- 支援感官寫作與節奏控制
- 提供沉浸式敘事的具體技術
- 作為 c15 的技法擴充庫

**本模組不做**：
- 不做文字任務路由（那是 text-alchemy 的工作）
- 不做語言張力原則制定（那是 c15 的工作）
- 不做敘事結構設計（那是 storytelling-engine 的工作）
- 不做邏輯結構框架（那是 consultant-communication 的工作）
- 不覆寫 DNA27 的迴圈/模式/護欄判斷
- 不替使用者做最終創作選擇

## 核心原理：七大工藝法則

所有寫作大師都認同的核心法則：

| # | 法則 | 來源 | 應用 |
|---|-----|-----|-----|
| 1 | **省略比書寫更有力** | Hemingway | 寫下的只是冰山一角，7/8 在水下 |
| 2 | **展示，不說明** | 全體大師 | 用行動、細節、感官取代形容詞 |
| 3 | **節奏即情緒** | 文學傳統 | 長句=舒緩，短句=緊張，斷裂=衝擊 |
| 4 | **細節必須開槍** | Chekhov | 每個細節都要有後續意義 |
| 5 | **空白即畫布** | 極簡主義 | 留白讓讀者參與創作 |
| 6 | **日常即魔幻** | 村上春樹 | 在平凡中發現不平凡 |
| 7 | **視覺先於語言** | 電影劇本 | 先看見，再書寫 |

## 六大寫作大師技法庫

### 1. 冰山理論 Iceberg Theory（Ernest Hemingway）

**核心主張**：省略你知道的，讀者會感受到水面下的重量

**創作公式**：
```
你知道的全部（100%）
    ↓ 刻意省略
寫在紙上的（12.5%）
    ↓ 讀者推斷
讀者感受到的（100%+α）
```

**三大技術**：

| 技術 | 說明 | 範例 |
|-----|-----|-----|
| **刪除背景** | 不解釋角色過去，讓行為說話 | ❌「他因為童年創傷而害怕黑暗」→ ✅「他走進房間，先打開所有的燈」 |
| **刪除情緒** | 不說「他很傷心」，用動作取代 | ❌「她很緊張」→ ✅「她的手指在桌面上敲出不規則的節奏」 |
| **刪除結論** | 讓讀者自己得出結論 | ❌「他們的婚姻已經走到盡頭」→ ✅「她把咖啡杯移到桌子的另一端」 |

**Hemingway 修訂法**：
> 「我寫一頁傑作，配上九十一頁垃圾。我努力把垃圾丟進垃圾桶。」

---

### 2. 極簡主義 Minimalism（Raymond Carver）

**核心主張**：在最少的文字中，容納最大的情感

**風格標記**：
- 簡短、直述的句子
- 省略的對話（elliptical dialogue）
- 藍領階級的日常生活
- 結尾不給答案

**對話技巧——「說過頭」技法**：
```
角色 A 說的
    ↘
      ≠ 角色 A 真正想說的
    ↗
角色 B 理解的

（三者永遠錯位，這就是人性）
```

**Carver 的極簡清單**：
| 刪除 | 保留 |
|-----|-----|
| 華麗形容詞 | 精準動詞 |
| 內心獨白 | 外在行動 |
| 解釋說明 | 具體細節 |
| 圓滿結局 | 開放結尾 |
| 戲劇化衝突 | 日常張力 |

**經典示範**：
> ❌ 「房間裡瀰漫著一種令人窒息的沉默，兩個人都不知道該說什麼。」
> 
> ✅ 「她翻動雜誌。他看著窗外。冰塊在杯子裡融化。」

---

### 3. 潛台詞與細節 Subtext & Detail（Anton Chekhov）

**核心主張**：真正重要的從不直說

**契訶夫之槍（Chekhov's Gun）**：
> 「如果第一幕掛了一把槍，第三幕它必須開火。」

**應用法則**：
1. 每個細節都要有後續呼應
2. 無用的細節會削弱故事
3. 重要的事物要早早埋下

**潛台詞三層結構**：

```
表面對話（Surface）
    ↓
角色真正想說的（Subtext）
    ↓
作者真正想傳達的（Theme）
```

**細節選擇原則**：

| 類型 | 說明 | 範例 |
|-----|-----|-----|
| **預示細節** | 暗示未來發展 | 「她總是把鑰匙放在固定的地方」（暗示將失去秩序） |
| **性格細節** | 揭示人物本質 | 「他數找零時會多數一遍」（暗示不信任） |
| **氛圍細節** | 建立情緒基調 | 「牆角的仙人掌已經三週沒澆水」（暗示忽視） |
| **象徵細節** | 承載更大意義 | 「她一直盯著窗外那棵被砍掉一半的樹」 |

---

### 4. 氛圍營造 Atmosphere（村上春樹 Haruki Murakami）

**核心主張**：在日常中發現超現實，在孤獨中找到連結

**村上風格標記**：
- 極簡但充滿細節的日常描寫
- 孤獨但不悲傷的語調
- 音樂與食物的具體引用
- 現實與夢境的邊界模糊

**日常細節法**：
```
不說「他很孤獨」
而是寫：
「他煮了一人份的義大利麵，
 把收音機調到古典樂頻道，
 聽著蕭邦的夜曲，
 在空蕩蕩的公寓裡用叉子捲起麵條。」
```

**氛圍建構公式**：

| 元素 | 功能 | 範例 |
|-----|-----|-----|
| **時間** | 定錨感受 | 「下午三點十五分」比「下午」更有真實感 |
| **空間** | 投射心理 | 「房間裡只有一張椅子面對窗戶」 |
| **聲音** | 填補寂靜 | 「冰箱的嗡嗡聲成了唯一的陪伴」 |
| **氣味** | 喚醒記憶 | 「咖啡的香氣讓他想起那個雨天」 |
| **物件** | 承載情感 | 「那本書的書籤還停在第 47 頁」 |

**魔幻日常轉換**：
> 從「他在等公車」
> 到「他已經等了四十分鐘，公車站牌上的數字似乎永遠停在同一個時間。他開始懷疑這個世界是否在某個他不知道的瞬間，悄悄地停止了運轉。」

### 5. 感官寫作 Sensory Writing

**核心主張**：讓讀者用身體閱讀，而不只是用眼睛

**五感+三感 完整清單**：

| 感官 | 傳統描述 | 深層應用 |
|-----|---------|---------|
| **視覺** | 顏色、光影、形狀 | 光的質感、視線的方向、看不見的暗示 |
| **聽覺** | 聲音、對話、音樂 | 沉默的質感、聲音的距離、聽不見的聲音 |
| **嗅覺** | 氣味、香氣、臭味 | 氣味引發的記憶、消失的氣味 |
| **味覺** | 味道、口感 | 情緒的味道（苦澀的勝利） |
| **觸覺** | 溫度、質地、痛感 | 內在的觸感（胃的收縮、心跳） |
| **本體感** | 身體位置、肌肉張力 | 「他的肩膀不自覺地縮了起來」 |
| **平衡感** | 重心、暈眩 | 「世界似乎傾斜了幾度」 |
| **內臟感** | 飢餓、噁心、心悸 | 「一陣反胃從胃底升起」 |

**感官疊加技術**：
```
單一感官：「她聽到門鈴響。」

感官疊加：
「門鈴響的時候，
 她正聞到隔壁飄來的咖啡香（嗅覺），
 腳底感覺到木地板的涼意（觸覺），
 窗外的光線正好照在她的手背上（視覺），
 形成一個完美的三角形。」
```

**感官錨點法**：
> 選擇一個主導感官，其他感官圍繞它展開
> 
> 範例：以「聽覺」為錨點
> 「她聽見了自己的呼吸聲（錨點）。太響了。整個房間似乎都在放大這個聲音。她看見窗戶上自己的倒影（視覺輔助），模糊得像一個正在消失的人。」

### 6. 電影感寫作 Cinematic Writing

**核心主張**：像導演一樣思考，像攝影師一樣構圖

**劇本思維三原則**：

| 原則 | 說明 | 應用 |
|-----|-----|-----|
| **視覺優先** | 先問「觀眾看到什麼」 | 用畫面說故事，不用旁白 |
| **動作說話** | 行動 > 對話 > 內心 | 「他摔門出去」比「他很生氣」有力 |
| **剪接思維** | 場景之間的跳接 | 用空白行模擬鏡頭切換 |

**鏡頭語言轉文字**：

| 鏡頭類型 | 對應文字技法 | 範例 |
|---------|------------|-----|
| **遠景** | 環境總覽 | 「城市在暮色中變成了一片橙紅色的剪影。」 |
| **中景** | 人物與環境 | 「他站在月台上，列車呼嘯而過。」 |
| **特寫** | 細節放大 | 「她的指甲在桌面上留下一道淺淺的刮痕。」 |
| **主觀鏡頭** | 第一人稱感知 | 「他的視線落在那封信上，字跡模糊了。」 |
| **慢動作** | 延長時間感 | 「杯子傾斜。水流出來。沿著桌緣。一滴。一滴。」 |

**場景轉換技術**：
```
硬切（Hard Cut）：
「她關上門。
 
 陽光刺眼。他已經在機場了。」

疊化（Dissolve）：
「她的手指在琴鍵上停下來。
 那個音符在空氣中顫抖，
 慢慢變成了窗外的雨聲。」

跳接（Jump Cut）：
「八點。
 九點。
 十點。
 門沒有響。」
```

## 節奏控制術 Rhythm Control

### 句子長度與情緒

| 句長 | 情緒效果 | 適用場景 |
|-----|---------|---------|
| **極短句**（1-5字）| 衝擊、緊張、斷裂 | 高潮、轉折、震驚 |
| **短句**（6-15字）| 緊湊、直接、清晰 | 動作場景、對話 |
| **中句**（16-30字）| 平穩、敘述、過渡 | 一般敘事、描寫 |
| **長句**（30字以上）| 舒緩、沉思、流動 | 氛圍營造、內心獨白 |

### 節奏組合模式

**Staccato（斷奏）**：
```
他停下來。
看。
聽。
什麼都沒有。
```

**Legato（連奏）**：
```
月光從窗簾的縫隙流進來，在地板上畫出一道細長的銀線，
像是某種神秘的指引，通往一個他不確定是否存在的出口。
```

**混合節奏**：
```
她等了很久。
等到咖啡涼了，等到窗外的光線從金黃變成灰藍，
等到街燈一盞一盞亮起來，像是城市正在緩慢地睜開眼睛。
然後她站起來。
穿上外套。
走出門。
```

### 段落呼吸

**短段落**（1-2句）：
- 製造緊張感
- 強調重要訊息
- 模擬心跳加速

**長段落**（5句以上）：
- 建立沉浸氛圍
- 允許讀者「住進」場景
- 適合描寫與內省

**單句成段**：
```
她終於明白了。

那不是愛。
```

## 標點符號的藝術

| 標點 | 功能 | 效果 |
|-----|-----|-----|
| **。** | 完結 | 確定、肯定、結束 |
| **，** | 停頓 | 喘息、連接、節奏 |
| **——** | 插入/轉折 | 思緒跳躍、補充說明 |
| **……** | 未完 | 懸念、猶豫、消逝 |
| **？** | 疑問 | 不確定、質疑、邀請 |
| **！** | 強調 | 衝擊、驚訝（慎用） |
| **：** | 引出 | 期待、解釋 |
| **「」** | 對話/引用 | 聲音、他者的話語 |

**標點節奏範例**：
```
「你確定嗎？」
「我……」她停下來。外面有人在按喇叭。「我不知道。」
「好。」他站起來。「那就——」
門開了。
```

## 工作模式

### 模式一：文字診斷（/craft diagnose）

**輸入**：使用者提供的文字段落

**輸出**：
1. **七法則檢核**：對照七大工藝法則，標記問題點
2. **技法識別**：識別已使用的技法與缺失的技法
3. **節奏分析**：句長分布、段落呼吸狀態
4. **感官清點**：五感覆蓋率，主導感官識別
5. **優化建議**：具體的改寫方向與示範

**診斷問句清單**：
- 哪些句子是「說明」而不是「展示」？
- 哪些形容詞可以用動作取代？
- 讀者需要自己推斷什麼？還是你都說完了？
- 節奏是否太單調？需要更多變化嗎？
- 感官描寫夠具體嗎？還是太抽象？

---

### 模式二：場景建構（/craft build）

**輸入**：
- 場景的核心情緒（例如：孤獨、緊張、釋懷）
- 關鍵元素（人物、地點、物件）
- 篇幅限制（選填）

**輸出**：
1. **感官設計**：為場景選擇主導感官與輔助感官
2. **節奏藍圖**：段落結構與句長變化計劃
3. **細節清單**：契訶夫之槍——每個細節的後續意義
4. **省略清單**：刻意不說的事（冰山水下部分）
5. **完整場景**：成品段落

**建構步驟**：
```
Step 1: 定義核心情緒（這段文字要讓讀者感受什麼？）
Step 2: 選擇感官錨點（用哪個感官引導讀者進入？）
Step 3: 設計開場畫面（讀者「看到」的第一個畫面是什麼？）
Step 4: 埋下細節種子（哪些細節會在後面「開槍」？）
Step 5: 決定省略什麼（什麼不說，讓讀者自己感受？）
Step 6: 規劃節奏曲線（句長變化、段落呼吸）
Step 7: 寫作與修訂
```

## 快速工具箱

### 開場技術

| 類型 | 說明 | 範例 |
|-----|-----|-----|
| **感官開場** | 用具體感官拉讀者進入 | 「咖啡涼了。她沒有注意到。」 |
| **動作開場** | 用動作製造好奇 | 「他把信撕成兩半，然後又撕成四半。」 |
| **對話開場** | 用對話製造懸念 | 「『你不應該來的。』她沒有回頭。」 |
| **時間開場** | 用具體時間定錨 | 「凌晨三點四十七分，電話響了。」 |
| **反常開場** | 用違反預期製造張力 | 「葬禮結束後，她笑了。」 |

### 轉場技術

| 技術 | 效果 | 範例 |
|-----|-----|-----|
| **感官轉場** | 用感官連接兩個場景 | 「雨聲漸遠。當他再次睜開眼，陽光已經穿過窗簾。」 |
| **物件轉場** | 用同一物件穿越時空 | 「她把戒指放進抽屜。二十年後，她的女兒打開了它。」 |
| **空白轉場** | 用留白製造跳躍 | 三個空行，然後新場景 |
| **聲音轉場** | 用聲音連接 | 「門關上的聲音還在迴盪。然後是電話鈴聲。」 |

### 結尾技術

| 類型 | 說明 | 範例 |
|-----|-----|-----|
| **開放結尾** | 不給答案，留給讀者 | 「她站在門口。進去，還是離開？風吹過來，帶著雨的氣息。」 |
| **迴圈結尾** | 回到開頭的元素 | 開頭的咖啡杯，結尾她終於喝了一口 |
| **細節開槍** | 早埋的細節在此刻爆發 | 第一幕的槍，終於開火 |
| **感官餘韻** | 用感官留下餘味 | 「門關上了。走廊裡只剩下她香水的氣息，慢慢變淡。」 |

## 護欄

### 硬閘

**HG-NC-PLAGIARISM**：拒絕抄襲特定作家的整段文字。技法可學，文字要原創。

**HG-NC-HARM**：拒絕用技巧美化傷害性內容。好文字不是武器。

**HG-NC-MANIPULATION**：拒絕製造虛假的情感操控。真實是工藝的基礎。

### 軟閘

**SG-NC-OVERCRAFT**：提醒技巧過度會變成炫技。「少即是多」永遠是第一原則。

**SG-NC-OBSCURE**：提醒省略過多會變成晦澀。讀者需要足夠的線索。

**SG-NC-FANCY**：提醒節奏花俏不等於好文字。服務故事，不是服務技巧。

## 範例：同一場景的六種改寫

**原始版本（平鋪直述）**：
> 她很緊張地等著那個人來。她聽到了門鈴聲，然後聞到了咖啡的香味。她想那可能是昨晚留字條的人。

**Hemingway 版（省略法）**：
> 門鈴響了。
> 
> 她沒有動。

**Carver 版（極簡對話）**：
> 「有人嗎？」
> 
> 她把杯子放下。
> 
> 「我來了。」

**Chekhov 版（細節預示）**：
> 昨晚那張字條還在桌上，角落被咖啡漬染黃了。門鈴響的時候，她正盯著那個漬痕看。

**村上版（日常魔幻）**：
> 下午三點十五分，她已經等了四十七分鐘。收音機裡的爵士樂剛好播完，換成一段讓人不安的靜音。然後門鈴響了——剛好在靜音的正中央。

**電影感版（鏡頭思維）**：
> 風。
> 
> 窗簾飄起來。她的視線從窗外收回，落在門上。
> 
> 鈴聲。
> 
> 走廊裡飄來咖啡的香氣。腳步聲。
> 
> 她站起來。

**感官疊加版**：
> 她聽見風穿過門縫的聲音，然後是鈴聲——尖銳，短促，像是被掐斷的音符。咖啡的香氣隨著腳步聲飄進來，混著雨後泥土的濕氣。她的手心開始出汗。那張字條的紙角戳著她的口袋。
