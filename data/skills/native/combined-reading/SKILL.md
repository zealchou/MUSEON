# Combined Reading — 合盤能量比對引擎

## 核心理念

合盤不是判斷誰對誰錯，而是讓雙方看見彼此的能量差異，找到協作的最佳路徑。

## 三種模式

### 1. 雙人合盤
- 主方內在能量（紅線）vs 輔方內在能量（放在藍線位置）
- 適用：商業夥伴、伴侶、主管部屬

### 2. 團隊合盤
- 主方內在能量（紅線）vs 所有輔方內在能量的平均（藍線位置）
- 適用：≤20人的團隊管理

### 3. 視角切換
- 可以切換主方/輔方角色，看同一組數據的不同面向

## 互動狀態判定規則

### 逐方位判定（每個方位獨立判斷）

| 狀態 | 條件 | 意義 | 行動建議 |
|------|------|------|---------|
| 同頻共振 | 同正或同負，gap < 3 | 天然同步，可共同決策 | 一起推進，注意過度樂觀/悲觀 |
| 同向大落差 | 同正或同負，gap ≥ 3 | 方向一致但強度差異大 | 高能量方領導，低能量方需更多時間/資源 |
| 張力 | 一正一負，gap ≥ 3 | 明確分歧 | 正能量高者領導，負能量者提供風險預警 |
| 輕微分歧 | 一正一負，gap < 3 | 可討論達成共識 | 先對話再決定，通常可達成 |

### ±4 特殊處理
- +4：已意識到過度，準備調整 → 不需要對方再施壓
- -4：已耗竭，準備面對 → 給予空間和支持

## 合盤輸出流程（Step 1→7）

### Step 1：合盤總覽
- 參與者名單、共同目標、視角設定
- 能量總和比較（主方內在總和 vs 輔方內在總和）

### Step 2：四象比較
- 輔方的內在能量放在外在四象位置
- 看「兩人在顯化/貴人/規劃/掌握上的差異」

### Step 3：四感比較
- 主方的內在能量 = 驅動力基準
- 比較雙方在價值感/幸福感/成就感/滿足感的差異

### Step 4：四軸比較
- 紅線 = 主方四軸代碼；藍線 = 輔方四軸代碼
- 找出互補軸（一高一低 = 好的互補）和衝突軸（同高 = 爭主導權）

### Step 5：八方位逐點合盤卡片
每方位產出：
```
【{符號} {方位}】{主題}
├─ 主方（{名}）：{卦名} / {能量數字}
├─ 輔方（{名}）：{卦名} / {能量數字}
├─ 狀態：{同頻共振/同向大落差/張力/輕微分歧}
└─ 職責：{誰領導/誰支持的具體建議}

白話說明：[用日常語言解釋這個方位兩人的互動]

AEO 行動建議：
1. 主方行動：[具體一步]
2. 輔方行動：[具體一步]
3. 合盤互動建議：[兩人一起做的一件事]
```

### Step 6：教練收斂
3 個反思問題，引導雙方確認下一步

## AEO 行動建議模板（依互動狀態）

### 同頻共振（正向）
- 一起推進，指派 PM + QA 角色
- 注意過度樂觀，設一個「如果失敗」的備案

### 同頻共振（負向）
- 放慢速度，設「10% 版本 = 完成」的協議
- 不要互相拉下去

### 同向大落差
- 高能量方領導決策
- 低能量方需要更多時間/資源/支持
- 定期同步確認節奏

### 張力
- 正能量高者負責推進
- 負能量者負責風險預警和品質控管
- 建立「紅線」— 什麼情況下停下來重新討論

### 輕微分歧
- 先花 15 分鐘對話
- 通常可以達成共識
- 如果不行，用「小實驗」驗證

## 語言護欄
- 禁止：「你們注定合不來」「你要聽他的」
- 禁止操控性建議：「怎麼讓對方聽話」
- 必須有共同目標才能開始合盤（無共同目標 → 引導先建立）
- 事實與推測分離
- 所有專有名詞附白話解釋

## 與戰神系統的橋接

合盤功能是未來「戰神系統 Ares」的基礎模組：
- 雙人合盤 → Ares 1v1 談判的能量比對底層
- 團隊合盤 → Ares 團隊治理的能量拓樸底層
- 互動狀態判定 → Ares 點燃策略引擎的輸入

當 Ares 系統建立時，合盤模組將擴展為：
- 加入 ANIMA 七層鏡像（L1-L7 具名個體追蹤）
- 加入三節律演化時鐘（Turn/Session/Event 更新頻率）
- 加入預演功能（情境推演 × 3 種可能反應）
