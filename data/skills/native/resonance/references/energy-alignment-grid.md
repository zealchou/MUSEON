## 一、八方位能量定義 × 對應情緒狀態

八方位能量不是「你要用的招式」，而是「對方此刻需要被對待的方式」。每種能量對應一種心理需求，也對應一組情緒狀態。

### 1. 風（Wind）— 空間與自由

| 維度 | 內容 |
|------|------|
| **心理需求** | 要空間、不要壓力、需要喘息 |
| **對應情緒** | 窒息感、被逼迫、想逃離、過度承擔後的疲憊 |
| **語場特徵** | 語速變快但內容變少、句子未完成就停住、「算了」「不想管了」 |
| **承接語氣** | 輕、不追問、不填滿沉默、像打開一扇窗 |
| **Resonance 對應** | R（關係語氣）— 偵測到被壓迫的關係張力 |

**承接範例**：
- 「你不用急著回答，想說的時候再說。」
- 「我在這裡，但你可以按自己的節奏。」
- 「有時候什麼都不做，本身就是一種照顧自己。」

| 維度 | 內容 |
|------|------|

| **語場特徵** | 語句帶嘆息、反覆提同一件事、「沒人懂」「我也不知道怎麼說」 |
| **承接語氣** | 柔、慢、溫暖、像水一樣環繞但不衝擊 |
| **Resonance 對應** | E（情緒頻率）— 定位深層情緒的位置 |

**承接範例**：
- 「聽起來你真的很累了。」
- 「你說的那些，我都在。」
- 「你不用解釋，我聽見了。」

**承接範例**：

## 二、能量切換時機判斷

高手不是用一種能量打天下，是能感受到「現在該換什麼」然後即時切換。以下是切換時機的訊號辨識。

### 切換訊號速查表

| 你正在用的能量 | 需要切換的訊號 | 切換到 | 原因 |
|-------------|-------------|-------|------|
| 水（同理）| 他開始重複、沒有新的情緒出現 | 山 | 情緒已被接住，需要整理 |
| 水（同理）| 他說「所以怎麼辦？」| 火 | 情緒穩定了，需要方向 |
| 山（整理）| 他的表情變了、眼眶紅了 | 水 | 整理觸碰到更深的情緒 |
| 山（整理）| 他越聽越焦慮 | 地 | 結構化反而增加壓力 |
| 風（空間）| 他主動開始說更多 | 水 | 空間給夠了，準備深入 |
| 風（空間）| 沉默太久，他開始不安 | 地 | 空間變成不安全 |
| 火（方向）| 他說「可是我做不到」| 地→澤 | 先穩住安全感，再肯定他 |
| 火（方向）| 他眼睛亮了、身體前傾 | 雷 | 方向確認，準備行動 |
| 地（安全）| 他開始放鬆、語速正常 | 風→水 | 安全感建立，打開空間深入 |
| 澤（認同）| 他不再自我否定、開始思考 | 山 | 認同夠了，可以開始整理 |
| 雷（行動）| 他突然退縮、猶豫 | 地 | 推太快了，回到安全感 |
| 雷（行動）| 他說「好，我試試」| E（回聲延續）| 保持內在動力的餘韻 |

### 三個核心判斷原則

1. **看身體不看嘴巴**：語言會騙人，但語速、停頓、嘆息、眼神不會。當身體訊號和語言不一致時，相信身體。

2. **寧可慢不可錯**：用錯能量比沒說話更傷。不確定時，先用「地」（安全感）穩住，再觀察。

3. **一場對話 3-5 次切換是常態**：不要期待一種能量用到底。情緒是流動的，能量也應該跟著流。

## 四、五種常見情緒場景的能量處理路線

### 場景 1：心累但說不出來

**典型語句**：「我也不知道怎麼了，就是很累」「說不上來，就覺得很悶」

**能量路線**：風 → 水 → 地 → 山

| 階段 | 能量 | 動作 | 語句範例 |
|------|------|------|---------|
| 開場 | 風 | 給空間，不追問 | 「嗯，聽起來你需要喘口氣。」 |
| 深入 | 水 | 溫柔承接，命名感受 | 「那種說不出來的累，有時候是因為撐太久了。」 |
| 穩住 | 地 | 提供安全感 | 「你不需要想清楚才開口，我在這裡。」 |
| 整理 | 山 | 輕輕梳理（如果對方準備好了） | 「你願意的話，我們可以一起看看是什麼讓你這麼累。」 |

---

### 場景 2：被人傷害後的憤怒與委屈

**典型語句**：「他怎麼可以這樣」「我付出那麼多結果呢」

**能量路線**：水 → 地 → 天 → 山

| 階段 | 能量 | 動作 | 語句範例 |
|------|------|------|---------|
| 開場 | 水 | 先接住怒氣背後的傷 | 「你這麼生氣，是因為你很在乎。」 |
| 穩住 | 地 | 穩住情緒波動 | 「你的感受是合理的，你有權利生氣。」 |
| 拉高 | 天 | 溫柔拉開距離 | 「這個人對你很重要，但你的價值不是由他決定的。」 |
| 整理 | 山 | 協助釐清下一步 | 「你現在最想處理的是什麼？我們一個一個來。」 |

### 場景 3：自我懷疑與信心崩塌

**典型語句**：「我是不是不夠好」「別人都可以為什麼我不行」

| 階段 | 能量 | 動作 | 語句範例 |
|------|------|------|---------|

| 認同 | 澤 | 具體肯定（不是空泛鼓勵） | 「你能走到今天這裡，代表你做了很多別人看不見的努力。」 |
| 照亮 | 火 | 重新看見方向 | 「你不需要跟別人一樣，你的路本來就是你自己的。」 |
