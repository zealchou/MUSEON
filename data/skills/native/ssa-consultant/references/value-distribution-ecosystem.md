# 價值分配與共創生態系

> Zeal 原創的組織動力與價值分配方法論。
> 來源：SSA-I 學員手冊 Chapter 3（SSA-I.19 ~ I.27）+ PCT 績效教練

---

## 核心命題

企業的穩定不是靠制度，而是靠共振。
價值如果分配錯，就會崩解——人留不住、客戶不續約、合作不穩定。
這不是管理問題，而是設計問題。

---

## 一、價值分配核心邏輯

### 三原則
1. **公平（Fairness）**：每個人都應清楚自己貢獻什麼、得到什麼
2. **激勵（Incentive）**：分配機制要能推動行動，而非抑制行動
3. **共榮（Resonance）**：當利益與使命一致，團隊才會長期共振

### 傳統 vs SSA 的分配觀
- 傳統：用「薪資」分配，獎勵「出現時間」，無法獎勵「創造價值」
- SSA：進化成「價值回饋系統」，讓個人、團隊、顧客與合作夥伴，都能根據實際貢獻得到合理回報

### 指標
| 指標 | 理想值 |
|------|--------|
| 貢獻對應率 | ≥80%（團隊成員是否清楚自己被回饋的依據） |
| 內部信任度 | ≥8/10（匿名調查「我覺得分配是公平的」） |
| 留任率 | 追蹤（制度導入後三個月內團隊留任比例） |
| 自主提案率 | 追蹤（成員主動提出改進或新專案的次數） |

## 二、One Muse 內隱四感（The Four Inner Feelings）

### 定義
組織能量平衡的心理模型。制度可以讓公司穩定，但唯有人能讓公司溫暖。
當人願意留下，不是因為制度設計得好，而是因為在這個系統裡，他「感覺自己被需要、被看見、被實現」。

### 四感
1. **價值感（Value Sense）**：我做的事有意義，他人會對我感恩
2. **幸福感（Joy Sense）**：我在這裡快樂、有安全感，身旁的同事都會即時的協助我
3. **成就感（Pride Sense）**：我能看到自己的進步與影響力
4. **滿足感（Peace Sense）**：我能即時得到付出應得的回報，可能是錢、掌聲、舞台

### 應用
- 四感平衡 → 團隊自發穩定運轉
- 任一感長期缺失 → 組織出現能量流失
- 領導者的任務不是激勵，而是**校準能量感受**

### 四感檢測表（每月/每季一次）
每位成員在回顧會中回答四個問題：
1. 這個月你覺得自己的工作有價值嗎？（價值感）
2. 你對團隊氣氛的滿意程度如何？（幸福感）
3. 你對自己這個月的表現有多滿意？（成就感）
4. 你覺得生活與工作的平衡度如何？（滿足感）

管理層將分數低於 7 的部分納入「能量對談」。

### 指標
| 指標 | 理想值 |
|------|--------|
| 四感平均分 | ≥8 |
| 能量平衡度 | 四感最高與最低分差距 <2 |
| 留任意願 | ≥80%（匿名「我願意在這裡再待一年」） |
| 自發行動率 | 追蹤（成員主動發起活動或改進提案的次數） |

## 三、遊戲化動力設計（八角框架應用）

### 定義
Yu-kai Chou 的 Octalysis 框架在 SSA 中的應用。遊戲化的關鍵不是做遊戲，而是讓人感覺正在玩一個有意義的任務。

### 八核心驅動力
1. **天：使命與意義（Epic Meaning）**——我在做有價值的事
2. **風：稀缺與焦慮（Scarcity & Impatience）**——我想要但還沒得到
3. **水：社交影響（Social Influence）**——我能與他人連結、比較、協作
4. **山：損失恐懼（Loss & Avoidance）**——我不想錯過或失去
5. **地：所有權與占有（Ownership & Possession）**——這是我的成果，我想保護它
6. **雷：不可預測與驚喜（Unpredictability & Curiosity）**——我期待接下來會發生什麼
7. **火：進步與成就（Development & Accomplishment）**——我能看見自己的成長
8. **澤：創造與回饋（Empowerment of Creativity & Feedback）**——我能發揮創意並獲得回應

### 指標
| 指標 | 意義 |
|------|------|
| 任務參與率 | 被遊戲化的活動參與人數 / 總人數 |
| 完成率 | 任務完成次數 / 目標次數 |
| 自主提案率 | 成員主動設計挑戰或活動的比例 |
| 能量分布圖 | 以四感為橫軸、八角動力為縱軸的觀察矩陣 |

## 四、三重螺旋理論（Triple Helix Theory）

### 定義
一個模式賣三次——但不是「多賣」，而是讓同一個價值在不同角色身上重複兌現。

### 三層角色
1. **使用者（User）**：以產品體驗價值
2. **志工（Volunteer）**：透過參與或回饋，成為品牌的協助者
3. **共創者（Co-creator）**：以協作放大價值

### 升級路徑設計
每一層角色之間應有明確的「升級條件」與「回饋方式」，讓顧客知道——下一步能怎麼參與、怎麼被看見、怎麼被回饋。

### SSA 實例
- 使用者層：學員參與 SSA-I 課程，體驗系統思維的轉化力量
- 志工層：畢業學員成為「班導教練」，協助新學員上課、整理共學資料
- 共創者層：志工進一步參與 SSA-II 或 SSA-III，設計教材、開發分潤課程，甚至帶領新一屆學員

結果：SSA 平均課程續班率超過 80%，五成以上新學員來自舊生推薦。

### 指標
| 指標 | 理想值 |
|------|--------|
| 志工轉化率 | 10-20%（從使用者升級為志工的比例） |
| 共創參與度 | 追蹤（共創者參與新專案或課程開發的次數） |
| 志工續任率 | 追蹤（完成任期後選擇繼續擔任志工的比例） |
| 回饋循環時間 | 追蹤（從顧客體驗到再次參與的平均週期） |

## 五、高毛利分潤模型（5%-50%）

### 三級分潤
- **5%**：象徵啟動獎勵——讓人願意推
- **15-25%**：常態合作夥伴分潤——讓人願意長期參與
- **50%**：共創或聯合開發的高毛利專案——讓人願意共創未來

### 核心原則
1. **貢獻對應**：分潤比例依實際影響與價值流位置決定
2. **槓桿驅動**：高毛利產品應有更大分潤空間，用於激發推動者
3. **閉環回饋**：每次分潤都要能導回系統、促進下一輪行動

### 指標
| 指標 | 理想值 |
|------|--------|
| 分潤覆蓋率 | ≥80%（有被設計分潤機制的產品比例） |
| 合作推廣率 | 追蹤（由外部夥伴帶來的成交佔比） |
| 分潤回饋週期 | 越短越有動能 |
| 分潤重啟率 | 追蹤（領過分潤後再次參與專案的比例） |

## 六、PCT 控制論閉環

### 定義
源自行為心理學與系統理論的思考模型。核心主張：行為的目的，是為了控制感知，而非回應命令。

### 三環節
1. **感知（Perception）**：我看見了什麼？
2. **行動（Action）**：我做了什麼？
3. **回饋（Feedback）**：我的行動改變了什麼？

### 企業應用
老闆不必控制人，只要設計感知與回饋的機制，團隊就能自己調整。
以控制代替命令、以回饋代替監督。

### 每週三問（PCT 健康閉環）
1. 這週，我在哪個時刻最有能量？
2. 這週，我在哪個時刻最消耗？
3. 下週，我想保留 / 放下 / 新增什麼？

### 指標
| 指標 | 理想值 |
|------|--------|
| 回饋週期穩定度 | 90%（回饋會議是否按節奏執行） |
| 主動修正率 | 追蹤（非主管指令下，團隊主動提出修正方案） |
| 行動閉環率 | ≥70%（每次回饋後實際執行修正的比例） |
| 系統穩定指數 | 追蹤（目標達成波動幅度下降率） |

## 七、PCT 績效教練方法論

### 教練/講師/顧問三角色
- **講師**：針對無經驗者提供知識和經驗分享
- **教練**：協助有一定經驗者發展高績效的個人方法
- **顧問**：提供行業經驗和成功關鍵因素的具體指導
- 教練的任務是挑戰個案的知識含量，幫助其清晰下一步行動

### X 模型
1. 現況（現狀）
2. 目標（理想狀態）
3. 落差（現況與目標的差距）
4. 解決方案（用來縮小落差的方法）
5. 資源（內部和外部資源）

### 目標設定四層次
1. **使命**：抽象且感覺性的目標
2. **終極目標**：一生中希望達成的目標
3. **階段性目標**：分階段達成的目標
4. **行動計劃**：具體的行動步驟

### 過度承諾與時間管理
- 過度承諾的根因：評估完成任務時間時忽略了其他人生目標和承諾
- 解方：明明知道十天到十五天可以完成的任務，應該承諾三十天
- 人生五大面向的承諾要同時納入時間規劃
- 時間是最公平的資源，每個人每天都有相同的 24 小時

### 槓桿思維
- 先學槓桿加點，再學工具，最後增強自身能力
- 三層資源：可見資源 → 關係資源 → 認知資源
- 槓桿加點的效果大於工具，工具大於能力
