# 三重漏斗系統（Triple Funnel System）

> Zeal 原創商業引擎。三個漏斗不獨立存在，而是「並聯＋互饋」。
> 來源：SSA-I 學員手冊 Chapter 2（SSA-I.09 ~ I.18）

---

## 核心理念

產品產生體驗 → 行銷放大故事 → 社群強化信任 → 再導回產品。
當三者形成迴圈，企業就從一次銷售進化為持續流動的價值生態。
整合的關鍵不是畫圖，而是「對齊節奏」——產品節奏（推出與更新）× 行銷節奏（敘事與活動）× 社群節奏（互動與再啟動），三者若同頻共振，價值流就會穩定。

---

## 一、產品漏斗（Product Funnel）

### 定義
企業將產品按「體驗 → 核心 → 高毛利」的邏輯階層排列，讓顧客能循序漸進地從初次接觸走向深度合作。

### 三層結構
1. **體驗產品**（低門檻、低風險）—— 讓顧客先嘗試。建立初步信任。
2. **核心產品**（主要價值來源）—— 提供關鍵解決方案。兌現價值。
3. **高毛利產品**（深度價值延伸）—— 創造共創與長期合作。

### 核心洞見
- 三層不只是價格梯度，更是**信任梯度**
- 體驗建立信任、核心兌現價值、高毛利轉化為關係
- 當企業能讓顧客在漏斗中「向上移動」時，品牌就不需要不斷開發新客，而是能靠信任累積成長

### 延伸指標
| 指標 | 理想值 | 意義 |
|------|--------|------|
| 體驗轉換率 | ≥20% | 體驗產品升階至核心產品的比例 |
| 核心留存率 | ≥60% | 核心產品使用或續約的顧客比例 |
| 高毛利轉化率 | 10-20% | 從核心進入高毛利產品的比例 |
| 全漏斗續約率 | 追蹤 | 顧客從第一接觸到第三層留存的平均週期 |

### 實戰案例
健康管理中心：體驗產品（免費身體年齡測驗）→ 核心產品（年度健康計畫）→ 高毛利產品（專屬教練長期追蹤與 VIP 社群）。一年內平均客單價提升 2.5 倍，續約率達 78%。

## 二、行銷漏斗（Marketing Funnel）

### 定義
行銷是價值流的語言管道。它讓理念被看見、被理解、被轉化為行動。

### 三層結構
1. **理念層**（Awareness）—— 啟動信任，回答「為什麼要聽你說」
2. **案例層**（Engagement）—— 建立共鳴，回答「這和我有關嗎」
3. **行動層**（Conversion）—— 促發決策，回答「我現在該怎麼做」

### 核心洞見
- 人類決策順序：**感覺 → 理解 → 行動**
- 多數企業的問題是直接從「行動」開始（折扣、推廣），卻忽略顧客的情緒節奏
- 行銷漏斗的任務是讓這三步變得可設計

### 語言三步節奏
每一則行銷內容都應包含：
1. **共感句（Empathy Line）**：讓顧客感覺「你懂我」
2. **對焦句（Focus Line）**：幫顧客看見問題的核心
3. **召喚句（Action Line）**：提供可行的下一步

範例：
> 共感：「我知道你已經嘗試很多方法了」
> 對焦：「但真正讓事業穩定的關鍵不在努力，而在結構」
> 召喚：「加入三週挑戰營，我帶你把結構搭起來」

### 延伸指標
| 指標 | 意義 |
|------|------|
| 理念層互動率 | 觀看或閱讀理念型內容的平均停留時間 |
| 案例層轉化率 | 閱讀案例後點擊「了解更多」的比例 |
| 行動層成交率 | 從行動呼籲（CTA）進入實際購買的比例 |
| 語言一致性 | 三層內容中的語氣與價值觀是否保持統一 |

### 實戰案例
顧問公司重組行銷語言：理念層（「創業不該靠意志力，而是靠結構」）→ 案例層（三位學員轉變案例）→ 行動層（「現在報名體驗班」）。廣告點擊率下降 20%，但報名轉換率提升 2.4 倍，顧客平均停留時間延長 3 倍。

## 三、社群漏斗（Community Funnel）

### 定義
社群不只是顧客的集合，而是價值循環的第二個引擎。行銷能引流，但社群才能留人。

### 三階段
1. **沉澱（Retention）**：讓顧客留下來。提供歸屬感、儀式與認同語。
2. **餵養（Nurturing）**：讓顧客活起來。定期輸入內容、學習、互助。
3. **再啟動（Reactivation）**：讓價值重生。透過活動、任務或共創，讓社群重新流動。

### 核心洞見
- 多數社群經營失敗的原因，是只做「活動」而沒設計「節奏」
- 一個成熟的社群，是品牌的延伸器官——它能自我調節、互相回饋、持續生長
- 社群不再是行銷工具，而是一個價值生態

### 社群黏性設計三核心
1. **行動目標**：明確知道該做什麼
2. **反饋迴路**：完成後能被看見、被回應
3. **情感節奏**：讓任務不只是工作，而是遊戲

### 延伸指標
| 指標 | 理想值 | 意義 |
|------|--------|------|
| 活躍率 | 追蹤 | 每週社群內有互動（留言或按讚）的人數比例 |
| 留存率 | ≥80% | 社群成員三個月後仍在群內的比例 |
| 再啟動率 | 追蹤 | 過去沉默三十天以上的顧客重新互動的比例 |
| 共創參與度 | 追蹤 | 顧客自發內容（UGC）佔整體貼文的比例 |

### 實戰案例
果飲品牌「月訂會員圈」：沉澱（果飲卡＋品牌故事）→ 餵養（每週果飲小知識＋挑戰任務「七日補水行動」）→ 再啟動（每月線下「共創口味日」）。六個月後月訂續約率達 82%，社群內自發貼文數成長三倍。

## 四、三漏斗整合實作

### 整合結構
三重漏斗由三個子系統組成：
1. 產品漏斗（Product Funnel）—— 定義顧客的價值階梯
2. 行銷漏斗（Marketing Funnel）—— 設計理念與語言的流動
3. 社群漏斗（Community Funnel）—— 建立信任與關係的循環

### 月節奏循環範例
- 第一週：推廣理念內容（行銷）
- 第二週：開啟體驗課（產品）
- 第三週：進行社群共學（社群）
- 第四週：收斂成果並預告下一期

### 整合指標
| 指標 | 理想值 | 意義 |
|------|--------|------|
| 三漏斗連動率 | ≥70% | 產品、行銷、社群間是否能互導顧客 |
| 循環週期長度 | 追蹤 | 顧客從初次接觸到再次互動的平均週期 |
| 回購率 / 轉介紹率 | 追蹤 | 反映整合系統健康度 |
| 整合投資報酬率 | 追蹤 | 整體投入與價值放大的比值 |

## 五、漏斗 × 顧客旅程對應

將三重漏斗縱向串接於顧客的橫向旅程，形成動態「價值流路線圖」：

| 旅程階段 | 對應漏斗 | 任務 |
|---------|---------|------|
| 1. 認知（Awareness） | 行銷漏斗・理念層 | 讓人看見你 |
| 2. 興趣（Consideration） | 行銷漏斗・案例層 | 讓人感覺被理解 |
| 3. 體驗（Experience） | 產品漏斗・體驗產品 | 建立初步信任 |
| 4. 承諾（Conversion） | 產品漏斗・核心產品 | 兌現價值 |
| 5. 共創（Advocacy） | 社群漏斗 | 顧客成為夥伴，進入循環 |

漏斗讓你有結構，旅程讓你有溫度；兩者結合，品牌才能有生命。

## 六、異業共創與槓桿設計

### 理想的異業槓桿關係三條件
1. **理念契合**：雙方價值觀相通
2. **受眾互補**：顧客旅程可串接
3. **利益共榮**：成果共享、風險共擔

### 資源盤點三層
1. **可見資源（Visible Assets）**：產品、數據、通路
2. **關係資源（Relational Assets）**：可借用的信任網絡與合作關係
3. **認知資源（Cognitive Assets）**：品牌聲譽、理念共鳴與社群影響力

### 指標
| 指標 | 理想值 |
|------|--------|
| 合作槓桿比 | ≥3（每投入一單位資源，產生三單位價值） |
| 品牌共振度 | 合作品牌的關鍵字重疊度 |
| 顧客交叉滲透率 | 合作期間雙方顧客互導的比例 |
| 合作續航率 | 合作後雙方是否願意延長或擴大範圍 |

## 七、自我運轉的三要素

### 節奏（Rhythm）
市場不是要你快，而是要你可預期。穩定就是力量。

### 重力（Gravity）
讓人願意留下。靠的不是薪水、不是獎金、不是合約，而是讓每個參與者——員工、夥伴、顧客——都能在這個系統裡感覺到自己是「一部分」。

### 共振（Resonance）
一個會共振的品牌，永遠不缺資源。因為它懂得「回饋」。
