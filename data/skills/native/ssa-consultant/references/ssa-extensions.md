# SSA 延伸系統：關係（SSA-R）與健康（SSA-H）

> SSA 不只是商業系統，它是一套可以用在人生每個面向的結構。
> Zeal 將 SSA 的核心邏輯（覺察→對齊→微調→顯化 + PCT 閉環）延伸到親密關係與身心健康。
> 來源：SSA-I 學員手冊附錄

## 一、SSA-R｜讓關係重新運作的系統

### 核心理念
關係的問題，多半不是因為不愛，而是沒有結構。
用清楚的結構讓關係重新運作。用覺察代替情緒，用節奏取代意氣。

### 四步驟
1. **覺察**：看見我們的互動模式（對話怎麼開始、怎麼結束？衝突時我通常進攻還是逃避？）
2. **對齊**：讓價值觀與節奏對上（不代表要一樣，而是知道不同，願意協調）
3. **微調**：在日常裡測試與修正（每天一點一點修正，像調音器一樣）
4. **顯化**：讓關係變成能被看見的成果（一起完成的作品、一次旅行、一場有意義的對話）

### 關係八方位
每一段關係像一個八邊形，有八種力量在互相影響：

| 面向 | 問自己 |
|------|--------|
| 天：共同方向 | 我們想要走向什麼樣的未來？ |
| 風：溝通流動 | 我們能自在地說真話嗎？ |
| 水：情感深度 | 我們願意讓對方看到脆弱嗎？ |
| 山：界線與尊重 | 我知道我不願意的是什麼嗎？ |
| 地：日常照顧 | 我們的生活安排穩定嗎？ |
| 雷：行動節奏 | 我們的步調合得來嗎？ |
| 火：熱情與創造 | 我們還有共同的靈感與新鮮感嗎？ |
| 澤：欣賞與感謝 | 我們多久表達一次「謝謝」或「我喜歡」？ |

### 關係健康三指標
1. **安全感**：我能否自在地說真話？
2. **信任感**：對方做不到時，是否仍相信他在努力？
3. **回復力**：衝突後多久能重新連線？

### 關係卡住時五步驟
1. 先冷卻，不評論（深呼吸三次）
2. 說觀察，不說情緒（「你昨天沒回訊息」比「你都不理我」更容易溝通）
3. 講自己的感受（「我有點不安」比「你讓我不安」更不容易引起防衛）
4. 明確提出請求（「希望下次能提前說忙碌」比「你以後可不可以多體貼一點」更有效）
5. 確認對方理解（「你聽起來覺得我太急，是嗎？」讓對話有閉環）

### 不同關係的應用
- **伴侶關係**：穩定節奏 + 安全溝通（每週一次修復會、每月一起做一件小專案）
- **朋友關係**：分清楚「我能幫的」和「我想幫的」
- **團隊與工作關係**：定期彼此回饋，把界線和責任寫清楚
- **家庭關係**：把關心具體化，安排固定的家庭儀式

### 九十天關係更新計畫
- 第 1 週：寫下關係中讓你最感謝的三件事
- 第 2 週：一起定一個「對齊規則」
- 第 3-4 週：開第一次修復會，練習說觀察與感受
- 第 2 個月：挑一個共同任務一起執行
- 第 3 個月：製作一份「我們的說明書」

## 二、SSA-H｜健康不是狀態，是迴圈

### 核心理念
健康不是靜態的結果，其實是一個持續能修復的系統。
只要能量在流動，生命就能自我調整。當流動卡住，那就是系統出現了訊號延遲。

### 氣血水三重循環

| 層次 | 對應元素 | 功能 | 常見失衡 | 修復方式 |
|------|---------|------|---------|---------|
| 氣態循環 | 呼吸、姿態、情緒 | 決定能量的輸入與釋放 | 呼吸短、焦慮、胸悶 | 深呼吸、伸展、冥想 |
| 液態循環 | 血液、水分、睡眠 | 負責運輸營養與代謝廢物 | 手腳冰冷、疲倦、失眠 | 溫水泡腳、規律睡眠 |
| 固態循環 | 食物、肌肉、行動 | 決定身體的結構與力量 | 久坐、無力、腸胃脹氣 | 慢食、運動、充足蛋白 |

三者互為漏斗：氣帶動血，血滋養水，水承載氣。
其中任何一個堵塞，其他兩個也會受影響。

### 日週月三層節奏

**每日節奏（氣態層）**
- 晨間三步：呼吸三次、喝溫水、伸展三分鐘
- 中午一停：觀察飢餓與疲倦，不硬撐
- 夜間一收：睡前不滑手機、深呼吸十次

**每週節奏（液態層）**
- 固定運動兩次
- 安排一次「靜態活動」：散步、泡澡、閱讀
- 記錄一件「讓我覺得輕鬆的事」

**每月節奏（固態層）**
- 整理一次居家或工作環境
- 檢查飲食比例：蔬菜、蛋白質、水分
- 訂立一個「身體學習任務」：學按摩、學煮湯、學呼吸法

### 八方位健康盤

| 方位 | 身體對應 | 問自己一句話 | 修復行動 |
|------|---------|----------|---------|
| 天 | 呼吸/頭部 | 我還記得自己為何活著嗎？ | 晨間呼吸 + 設定願景 |
| 風 | 肺與喉嚨 | 我有說出真實的話嗎？ | 每天說一次「我想要…」 |
| 水 | 腎與膀胱 | 我的恐懼藏在哪裡？ | 泡腳、靜坐、寫情緒日記 |
| 山 | 脊椎與骨骼 | 我站得直嗎？ | 拉伸背部、練核心 |
| 地 | 胃與脾 | 我有好好吃飯嗎？ | 慢食、規律進食 |
| 雷 | 四肢與筋膜 | 我有在行動嗎？ | 每天走路 20 分鐘 |
| 火 | 心與眼 | 我有熱情嗎？ | 晨光曝曬、少滑螢幕 |
| 澤 | 皮膚與表情 | 我有笑嗎？ | 感謝三件事、微笑練習 |

### PCT 健康閉環
每週問自己三個問題：
1. 這週，我在哪個時刻最有能量？
2. 這週，我在哪個時刻最消耗？
3. 下週，我想保留 / 放下 / 新增什麼？

### 九十天身心重啟計畫
| 週期 | 主題 | 具體任務 |
|------|------|---------|
| 第 1-4 週 | 觀察期 | 每天記錄情緒、睡眠、飢餓指標 |
| 第 5-8 週 | 微調期 | 調整一項飲食、一項作息 |
| 第 9-12 週 | 穩定期 | 建立固定運動與晚間關機儀式 |

### 健康的完整公式
健康 =（氣 × 血 × 水）×（覺察 → 對齊 → 微調 → 顯化）

## 三、後記七層反思——達哥的起源故事

### 第一層：覺察——看見自己的人生商模
格子店打拼三年，白天談合約，晚上補貨，凌晨核帳。
直到有一天靜下來看自己的生活，發現自己其實是一間「失衡的公司」。

### 第二層：對齊——從品牌三角到人生三角
品牌三角：理念 × 產品 × 顧客認知。
人生三角：信念 × 行為 × 關係。
當不再只用頭腦經營，而是用覺察連結，關係就開始回溫。

### 第三層：結構——三重漏斗與人生的閉環
產品漏斗 = 每天在產出的價值
行銷漏斗 = 如何讓這些價值被看見、被理解
社群漏斗 = 如何讓能量循環——支持別人，也讓自己被支持

### 第四層：能量——學會調頻而不是燃盡
火與雷太旺，差點燒光自己。花了好幾年才學會水的柔軟、地的穩定。

### 第五層：控制——從反應到運行的自我修正
PCT 教會的最重要的事：所有行為的目的，都不是改變世界，而是穩定自己。

### 第六層：收束——創業是創己
SSA 教的從來不是創業，而是創己。因為你創的每一間公司，最後都只是一個放大你的鏡子。

### 第七層：邀請——你的故事也能被重寫
人生的每個環節，其實都能用結構修復。
