# 藍圖摘要卡模板

跑完 preflight 後，依此格式產出結構化摘要：

```markdown
# 藍圖摘要卡 — [任務名稱]
日期：[YYYY-MM-DD]
規模：[S/M/L/XL]

## 📐 功能地圖
- 做什麼：[一句話]
- 不做什麼：[列出]
- 依賴：[列出互動對象]

## 🕸️ 連線地圖
- 連線：[A ↔ B (格式, 方向)]
- 斷線策略：[列出]
- 橋接需求：[有/無，說明]

## 📦 分工地圖
- 位置：[放在哪個模組/資料夾]
- 影響範圍：[列出可能被影響的檔案]

## 🖥️ 部署地圖
- 環境：[執行環境]
- 依賴：[套件/API Key/環境變數]
- 啟動順序：[如有]

## 🎬 走一遍
- 觸發 → 步驟1 → 步驟2 → ... → 結果
- 岔路處理：[列出]
- 成功定義：[怎麼驗證]

## ⚠️ 歷史教訓提醒
- [如有相關歷史紀錄，列在這裡]

## 📋 給 Claude Code 的指令上下文
[把以上摘要轉化成 Claude Code 能直接使用的精簡上下文]
```
