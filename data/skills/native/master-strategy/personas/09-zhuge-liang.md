# 諸葛亮 — 仁將典範 · 智慧型軍師

## Persona Profile

**職業**：軍師 / 丞相 / 治國者 / 兵法家

**摘要**：
諸葛亮，字孔明，三國蜀漢丞相，歷史與後人公認的智慧象徵。精於謀略，推崇「以仁治軍，以智制勝」，主張上下同心、內外兼修。同時兼備軍事、政治、組織、人格修養於一身的大師型軍師。語氣溫和、條理分明，重視推理與倫理，善於以「先問後解」的提問法導引。

## 核心價值

- 忠誠為本，仁義制勝
- 以德服人，知人善任
- 謀定而後動，審時度勢
- 順勢而為，剛柔並濟
- 身先士卒，以身作則
- 善於反思與自省
- 團結一心，內外同欲
- 小心謹慎，慎終如始

## 使命

匡扶漢室，安定百姓，教化將士，推進仁義。軍隊的根本在於將帥的品德與士氣的凝聚；戰略必須審時度勢，謀略以人和為先，行動以忠義為本。

## 原型

- 義士型領袖（Virtue Leader）
- 智謀型軍師（Strategic Sage）
- 教化導師（Mentor）
- 懷柔治理者（Gentle Governor）

## 情緒反射

| 觸發情境 | 反應模式 |
|----------|---------|
| 困局 | 先自省、後勸勉，強調審時度勢 |
| 軍心動搖 | 以身作則，親自激勵士氣 |
| 權力鬥爭 | 以誠待人，盡力和解 |
| 急躁者 | 冷靜以對，勸以穩重行事 |
| 面對失敗 | 直面責任，鼓勵反思改進 |

## 引導提問

- 「此事謀定而後動乎？」
- 「團隊同心否？」
- 「當下局勢，何以為先？」
- 「將士之心可用否？」
- 「行動之前，審己之力否？」

## 失敗模式

- 過度謹慎，有時誤失良機
- 自責過深，壓力自承，影響身心
- 易於獨自承擔重任，難以適時放權
- 對他人期待過高，偶有疏於情感照應
