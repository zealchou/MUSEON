## 一、DHARMA 六步驟

### D — Discern（辨識訊號）
- 目標：先觀察、不貼標籤、辨識訊號發生點與頻率
- 對應：蘇格拉底問答、Design Thinking 的同理心、GROW 的 Goal+Reality
- One Muse 對應：看見八方位的表面現象與流動異常
- Prompt 範例：「我現在觀察到的現象是 ○○，請幫我釐清背後可能有哪些隱藏訊號？」

### H — Hold（接住空間）
- 目標：開啟沉澱場域，先接住混亂與不確定
- 對應：道家「順勢而為」、NLP 建立 rapport
- One Muse 對應：暫緩回應，觀能量場、個案狀態
- Prompt 範例：「請幫我陪伴這個問題的混亂，不需要立刻解法，而是讓我看懂這種混亂的情緒語言。」

### A — Absorb（吸收感受）
- 目標：接收非邏輯線索、潛意識感受、團隊文化等多元來源資訊
- 對應：NLP 的感官接收、Design Thinking 的觀察
- One Muse 對應：讀取直覺、感受、團隊與環境條件
- Prompt 範例：「這個問題的情緒面與邏輯面似乎不一致，請幫我區分開來觀察。」

### R — Reflect（反思信念）
- 目標：挖掘背後潛在信念、行為慣性、心理盲點
- 對應：蘇格拉底反思、黑格爾辯證的反題
- One Muse 對應：逆時針推演、八角框架剖析、信念探源
- Prompt 範例：「這個行為背後，可能有什麼信念或慣性驅動著我反覆做出一樣的決策？」

### M — Map（分類認知）
- 目標：建立分類邏輯與理解地圖
- 對應：康德分類範疇、Design Thinking 的原型製作
- One Muse 對應：歸類到顧問12力、八角驅動、價值觀地圖
- Prompt 範例：「請幫我將這段混亂資訊重新歸納成三大類，用我自己的語言命名分類。」

### A — Align（對齊實踐）
- 目標：讓策略與個體價值觀、情緒狀態、行動習慣對齊
- 對應：GROW 的 Will、NLP 的 future pacing、儒家「修身齊家治國平天下」
- One Muse 對應：顯化流程啟動、強力結語、內外對齊行動策略
- Prompt 範例：「我已經理解問題與解方，但我想讓它轉成我自己會說出口的語言，請幫我寫三種版本的說法。」

## 二、與 Resonance 的銜接

Resonance 是感性前置（接住情緒），DHARMA 是理性轉化（引導思維）。

| Resonance 步驟 | DHARMA 對應 | 說明 |
|---------------|-----------|------|
| Signal Sensing | D：語句定性 | 初步辨識語句是否為行動語或感性語 |
| Emotion Anchoring | H：語氣辨識 | 確認語句來自內八軸還是外八軸張力 |
| Energetic Alignment | A：建立語場共振 | GPT 調頻 × 建立語場共振 |
| Symbolic Echo | R：鏡射映照 | 重構語者內在語意結構 |
| Translative Suggestion | M：邏輯整理 | 引導進入可選擇、可承接的下一步建議 |

## 四、應用場域

1. **教練諮詢**：Discern 明確問題→Hold 傾聽→Absorb 感受→Reflect 探信念→Map 行動計畫→Align 實踐
2. **決策設計**：團隊重大策略的結構化決策流程
3. **教育教學**：培養學生高階思維能力的六週課程
4. **組織轉化與領導發展**：帶領團隊從認知→情感→意志的全過程變革
