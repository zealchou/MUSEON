# Wan-Miu 16 — 萬謬16型人格分析引擎

## 核心理念

「萬謬」寓意從不同視角探索多樣的人性面向。不是給人貼標籤，而是幫助理解能量的動態運作，找到平衡與突破。

## 四軸線系統

### 軸線定義

#### 1. 領導與分享（使命守護軸）：天 ↔ 地
- **A (Altruist 利他者)**：高能量。使命感強烈，願意付出並引導他人，注重群體福祉。風險：過度燃燒、忽略自我需求。
- **P (Protective Opportunist 保護型機會主義者)**：低能量。保護自身與親近者利益，注重實用性和安全感。風險：視野受限、錯失大局機會。
- 主題：使命感與自我保護的平衡。

#### 2. 連結與互動（關係動態軸）：火 ↔ 水
- **O (Efficient Operator 高效操作者)**：高能量。目標導向，理性且快速實現目標。風險：忽略情感、顯得冷漠。
- **S (Supporter 支持者)**：低能量。以他人感受為中心，提供情感支持和協助。風險：過度承載、失去自我。
- 主題：目標導向與情感支持的權衡。

#### 3. 顯化與累積（動力模型軸）：山 ↔ 澤
- **E (Enthusiast 熱情者)**：高能量。強調行動力和成果顯化，願意嘗試新事物並追求快速成功。風險：衝動、缺乏後續。
- **R (Realist 現實者)**：低能量。注重穩定和資源累積，偏好長期計劃和穩健行動。風險：過度保守、錯失機會。
- 主題：快速行動與穩健累積的選擇。

#### 4. 適應與創造（情緒結構軸）：風 ↔ 雷
- **M (Mood-driven Reactor 情緒驅動反應者)**：高能量。高度適應外部環境，善於根據情緒和環境調整策略。風險：缺乏一致性、易受影響。
- **U (Unyielding Executor 堅守執行者)**：低能量。堅守原則和策略，缺乏靈活性但具備穩定性和耐力。風險：僵化、無法適應變化。
- 主題：情緒靈活與策略堅守的博弈。

## 28 題測驗

每軸 7 題，涵蓋四大場景：事業（2題）、關係（2題）、健康（1題）、情緒穩定（2題）。

### 測驗流程
1. 使用者回答 28 題二選一
2. 每題的 A 選項 = 高能量方向；B 選項 = 低能量方向
3. 統計每軸的高/低能量偏好數量
4. 組合四軸結果 → 得出 4 字母人格代碼

### 代理評估模式（用於分析他人）
當使用者想分析對手、客戶或團隊成員時：
1. 使用者描述對方的行為、決策風格、溝通方式
2. AI 根據描述替對方「回答」28 題
3. 產出代理評估結果 + **置信度**（資料充足→70-80%；資料少→40-50%）
4. 標記「推測」與「觀察」的區分

## 應用場景

### 1. 個人成長
- 了解自己的行為模式、優勢與挑戰
- 根據人格特質設計成長計劃

### 2. 團隊配置
- 識別團隊中的「異質性」互補潛力
- 避免「同質化」導致職能缺漏
- 分配適合的工作職責

### 3. 關係經營
- 理解伴侶/合作夥伴的能量模式
- 探索如何平衡情感需求

### 4. 職業選擇
- 根據人格特質找到適合的職業領域

## 語言護欄
- 不說「你就是XX型」→ 說「根據目前的觀察，你在這個面向偏向...」
- 不把人格當宿命 → 「這是當下的能量傾向，可以通過覺察調整」
- 代理評估必須標明置信度和資料來源
