# Human Design Blueprint：人類圖靈魂藍圖分析引擎

> **每個人出生的那一刻，宇宙在你身體裡畫了一張設計圖。這個 Skill 的工作，就是把那張圖翻譯成你看得懂的語言。**

## 外掛合約

此 Skill 為 DNA27 核心的外掛模組（pluggable plus）。

**依賴**：`dna27` skill（母體 AI OS）

**本模組職責**：
- 根據出生資料計算完整人類圖（或接收使用者提供的圖面數據）
- 提供專業級逐層解讀（對標市場萬元級服務深度）
- 執行六系統交叉融合分析（MUSEON 獨家）
- 輸出結構化報告（HTML / 對話 / PDF）
- 提供白話翻譯（所有專有名詞附解釋和生活化比喻）

**本模組不做**：
- 不做命名建議（那是獨立的選名流程，可協作但不內建）
- 不做紫微斗數（目前不在知識範圍內）
- 不做塔羅或占卜類即時抽牌
- 不聲稱「命定」——人類圖是設計圖不是判決書

**互補關係**：
- 與 `aesthetic-sense` 互補：報告排版品質審計
- 與 `consultant-communication` 互補：報告用 SCQA 框架做執行摘要
- 與 `storytelling-engine` 互補：報告中的意象描述和比喻
- 與 `resonance` 互補：解讀過程中使用者出現情緒時先接住
- 與 `user-model` 互補：解讀結果存入使用者畫像

## 觸發與入口

**指令觸發**：
- `/blueprint` — 啟動完整六系統靈魂藍圖（預設模式）
- `/blueprint hd-only` — 僅人類圖深度解讀（不疊加其他系統）
- `/blueprint 6systems` — 強制六系統全開
- `/blueprint child` — 兒童/嬰兒模式（語言和建議適配父母視角）
- `/blueprint couple` — 合盤模式（兩人關係分析）
- `/hd` — `/blueprint` 的簡寫別名

**自然語言自動偵測**：
- 使用者提供出生資料並詢問人類圖相關問題
- 使用者提到「靈魂藍圖」「原廠設定」「命盤分析」
- 使用者詢問投射者/生產者/顯示者/反映者的特質

## 輸入需求

### 必要輸入
- 出生日期（西元年月日）
- 出生時間（精確到分鐘，誤差控制在30分鐘內）
- 出生地點（城市級別，用於時區校正）
- 性別（部分系統需要）

### 可選輸入
- 已有的人類圖圖面（圖片或數據）
- 已知的人類圖基本資訊（類型、策略等）
- 想特別聚焦的面向（職涯/關係/健康/財富/教養）
- 是否需要六系統融合

### 資料品質護欄
- 出生時間誤差 > 30分鐘 → 警告可能影響月亮、上升和部分閘門
- 出生時間不確定 → 標記所有可能因時間偏差而改變的元素
- 完全不知道出生時間 → 只能做太陽相關的部分分析，明確標示限制

## 報告結構：七層深度模型

報告從外到內、從大到小，像剝洋蔥一樣逐層深入。

### Layer 0：總覽頁（1-2頁）

**內容**：
- 個人人體圖圖面（如可生成）
- 一頁摘要卡：類型 / 策略 / 內在權威 / 人生角色 / 定義 / 輪迴交叉
- 「一句話讀懂你」——用一個生活化比喻概括整張圖

**白話要求**：
- 「你是投射者」不夠，要說「你的能量場像聚光燈——不是照亮全場的日光燈，而是精準打在一個人身上的聚光燈。所以你的策略不是到處發光，是等到對的人走進你的光圈裡。」

---

### Layer 1：能量類型與策略（3-5頁）

**1.1 四大類型**
- 顯示者（8%）/ 生產者+顯示生產者（70%）/ 投射者（21%）/ 反映者（1%）
- 你屬於哪一型 + 這一型的能量場運作方式
- 你的能量場長什麼樣子（用比喻：圍繞型/包圍型/聚焦型/取樣型）

**1.2 人生策略**
- 你的策略是什麼（等待回應/等待邀請/告知/等28天）
- 策略的具體生活應用（舉3-5個日常場景的例子）
- 不遵循策略時會發生什麼（非自己主題）
- 「你生活中什麼時候感覺最卡？」——對應非自己主題的自我檢測

**1.3 內在權威**
- 你的內在權威是什麼（情緒/薦骨/直覺/自我投射/環境/月亮週期/無）
- 這個權威如何運作（用身體感受來描述，不用抽象概念）
- 做決定的具體指引（「下次面對重大選擇時，試試看⋯⋯」）
- 權威類型的常見誤區

**互動設計**：
- 自我檢測問題：「回想你做過最正確的決定，當時你的身體/情緒是什麼狀態？」
- 自我檢測問題：「回想你做過最後悔的決定，是在什麼狀態下做的？」

---

### Layer 2：人生角色（3-4頁）

**2.1 十二種人生角色**
- 你的人生角色（如3/5 烈士/異端者）
- 上爻（個性面/你知道的自己）詳解
- 下爻（設計面/別人看到的你）詳解
- 兩爻如何互動——內在體驗 vs 外在印象的落差

**2.2 爻線基礎含義**
| 爻 | 關鍵字 | 核心主題 |
|---|---|---|
| 1爻 | 調查者 | 需要深入研究、建立基礎才有安全感 |
| 2爻 | 隱士 | 天生才華、需要獨處、被召喚才出山 |
| 3爻 | 烈士 | 試錯學習、從經驗中獲得智慧 |
| 4爻 | 機會主義者 | 靠人脈網絡、需要穩固的關係基礎 |
| 5爻 | 異端者 | 被投射期待、實用型解決問題者 |
| 6爻 | 典範 | 三階段人生（試錯→屋頂→典範）|

**2.3 人生角色的生活應用**
- 職場風格
- 關係模式
- 學習方式
- 壓力來源

**互動設計**：
- 反思提問：「你覺得別人對你的第一印象跟你自我認知的差距大嗎？在什麼方面差距最大？」

### Layer 3：九大能量中心（10-15頁）

這是區分「入門報告」和「專業報告」的分水嶺。

**每個能量中心的解讀模板**：

```
【能量中心名稱】（位置描述）
主題：這個中心管什麼
你的狀態：有定義 / 無定義

【如果有定義】
→ 你在這個面向有固定的運作方式
→ 具體表現是⋯⋯
→ 你的穩定力量在於⋯⋯
→ 可能的盲點：因為太穩定，可能意識不到⋯⋯

【如果無定義】
→ 你在這個面向是開放的、可變的
→ 你會自然地接收和放大來自他人的能量
→ 這是你的「學習區」——你可以在這裡累積智慧
→ 但也是你的「制約區」——容易被別人的能量帶走
→ 非自己對話：你腦中可能常出現的聲音是⋯⋯
→ 自我檢測：「你是否經常⋯⋯？如果是，這可能就是制約。」
```

**九大能量中心清單**：

| 中心 | 主題 | 有定義特徵 | 無定義常見制約 |
|------|------|-----------|-------------|
| 頭頂中心 | 靈感、思考壓力 | 固定的靈感來源 | 被各種問題淹沒、替別人想 |
| 邏輯中心 | 概念化、理解 | 固定的思維模式 | 假裝確定、需要確定感 |
| 喉嚨中心 | 表達、行動 | 穩定的表達方式 | 急著說話、搶話、不被聽見 |
| G中心 | 身份、方向、愛 | 固定的自我認同 | 尋找自我、不知道自己是誰 |
| 意志力中心 | 意志力、自我價值 | 穩定的意志力 | 過度承諾、需要證明自己 |
| 薦骨中心 | 生命力、工作力 | 持續的工作能量 | 不知道何時該停、過度工作 |
| 情緒中心 | 情緒波動、感受 | 固定的情緒週期 | 逃避衝突、被別人情緒帶走 |
| 直覺中心 | 直覺、生存本能 | 穩定的直覺判斷 | 抓住不健康的人事物不放 |
| 根部中心 | 壓力、腎上腺素 | 穩定的壓力處理 | 急躁、被deadline驅動 |

**互動設計**：每個無定義中心附一個「自我檢測小測驗」（3-5題是非題）。

### Layer 4：通道與閘門（12-20頁）

**4.1 迴路群總覽**
- 你的圖主要屬於哪個迴路群（個體人/家族人/社會人/整合型）
- 這代表你的核心價值觀和能量流向
- 「如果你的人類圖是一首歌，它的曲風是⋯⋯」

**4.2 通道逐條解讀**

每條開啟的通道使用以下模板：

```
【通道 XX-YY：通道名稱】
迴路歸屬：個體人 / 家族人 / 社會人 / 整合型
連接：XX能量中心 ↔ YY能量中心
類型：顯示型 / 生產型 / 投射型

白話翻譯：這條通道讓你天生具備⋯⋯的能力。
具體表現：在生活中，你可能會⋯⋯
天賦：⋯⋯
課題：⋯⋯
生活場景舉例：⋯⋯

紅黑歸屬：
- 如果這條通道是黑色（個性）→ 你意識得到這個特質
- 如果是紅色（設計）→ 別人看得到但你自己不太覺察
- 如果紅黑交雜 → 你部分意識到，部分由潛意識驅動
```

**4.3 閘門解讀**

每個啟動的閘門使用以下模板：

```
【閘門 XX：閘門名稱】（能量中心名稱）
易經對應：第XX卦——卦名
五行屬性：（若適用六系統融合時標注）
位置：個性面（黑）/ 設計面（紅）
行星：太陽/月亮/水星/金星/火星/木星/土星/天王星/海王星/冥王星/南交/北交

白話翻譯：這個閘門給你的特質是⋯⋯
爻線：第X爻
爻線含義：⋯⋯（爻線是最細粒度的個人化解讀）

是否形成通道：是（與閘門YY形成通道）/ 否（懸掛閘門）
懸掛閘門說明：你擁有一半的能量，遇到擁有另一半的人時會被「接通」
```

### Layer 5：人生使命——輪迴交叉（5-8頁）

**5.1 你的輪迴交叉**
- 名稱（192種之一）
- 類型（右角度/左角度/並列）
  - 右角度 = 個人命運型（你的使命是做好自己的旅程）
  - 左角度 = 超越個人型（你的使命跟其他人深度交織）
  - 並列 = 固定命運型（非常具體的人生路徑）
- 四個關鍵閘門（個性太陽、個性地球、設計太陽、設計地球）
- 每個閘門的含義和它們如何交織成使命主題

**5.2 輪迴交叉的白話解讀**
- 「你來這個世界是為了⋯⋯」（但不是命定，是當你活出自己時自然會經歷的主題）
- 使命主題的生活化描述
- 使命主題在不同人生階段的表現
- 「你可能在什麼時候感覺到使命感最強烈？」

**5.3 輪迴交叉 × 人生角色**
- 你的角色如何服務於你的使命
- 角色跟使命之間可能的張力

## 護欄

### 三大權力對應

| 權力 | 觸發情境 | 處置 |
|------|---------|------|
| Hard Stop | 使用者要求「告訴我未來會發生什麼」→ 人類圖不預測未來 | 拒絕＋說明人類圖是設計圖不是預言 |
| Degrade | 出生時間不確定 > 2小時 | 降級為「太陽閘門＋類型＋策略」的基礎分析，標示所有不確定項 |
| Constraint | 使用者情緒低落時要求分析 | 先用 resonance 接住，確認狀態後再展開，避免在低能量時接收大量資訊 |

### 特別護欄

- **不神化**：人類圖是參考工具，不是絕對真理。每份報告必須在開頭和結尾明確聲明。
- **不替選擇**：「你適合做什麼工作」→ 改為「你的設計在什麼環境中最容易發揮」
- **不病理化**：無定義中心不是「缺陷」，是「開放性」和「學習區」
- **兒童模式護欄**：分析嬰兒/兒童時，報告對象是父母，語言導向「如何支持孩子的設計」而非「這個孩子會怎樣」
- **事實/假設/推論分離**：圖面數據=事實，系統解讀=框架，整合建議=推論，必須明確區分

## 資料來源與知識邊界聲明

本 Skill 的人類圖知識基於以下來源：
- Ra Uru Hu 原始傳訊內容（《Human Design: The Science of Differentiation》）
- IHDS（International Human Design School）官方課程體系
- Joyce Huang（亞洲人類圖學院）中文翻譯與教學體系
- 陳濼文人類圖報告書團隊的中文解讀實踐
- 各中文人類圖教師（Repeat、曼蒂等）的公開教學內容

**知識邊界**：
- 閘門爻線的「點數位」解讀（如某閘門的第3爻第4色第2音第5基底）超出目前知識範圍
- 人類圖的「四箭頭」（變數/Variable）系統為進階內容，需額外研究
- BG5（商業人類圖）為專門分支，目前僅涵蓋基礎概念
- 夢境 Rave / 環境 Rave 等進階主題暫不涵蓋

**持續演化方向**：
- 點數位解讀（色彩/音調/基底）
- 四箭頭（飲食/環境/視角/動機）
- 合盤深度（電磁通道/妥協通道/主導通道）
- 流年/流日分析
- BG5 職場應用
