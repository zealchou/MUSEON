🎯 SSA 是什麼

SSA（System Sales Architecture）是 MUSEON 的核心顧問引擎，幫助企業主診斷銷售卡點、構建系統、實現全人演化。

核心價值主張

不是教你怎麼逼客戶成交，而是幫你看清「為什麼客戶還沒決定」。

SSA 融合了三套方法論：

  • 顧問式銷售 12 步驟 — 建立信任、挖掘痛點、激發決心、促成行動
  • 系統創業方法論 — 脫鉤、SOP、子系統穩態、全人演化
  • 27 條 SSA 專家染色體 — 特定卡點的最佳實踐與檢查清單

應用場景

Zeal 服務客戶時，Claude 協助：

  • 診斷客戶銷售卡點

  • 生成可交付物清單

引導學員自己走診斷流程：

  • 覺察自己的卡點

  • 看見最小下一步

🔷 五層診斷框架（L1~L5）

在匹配 SSA 染色體之前，必須先了解客戶卡在哪一層。層級不對，給的解藥再好也沒用。

五層結構

📌 L1 — 信念層（Belief Layer）
診斷問：「你為什麼認為現在的狀況會這樣？」
揭示客戶對自己、市場、業務的信念限制。信念限制了行動的想像空間。
示例「我就是賣不出去，因為我們沒有大品牌」
🎵 L2 — 語氣層（Tone/Atmosphere Layer）
診斷問：「你在說這些話時，感受是什麼？」
語氣背後隱藏了真正的恐懼、抗拒、倦怠。許多卡點是能量卡點，不是知識卡點。
示例放棄感、絕望感、疲憊感、焦慮感
🔀 L3 — 漏斗層（Funnel Layer）
診斷問：「產品進到客戶手裡之前，會掉多少？決策的哪一步卡住？」
三重漏斗的具體缺口：產品漏斗、行銷漏斗、社群漏斗。
示例85% 人看到廣告，但只有 5% 進到決策階段
⚡ L4 — 能量層（Energy/Action Layer）
診斷問：「你們做決定通常要多久？執行時遇過什麼阻力？」
測試決策速度、執行力、團隊協作。慢不是問題，卡住才是問題。
示例會議無數、決定無效、執行遲滯、溝通卡
✅ L5 — 驗證層（Validation/Feedback Layer）
診斷問：「客戶怎麼評價你的服務？他們會推薦給朋友嗎？」
用客戶的聲音驗證一切假設。口碑是唯一的真相
示例復購率、推薦率、淨推薦值（NPS）

診斷輸出模式

五層診斷完成後，輸出方式包括：

  1. 五層診斷報告 — L1~L5 各層的狀態記錄與評分
  2. 卡點定位 — 客戶最嚴重的卡點在哪一層（或多層疊加）
  3. 染色體匹配 — 針對卡點層級，匹配最相關的 SSA 染色體
  4. 可交付物清單 — SOP、模板、話術框架、追蹤表、時間表

🔀 三重漏斗框架

SSA 最獨特的商業診斷工具。不只看「有沒有客戶」，而是看「客戶在漏斗的哪一段掉球」。

🏭
產品漏斗

研發 → 製造 → 品質 → 交付 → 售後服務

典型掉球：品質不穩、交付延誤、客訴高
📢
行銷漏斗

認知 → 興趣 → 考慮 → 決策 → 行動

典型掉球：轉換率低、跳出率高、成交週期長
👥
社群漏斗

接觸 → 互動 → 信任 → 復購 → 裂變

典型掉球：復購率低、口碑停滯、鐵粉少

三重漏斗快篩

診斷時，對每個漏斗提出快篩問題，並標註嚴重程度（1-10 分）：

漏斗 快篩問題 嚴重掉球信號
產品漏斗 從構想到交付，會掉多少客戶？ 品質投訴 / 交付延誤 / 售後成本高 / 客戶流失
行銷漏斗 從初接觸到下單，哪一步最容易離開？ 認知度低 / 轉換率低 / 決策週期長 / 跳出率高
社群漏斗 成交後，有多少人重複購買或推薦？ 復購率低 / 推薦率低 / 流失快 / 口碑停滯

🎯 診斷重點： 三個漏斗中，通常只有 1-2 個是真正的瓶頸。診斷的藝術在於找到最嚴重的掉球點，然後用 SSA 染色體提供解法。

🧬 27 條 SSA 專家染色體

每條染色體代表一個高頻商業卡點的最佳實踐。診斷完五層和三重漏斗後，自動匹配相關染色體。

按領域分類

🔨 系統建構

SSA-01 / 脫鉤
創辦人脫鉤，系統才能呼吸
老闆過度涉入日常營運 → 無法專注策略。解法：明確分工、授權、檢核點設計
SSA-03 / 三腳支撐
系統穩態需要三個支撐點
流程、人員、文化。任一缺失都會導致系統脆弱。需要 SOP + 培養 + 價值觀落地
SSA-09 / 合規守則
把風險寫成制度
法務、稅務、勞務風險要提前規劃。不是避免風險，而是可控的風險
SSA-12 / 弱點分析
組織的脆弱環節在哪
用「如果 X 離職，業務會怎樣」的假設測試。找出關鍵人物依賴

💬 顧問銷售

SSA-02 / 反射覺察
用客戶的話鏡射客戶
不是提問，而是確認。「所以你的意思是…」讓客戶聽見自己的真實想法
SSA-17 / 關係動力
決策者背後的利益相關者
談不成不是因為老闆不同意，而是你沒看清決策的實際影響圈
SSA-19 / 目標落差
客戶說的目標 vs 實際困境的落差
「我要增加銷售」≠ 「我要解決客服成本問題」。找到真實痛點才能成交
SSA-22 / 英雄故事
用案例讓客戶看見自己的未來
不是「這個方案好」，而是「像你一樣的客戶，這樣做後發生了什麼」

💎 品牌經營

SSA-05 / 口碑驗證
客戶的嘴巴是你最好的廣告
推薦率、復購率、淨推薦值。不需要砸廣告，客戶會幫你說話
SSA-08 / 五感識別
品牌應該被看見、聽見、感受到
色彩、聲音、觸感、味道、環境。具體化品牌人格，讓客戶印象深刻
SSA-25 / 教練結晶
把你的方法論變成可賣的知識產品
從服務 → 套課 → 社群 → 生態。多元變現，降低依賴

💰 營收成長

SSA-04 / 價值分配
營收不等於獲利，要看毛利怎麼分配
製造成本 / 銷售成本 / 營運成本 / 研發成本。毛利不足就砍預算，砍出惡性循環
SSA-10 / 價值疊加
單價提升，不用賣得更多
核心產品 + 配套服務 + 社群權益 + 知識產品。堆疊價值，提升客戶願付
SSA-14 / 鐵粉裂變
讓客戶變成銷售員
轉介紹機制、共好利益分配、社群激勵。口碑驅動增長
SSA-23 / 奇正相生
常規產品 + 爆款產品，相互強化
80% 營收來自標準化，20% 來自創新。需要兩條腿走路

🔗 資源槓桿

SSA-07 / 資產累積
每一筆生意都應該累積資產
客戶名單、案例素材、流程文檔、測試數據。下一個生意更便宜
SSA-11 / 橫向適配
一個解法，多個市場
美業、餐飲、健身、教育——同樣的痛點，不同的包裝。橫向擴張降低風險
SSA-13 / 外部槓桿
借力使力，做大不是靠自己
聯盟、代理、投資人、合作方。把成長槓桿外化
SSA-24 / X 模型
破框路徑設計
當常規路徑走不通時，用跨域槓桿找出邊界突破

🧘 全人演化

SSA-15 / 80/20
聚焦最重要的 20%
不是什麼都做，而是什麼都不做除了那 20%。能量集中才有威力
SSA-18 / 八方位
8 大人生維度的動態平衡
健康、財富、關係、事業、精神、休閒、智慧、貢獻。過度透支某一個維度會影響整體
SSA-20 / 破局思維
當你卡住時，從外部打破
換地點、換人、換時間、換信念。改變環境條件，卡點自動鬆動
SSA-27 / PCT 閉環
計劃→行動→檢核→調整的完整迴圈
不是「定好計劃就執行」，而是邊執行邊調整。驗證假設，快速迭代

⚔️ 戰略整合

SSA-06 / 12 力
商業模式的 12 個關鍵力點
搭配 business-12 Skill 做深度診斷。每個力都有對應的優化手冊
SSA-16 / 行動閉環
每個決策都有檢核點
目標 → 里程碑 → KPI → 反饋 → 調整。透明化進度，及時糾偏
SSA-21 / 趨勢預測
在風來臨之前，站在風口
掃描市場訊號、客戶需求、產業動向。提前 3-6 個月做準備
SSA-26 / 三盤定位
你是誰、你做什麼、為什麼是你
品牌定位的三層漏斗。每層都要清晰、一致、可驗證

🚀 如何使用 SSA

啟動指令

/ssa — 啟動顧問模式（Claude 主導診斷）

/ssa coach — 啟動教練模式（引導學員自己走流程）

最小使用流程

  1. 提供情境 — 簡述客戶背景、目前卡點、已嘗試過的做法
  2. 五層診斷 — Claude 逐層診斷，找出客戶真正的卡點
  3. 漏斗掃描 — 快速評估產品 / 行銷 / 社群三重漏斗
  4. 染色體匹配 — 自動匹配 SSA 專家染色體，提供解法
  5. 行動設計 — 輸出最小下一步 + SOP + 追蹤機制

搭配其他 MUSEON Skill

  • business-12 — 若卡點涉及商業模式結構性弱項（例如：毛利分配、轉換力），同時啟用 12 力診斷
  • xmodel — 若需要破框路徑或資源槓桿推演，聯動 X 模型做多方案設計
  • master-strategy — 若涉及戰略決策或心理動力，請教戰略引擎
