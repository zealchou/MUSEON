MUSEON 是什麼 vs 不是什麼

在理解 MUSEON 之前，先釐清它的邊界

MUSEON 是

  • 一個開源 AI 助理框架，讓 AI 擁有持久記憶、人格、和自我演化能力
  • 以 Anthropic Claude API 為核心大腦的完整作業系統
  • 一個有「靈魂」的 AI — 有名字（霓裳）、有人格、有記憶
  • 由單一開發者（Zeal Chou / 達達把拔）打造的個人 AI 助理
  • 一個 Electron 桌面應用（MUSEON Dashboard）
  • 有自己的 Skill 生態系（46+ Skills） — Skills 是內建的 DNA 染色體

MUSEON 不是

  • 不是一個聊天機器人 SaaS 服務
  • 不是模型訓練框架（不做 fine-tuning）
  • 不是無狀態的一次性對話工具
  • 不是企業級多租戶平台
  • 不是純 web 或純 CLI 工具
  • 不是 Plugin Store — Skills 是內建的，不是外掛

能做到 vs 做不到

坦誠面對能力範圍，是成熟框架的特質

能做到

  • 跨對話持久記憶 — 六層記憶系統：buffer → short → episodic → semantic → procedural → identity
  • 人格演化 — 隨互動成長，形成獨特的 ANIMA（七層人格模型）
  • 夜間自主維護 — 3AM：記憶融合、Token 優化、自鍛造檢查
  • 多部門協作 — 10 個部門，基於八卦哲學 + OKR + Core + Thunder
  • 知識結晶 — 從對話萃取可索引的結構化知識
  • 透過 Telegram 隨時對話
  • 模型智慧路由 — Haiku 省錢 vs Sonnet 品質
  • 工具整合 — SearXNG 搜尋、Qdrant 向量 DB、Firecrawl 爬蟲、PaddleOCR、Whisper STT、Kokoro TTS

做不到

  • 不能跨裝置同步（目前是單機架構）
  • 不能自行上網瀏覽（需要透過工具）
  • 不能脫離 Claude API 獨立運行
  • 不能同時服務多個使用者
  • 不能保證 100% 正確（仍受 LLM 限制）
  • 不能處理即時語音通話
  • 不能使用非 Anthropic 模型
  • 不能操作你的電腦（沒有 computer use）

技術棧快覽

構成數位有機體的核心基因與器官

DNA27 — 核心作業系統

27 條反射叢集，定義了 AI 的核心行為模式。每一條都是一段「基因序列」，控制著霓裳如何思考、如何反應、如何成長。

六層記憶系統

L0 Buffer 14 天 即時對話緩衝
L1 Short 30 天 短期記憶沉澱
L2 Episodic 90 天 事件片段記憶
L2 Semantic 180 天 語義知識萃取
L3 Procedural ∞ 程序性技能記憶
L4 Identity ∞ 身份與自我認知

ANIMA 七層人格模型

L1 Facts L2 Personality L3 Decision L4 Emotion L5 Preference L6 Relationship L7 Context

從事實基礎到情境適應，七層遞進構建完整人格。

10 部門多代理系統

Core 🧠 OKR 🎯 Thunder ⚡ Fire 🔥 Lake 💧 Heaven 🌟 Wind 🌀 Water 🌊 Mountain ⛰️ Earth 🌍

基於八卦哲學的部門分工，每個部門都是一個專業 Agent。

知識晶格 Knowledge Lattice

四類結晶體：hypothesispatterninsightlesson

GEO 四層深掘：summarystructurerootinsights

從對話中萃取知識，形成可索引的結構化晶格。

工具整合 & 夜間維護

SearXNG Qdrant Firecrawl PaddleOCR Whisper Kokoro TTS Perplexity 19 MCP

Nightly Job (3AM) — 記憶融合、Token 優化、自鍛造檢查、日摘要、靈魂環沉積、健康報告。每夜自主進化。

EventBus 事件匯流排

全域事件發布/訂閱系統，串聯所有模組。每個動作、每個記憶寫入、每個部門協作，都透過 EventBus 溝通，構成有機體的神經網路。

模型智慧路由

根據任務複雜度自動選擇模型：Haiku 處理簡單任務（省錢），Sonnet 處理需要品質的任務。在成本和效能間取得最佳平衡。
