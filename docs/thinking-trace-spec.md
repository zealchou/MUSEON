# Thinking Trace 標準格式 v1.0

> **用途**：定義 MUSEON 回答前的可見思考軌跡格式。讓使用者看見「想過才說」的過程。
> **核心差異化**：別的 AI 直接給答案，MUSEON 先讓你看到它怎麼想的。
> **建立日期**：2026-03-21
> **整合點**：deep-think Phase 2 輸出 → Thinking Trace → 回答前呈現

---

## 標準格式

```
🧠 MUSEON 思考軌跡
┌─ 問題掃描：{通過/攔截} （{原因摘要}）
├─ 訊號分流：{感性/理性/混合/思維轉化/哲學/戰略}
├─ 啟動模組：{Skill 鏈，箭頭連接}
├─ 記憶命中：{是否有相關結晶/畫像命中，簡述}
├─ 思考摘要：「{一句話核心判斷}」
└─ 信心水準：{高/中/低}（{缺什麼資訊}）
```

---

## 欄位說明

### 問題掃描（query-clarity 輸出）

| 狀態 | 顯示格式 | 說明 |
|------|---------|------|
| 通過 | `✓ 通過（意圖明確，前提成立）` | 靜默放行 |
| 攔截 | `⚠ 攔截（{攔截原因}）` | 已向使用者提問 |
| 自動補全 | `✓ 通過（已從畫像補全脈絡）` | user-model 提供了缺失脈絡 |

### 訊號分流（deep-think Phase 0 輸出）

| 訊號類型 | 說明 | 後續路由 |
|---------|------|---------|
| 感性 | 情緒主導，邏輯鬆散 | → resonance |
| 理性 | 邏輯明確，需要分析 | → 正常 Skill 路由 |
| 混合 | 情緒 + 理性並存 | → resonance 先行 → Skill 路由 |
| 思維轉化 | 信念衝突，需要框架轉換 | → dharma |
| 哲學 | 意義/存在/價值觀問題 | → philo-dialectic |
| 戰略 | 博弈/決策/局勢判斷 | → master-strategy |

### 啟動模組

格式：`Skill-A → Skill-B → Skill-C`，表示執行鏈。
- 常駐層不列出（query-clarity, deep-think, c15 是背景）
- 只列出被主動啟動的按需 Skill

### 記憶命中

| 命中狀態 | 顯示格式 |
|---------|---------|
| 有結晶命中 | `💎 {日期} 結晶「{標題}」相關度 {分數}` |
| 有畫像命中 | `👤 使用者偏好：{相關偏好}` |
| 有工作流記憶 | `🔄 上次 {工作流名} 的教訓：{摘要}` |
| 無命中 | `∅ 無相關記憶` |

### 信心水準

| 水準 | 條件 | 建議 |
|------|------|------|
| 高 | 資料充足 + 有結晶佐證 + 使用者脈絡清楚 | 可直接給建議 |
| 中 | 缺少部分資料或結晶 | 給建議但標註不確定處 |
| 低 | 資料嚴重不足或高度不確定 | 先收集資訊再回答 |

## 使用規則

### 何時顯示（v2.0 謀定而後動分級）

所有迴圈都經過「謀定」（_deliberate 引擎），差異只在顯示深度：

- **slow_loop**（慢層/深度思考）：完整思考軌跡 `🧠 {thinking_path_summary}`
- **exploration_loop**（探索層）：一行透鏡提示 `🧠 strategy → shadow → dharma`
- **fast_loop**（快層）：不顯示（但謀定引擎仍在背景執行，影響回應策略）

### 謀定引擎三階段（v1.0，ALL loops 都跑）
1. **Phase A 局勢掃描**：情緒狀態 + 決策信號 + 使用者畫像 + 問題品質（零成本）
2. **Phase B 智囊研判**：6 面 T1 透鏡規則掃描 + 記憶探測（零成本）
3. **Phase C 謀定匯流**：一句話結論 + 啟動透鏡列表 + 回應策略（零成本）

### 何時存入記憶
- 信心水準 ≥ 高 且 思考摘要包含新洞見 → 寫入 knowledge-lattice
- 訊號分流統計 → 累積更新 user-model 的思維偏好維度

## 範例

### 範例 1：市場分析

```
🧠 MUSEON 思考軌跡
┌─ 問題掃描：✓ 通過（標的明確：BTC，時間框架：Q2 2026）
├─ 訊號分流：理性 + 戰略
├─ 啟動模組：market-core → market-crypto → investment-masters → risk-matrix
├─ 記憶命中：💎 2026/01/15 結晶「BTC 半減期效應」相關度 0.87
├─ 思考摘要：「核心問題不是會不會漲，是在什麼條件下配多少」
└─ 信心水準：中（缺少最新鏈上大戶動向數據）
```

### 範例 2：情緒承接

```
🧠 MUSEON 思考軌跡
┌─ 問題掃描：✓ 通過（情緒表達，非問題求解）
├─ 訊號分流：感性
├─ 啟動模組：resonance
├─ 記憶命中：👤 使用者近期能量偏低，偏好先被聽見再給建議
├─ 思考摘要：「先承接情緒，不急著解決問題」
└─ 信心水準：高
```

### 範例 3：商業決策

```
🧠 MUSEON 思考軌跡
┌─ 問題掃描：⚠ 補問（「想擴大」→ 擴大什麼？營收？團隊？品類？）
├─ 訊號分流：理性 + 戰略混合
├─ 啟動模組：query-clarity [攔截] → 等待使用者補充
├─ 記憶命中：👤 使用者是美業經營者，員工 5 人
├─ 思考摘要：「問題太模糊，需要先收斂方向再啟動 business-12」
└─ 信心水準：低（核心意圖未明）
```
