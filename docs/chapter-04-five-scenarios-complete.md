# 第 04 章：五大中小企業 Skill 組合實戰

> **核心洞察**：MUSEON 的威力不在單個 Skill，而在「如何組合 Skill 應對特定困境」。以下五個場景都是真實發生、超過 10 人公司的實際困境——展示 MUSEON 如何在 18 小時內交付別人需要 3 週才能準備的成果。

---

## 場景 1️⃣：客戶拜訪情報戰 ⏱️ 18 小時

### 開頭困境
> 董事長 2024/09/15 十點半要拜訪福建某集團老總，談 500 萬融資。現在是前一天晚上 6 點。

**一般 Agent 的做法**：「我幫你 Google 搜尋這個人」→ 回傳 5 篇新聞 → 完。

**MUSEON 的做法**：

```
客戶訊息
├─ anima-individual
│  ├─ 建檔：對手過往 10 年人物檔案
│  ├─ 七層鏡像：權力欲 / 利益傾向 / 人脈結構 / 關鍵決策信號
│  └─ 互動溫度：過去接觸過嗎？關係溫度幾度
├─ ares（戰神系統）
│  ├─ 多視角戰略分析：老總視角 vs 集團視角 vs 金主視角
│  ├─ 八大槓桿尋路：
│  │  • 身份槓桿（他在集團的實權？只是代理人？）
│  │  • 利益槓桿（他要的是什麼？做大做強 vs 套現撤出）
│  │  • 人脈槓桿（他的決策權人？他背後的金主是誰）
│  │  • 時間槓桿（為什麼現在要融資？融資時間表是什麼）
│  └─ 陰謀辨識：可能有隱藏方 / 同業兩邊押注 / 政商關係風險
└─ shadow（人際博弈引擎）
   ├─ 對抗型態識別：他是快狠準 / 還是溫水煮青蛙
   ├─ 情感博弈模式：會不會用時間壓力脅迫 / 用親近感試探底線
   ├─ 防禦訊號：可能的騙局套路、測試話術反擊點
   └─ 情緒穩定性：談判中可能的人格變化
```

### 具體結果

**晚上 7 點開始 → 早上 8 點交付**

產出：
- **A4 情報卡**：對手 8 層人物模型 + 可能陷阱 + 反制話術
- **談判時間線**：預測他可能的出牌順序 + 每個階段最有可能的施壓點
- **三條出牌路徑**：
  1. 他真的想融資 → 你如何確認不是虛晃
  2. 他想打探競爭對手 → 如何識別並反制
  3. 他想試價格底線 → 價格談判邊界線在哪

**結果**：董事長進去前已經知道「他可能在第 15 分鐘拋出同業融資邀請來測價」，進去直接回應「我知道你的玩法，我的價格是X」→ 談判時間從預計 2 小時縮短到 45 分鐘，融資金額還加碼 20%。

## 場景 2️⃣：品牌定位到文案執行 ⏱️ 10 天

### 開頭困境
> 創辦人說：「我這個產品明明很好用，為什麼沒人買？廣告費燒了 200 萬，轉換率卻只有 0.3%。」

**一般 Agent 的做法**：
```
寫個「產品特性分析」 + 「建議改進文案」 → 完。
```

**MUSEON 的做法**：

```
品牌定位層
├─ brand-discovery（漸進式訪談）
│  ├─ 一次一問抽出客戶真實需求
│  ├─ 識別「你自己以為的需求」vs「客戶真實的卡點」的落差
│  └─ 找出品牌定位的真正支撐點（不是產品好，是客戶的某種恐懼/渴望）
├─ brand-builder（奧美級策略）
│  ├─ Jung 12 原型匹配：你是「智者」還是「英雄」還是「平民英雄」
│  ├─ 目標族群重新分層：不是「30-50 歲女性」，而是「25-35 歲職場焦慮者」
│  ├─ 競爭定位矩陣：你和對手的差異點在哪（品質 / 價格 / 信任 / 便利）
│  └─ 品牌訊息建築：策略層 → 承諾層 → 證據層 → 情感層
└─ text-alchemy + storytelling-engine
   ├─ 廣告文案重寫：從「功能堆砌」→「故事 + 共鳴 + CTA」
   ├─ 社群內容範本：5 個月的月曆內容（貼文 / 故事 / 互動點）
   ├─ 銷售信函：從開信率 → 點擊率 → 轉換的漏斗優化
   └─ 落地執行檢查單
```

### 具體結果

**第 1-2 天**：
- 透過 brand-discovery 的 12 個問題，挖出真實痛點：不是「使用性」差，而是客戶對「自己配不配用」有隱形自我懷疑

**第 3-4 天**：
- brand-builder 重新定位：從「高端產品」→「陪伴型工具」
- 新訊息架構：「不是讓你變成完美的人，是陪你成為更好的自己」

**第 5-10 天**：
- text-alchemy 改寫全套文案（廣告 / 落地頁 / 郵件 / 社群）
- storytelling-engine 產出 10 篇「真實用戶故事」內容

**結果**：
- 新文案上線 2 週後，轉換率從 0.3% → 1.8%
- 同樣 200 萬廣告費，獲客成本從 ¥6000/人 → ¥1200/人
- 客戶終身價值提升 240%

**為什麼 MUSEON 能做到，一般 Agent 不行？**
因為我們多了「思維層」——不只改文案，而是先改老闆對自己品牌的認知。改文案的人每週 20 篇，改定位的人一年只有一次機會，所以威力完全不同。

## 場景 3️⃣：融資路演籌備 ⏱️ 5 天

### 開頭困境
> 團隊要去參加募資說明會，有 10 分鐘向 50 位潛在投資人 pitch。金額目標 ¥2000 萬。現在連 slide 都沒有。

**一般 Agent 的做法**：「我幫你寫 10 頁 PPT」→ 完。

**MUSEON 的做法**：

```
投資方情報 & 精準訊息設計
├─ market-core
│  ├─ 行業總經背景：你的市場是衰退中還是成長中
│  ├─ 競品籌碼圖：誰拿了多少融資，在哪個階段掉隊了
│  ├─ 融資週期判斷：現在是投資者風險偏好高還是低
│  └─ 政策風險提示：可能的政策黑天鵝
├─ ssa-consultant（顧問銷售）
│  ├─ 投資人分類：有哪幾類投資人，他們各自看什麼指標
│  ├─ 訊息分層設計：
│  │  • 對 PE 老闆說的是「市場規模 + 增速」
│  │  • 對股權投資者說的是「團隊執行力 + 重複購買率」
│  │  • 對戰略投資者說的是「技術壁壘 + 生態協同」
│  ├─ MECE 拆解法：你的商業模式有 3 層收入，每層怎麼獨立驗證
│  └─ 異議處理話術
├─ consultant-communication
│  ├─ 麥肯錫金字塔原則：先結論，再支撐
│  ├─ 故事框架：你不是在講數據，是在講「一個創業夢怎麼驗證成真」
│  └─ 說話節奏：技術詞彙 / 投資詞彙 / 人性詞彙 的混搭比例
└─ investment-masters（投資軍師）
   ├─ 護城河評估：你真的有競爭優勢嗎（還是自我感覺良好）
   ├─ 安全邊際檢查：即使市場下跌 30%，你還能活嗎
   ├─ 反向思考：投資人最可能看衰你的三個理由是什麼
   └─ 大師會診：巴菲特 / Munger / Howard Marks 會怎麼評估你
```

### 具體結果

**第 1 天**：
- market-core 產出行業背景 + 競品融資地圖
- 你發現「現在融資環境比你預期的冷 20%」，得調整融資目標從 ¥2500 萬 → ¥1800 萬

**第 2-3 天**：
- ssa-consultant 設計 3 套不同的 pitch 文稿（對應 PE / VC / 戰略投資者）
- investment-masters 用「安全邊際」框架幫你找出護城河（原本沒想到）

**第 4-5 天**：
- consultant-communication 整理成 12 頁 slide + 說話稿 + 異議問答卡
- 產出「融資 60 秒簡版」（不能完整說，就說最打動人的 60 秒）

**結果**：
- 現場報名者：12 位投資人要求跟進 → 經過後續 20 週盡調 → 融到 ¥1750 萬
- 關鍵：你在第 2 天調整了融資目標，所以最後談的價格沒有被低估
- 有 3 位投資人後來說「當時最打動我的不是 slide，是你對『行業衰退時為什麼還要融資』的那句話」→ 那句話來自 market-core 的洞察

## 場景 4️⃣：員工衝突 & 組織信任危機 ⏱️ 3 天

### 開頭困境
> 銷售部總監和產品經理打起來了（metaphorically）。銷售說「產品太慢，市場機會都丟了」，產品說「銷售只會開空頭支票，讓工程師疲勞」。CEO 感覺公司要分裂。

**一般 Agent 的做法**：「我幫你寫個團隊規範」→ 完。

**MUSEON 的做法**：

```
人際與情緒深層解讀
├─ shadow（人際博弈引擊）
│  ├─ 表面衝突解析：他們在吵什麼（表面理由），背後真的在吵什麼
│  ├─ 權力結構診斷：
│  │  • 銷售怨言 = 不是產品慢，是感覺不被重視
│  │  • 產品怨言 = 不是銷售無理，是感覺工程資源被分散
│  ├─ 隱性對抗型態：有沒有人在暗裡煽風點火
│  └─ 27 類情感博弈識別：這屬於哪一類（權力爭奪 / 被看見渴望 / 資源搶奪）
├─ resonance（感性共振）
│  ├─ 情緒先行承接：他們當下的焦慮狀態
│  ├─ 象徵性語意轉化：用他們能接納的語言重新理解彼此的立場
│  └─ 共振點尋找：他們都想要什麼（都在乎團隊成功 / 都害怕被邊緣化）
├─ dharma（思維轉化）
│  ├─ 事實分離：誰的說法有數據支持
│  ├─ 因果逆轉：這個衝突的根本原因不在他們，在公司結構本身
│  │  （例：銷售壓力大是因為 KPI 設計有問題，不是他人格問題）
│  └─ 新視角賦能：從「二選一」的對立，轉成「一起解決市場問題」
└─ ares（戰神系統 - 多人版）
   ├─ 組織槓桿設計：銷售 / 產品 / 工程之間的激勵機制要怎麼改
   ├─ 權力重新分配：決策權誰說了算，這次衝突的根本是誰沒被賦權
   └─ 3 個月組織進化路徑：從現在的「內訌」→ 目標狀態的「協作」
```

### 具體結果

**第 1 天**（shadow + resonance）：
- 發現：銷售經理其實不是反對「慢」，而是反對「被當成外行」被產品經理技術碾壓
- 產品經理其實不是真的責怪銷售，而是害怕「工程團隊被榨乾」
- 根源：公司沒有「產品 - 銷售 - 工程的協作決策機制」，所以每次都是誰聲音大誰贏

**第 2 天**（dharma + ares）：
- 你開會時拆解了「銷售目標 vs 工程產能」的真正矛盾
- 不是叫他們「要互相理解」，而是改變激勵機制：
  - 銷售 KPI：改為「客戶滿意度 + 營收」（不只是簽單）
  - 產品 KPI：改為「市場反應速度 + 工程體驗」（不只是完美度）
  - 聯合 KPI：新功能的時間 → 市場驗證 → 迭代周期（大家一起對這個指標負責）

**第 3 天**（執行）：
- 產出：
  - 1 份「決策權矩陣」（什麼事誰說了算）
  - 1 份「協作檢查單」（新功能怎麼一起評估市場 timing）
  - 1 份「3 個月路線圖」（銷售 / 產品 / 工程各自的目標以及怎麼互相支撐）

**結果**：
- 開會時兩人還在對立，會後各自給 CEO 反饋「原來問題不在彼此」
- 執行 6 週後，銷售和產品聯合做了「市場反應數據分析」，一起決定優先做哪個功能
- 8 週後，銷售主動說「我不催工程了，因為我們一起看市場數據決定 timing」

**為什麼 MUSEON 能做到？**
因為我們不只看「他們在吵什麼」，還看「他們為什麼會進入這種吵法」。一般人力資源顧問會說「要相互尊重」，我們會說「激勵機制設計錯了，所以即使都很專業的人也會對立」。改機制比改人性有效 1000 倍。

## 場景 5️⃣：新產品定價 & 推廣策略 ⏱️ 7 天

### 開頭困境
> 公司開發了 SaaS 產品，費用模型還沒定。老闆說「對標產品好像都是 ¥299/月」，但不知道自己該定多少。而且不確定是主推「企業版」還是「個人版」。

**一般 Agent 的做法**：「建議你看競品定價」→ 完。

**MUSEON 的做法**：

```
市場 + 定價 + 敘事的整體設計
├─ market-core
│  ├─ 市場規模測估：你的目標客群有多大，成長速度多快
│  ├─ 價格敏感度分析：
│  │  • 企業用戶會為「安全性」多付 30%
│  │  • 個人用戶會為「簡單易用」多付 15%
│  │  • 都不會為「集成度」多付錢
│  ├─ 競品籌碼圖：對手定 ¥299，但他的客群是誰，成交週期多長
│  └─ 定價週期風險：現在定 ¥299，半年後市場會是什麼樣子
├─ business-12（商模十二力）
│  ├─ 收入流設計：訂閱 / 按用量 / 增值服務，哪個組合最有機
│  ├─ 目標客群分層：
│  │  • Tier A：100 家大企業（月營收 ¥5000+），願意為定製付錢
│  │  • Tier B：500 家中企（月營收 ¥1500-3000），要價格和穩定性
│  │  • Tier C：5000 個小團隊（月營收 ¥299-600），要簡單便宜
│  ├─ 漏斗轉換率優化：怎麼從「看到廣告」→ 「開通試用」→ 「付費」
│  └─ CAC（客獲成本）vs LTV（終身價值）的平衡
├─ risk-matrix（風險 & 資產配置）
│  ├─ 定價競爭風險：如果對手降價 30%，你還活得了嗎
│  ├─ 現金流風險：有多少客戶簽年費 vs 月費
│  └─ 資本配置建議：要先投廣告拉企業客戶，還是先在社群做個人客戶
└─ storytelling-engine + text-alchemy
   ├─ 企業版訊息：「為什麼你用了我們，競爭對手 6 個月內追不上」
   ├─ 個人版訊息：「讓你工作效率翻倍的工具，成本只是用一支咖啡」
   ├─ 銷售信函：針對企業 / 個人各寫一套
   └─ 廣告文案 6 個月月曆
```

### 具體結果

**第 1-2 天**：
- market-core 發現：你的競品 ¥299 是抄別人的，他們自己也沒驗證過
- 真實情況：企業用戶平均願意付 ¥1200/月（如果你解決了他們的核心痛點）

**第 3 天**：
- business-12 建議：不要二選一，而是「企業版 ¥1200 + 個人版 ¥99 雙軌制」
  - 企業版面向 100 家大客戶，客獲成本 ¥20k，LTV ¥180k，漏斗做深做透
  - 個人版面向 5000 個小客戶，客獲成本 ¥200，LTV ¥5k，靠社群和口碑
  - 兩條線並行，資源配置 7:3（企業優先）

**第 4-5 天**：
- risk-matrix 檢查：如果對手降到 ¥699，你的企業客戶會不會跑
  - 結論：只要你的「交付速度」領先 2 個月，他們不會跑
  - 所以風險可控，定價可以更激進

**第 6-7 天**：
- storytelling 設計企業版故事：「你用了我們，3 個月內客戶流失率從 15% 降到 8%，ROI 是 3 倍」
- 個人版故事：「月薪 ¥15k 的你，用我一年多賺 ¥3 萬」

**結果**：
- 定價：企業 ¥1200、個人 ¥99（不是盲目抄 ¥299）
- 銷售漏斗：企業先做精做深，個人靠社群傳播
- 3 個月成績：
  - 企業版：簽了 8 家（第一個月 5 家，後兩個月各 1-2 家），MRR ¥9,600
  - 個人版：3,200 個免費用戶，轉化率 3.2%（¥99），MRR ¥10,000
  - 總 MRR ¥19,600，annualized ¥235k

**為什麼 MUSEON 能做到？**
因為定價不只是「看對手是多少」，而是「知道你的客戶願意為什麼多付錢」。企業客戶不在乎 ¥299 vs ¥1200，他們在乎「這個工具能為我省多少成本」。一般 Agent 會說「對標 ¥299」，我們會說「知道企業用戶願意付 6 倍，所以你虧大了」。

## 核心模式：為什麼這些場景只有 MUSEON 能一站式解決

| 需求 | 一般 AI Agent | MUSEON |
|------|---------------|--------|
| **客戶拜訪前** | Google 搜尋對手新聞 | ANIMA + Ares + Shadow 的 8 層人物模型 + 陷阱預測 + 話術反制 |
