## 📖 完整劇本

### 【開場引言】（15 秒）

```
👧心允：
「拉丁舞誰最經典？」

👧霓安：
「Rumba、ChaCha、鬥牛、森巴都有道理～」

👧同聲：
「一起看看！」
```

**指導**：語調溫暖、好奇，不急著說，給觀眾「反應時間」。

---

### 【場景一】快 vs 慢（約 25 秒）

```
👧倫巴舞者（優雅走出，語氣超溫柔）：
「Rumba 要～慢～，像摸棉花糖。」
（同時做誇張慢動作轉身）

👦恰恰舞者（跳躍著衝過來，語速快）：
「傻蛋！恰恰要快！啪啪啪啪！」
（連續點步）

👧倫巴舞者（皺眉，手放心口）：
「你不懂拉丁的靈魂～」

👦恰恰舞者（手插腰，挑釁）：
「我阿嬤來都比你快！」

👧倫巴舞者（也插腰，對峙）：
「叫她來啊！」
```

**指導**：
- 倫巴：用肢體語言表達「慢」（拖字尾、慢轉身）
- 恰恰：語速快、肢體快、充滿能量
- 對峙時製造「僵持」的眼神接觸（3 秒停頓）

---

### 【場景二+三】群雄亂舞（約 35 秒）

```
👦鬥牛舞者（從舞台邊挺胸走出）：
「誰叫阿嬤？我是西班牙最帥鬥牛舞者！」
（擺出超誇張 Pose）

👦恰恰舞者（打斷他，故意調皮）：
「哈！鬥牛舞其實來自法國，你知道嗎？」

👦鬥牛舞者（瞬間卡住，做「石化」狀，2 秒停頓）：
「……沒關係！西班牙最帥！Viva España!」
（雙手高舉、誇張表情）

👧全員（齊聲，做驚訝表情）：
「ㄘㄟˊ」（帶口部小的「噓」音）

---

👧森巴舞者（邊走邊搖擺 Samba Walk，作睡眼惺忪狀）：
「不要吵啦！都吵到我睡不好了……」

👦鬥牛舞者（轉向森巴，假裝罵）：
「哪有人打扮得像隻熱帶鳥去睡覺啦！」

👧森巴舞者（立刻清醒，反駁，做誇張扭腰動作）：
「隨時能上台才專業啦！森巴最熱情～」
（扭扭屁股）

👧倫巴舞者（加入，做嬌媚手勢）：
「要有感情～」

👦恰恰舞者（也加入，做出力量感的 Pose）：
「要有力量！」

👧森巴舞者（決定性台詞，指向觀眾）：
「那就比給觀眾看啦！」

👧全員（齊聲，興奮）：
「好啊！比就比！」
```

**指導**：
- 鬥牛的「卡住」要做 2 秒完全靜止（製造笑點）
- 全員的「ㄘㄟˊ」要同時做驚訝臉
- 森巴出現時製造「睡眼」vs「瞬間清醒」的反差
- 倫巴、恰恰的加入要逐步遞進（不要一起說）
- 最後「那就比給觀眾看」是**進舞蹈的觸發點**

---

### 【第一段舞蹈：各自風格 Solo】（約 50 秒）

```
🎵 四人輪流秀自己的舞種：

1️⃣ 倫巴（約 12 秒）
   ➜ 優雅的轉身 + 髖部搖擺（Rumba Motion）
   ➜ 配樂：慢版拉丁樂（120 BPM）

2️⃣ 恰恰（約 12 秒）
   ➜ 快速點步 + 轉圈（Cha-Cha Timing）
   ➜ 配樂：快版拉丁樂（130 BPM）

3️⃣ 鬥牛（約 13 秒）
   ➜ 挺胸闊步 + 誇張 Pose（Paso Doble Attitude）
   ➜ 配樂：鬥牛進行曲風（Flamenco 元素）

4️⃣ 森巴（約 13 秒）
   ➜ 彈性搖擺 + 腳下節奏（Samba Bounce）
   ➜ 配樂：森巴節奏（165 BPM）

背景：掌聲 + 觀眾尖叫聲（溫和，不打斷舞蹈）
```

**指導**：
- 每人 Solo 時，其他三人在舞台邊觀看、點頭或鼓掌
- 可在觀眾席安排「家長掌聲」來製造現場感
- 舞蹈銜接處：上一個人最後一個 Pose → 下一個人跑上舞台

---

### 【場景四】反轉與轉場（約 15 秒）

```
（掌聲漸弱）

👦鬥牛舞者（仔細聽掌聲，驚訝地說）：
「大家掌聲都一樣耶！」
（做出「這很奇怪」的表情）

👧倫巴舞者（走向前，溫柔地說）：
「因為～每種舞都很特別啦～」
（伸出手，邀請全員）

👧森巴舞者 + 👦恰恰舞者（齊聲，興奮）：
「一起跳最美妙！再來一次！」
（開始拉手做圈）

```

**指導**：
- 鬥牛的「大家掌聲都一樣」要帶著「困惑+恍悟」的表情
- 倫巴的反轉台詞是**情感的高點**，要說得有感情
- 全員拉手做圈時製造「準備進第二段」的視覺

---

### 【第二段舞蹈：融合舞】（約 70 秒）

```
🎵 四人合舞 + 融合多種舞蹈元素：

第一部分（30 秒）：兩兩配對
  ➜ 倫巴 + 恰恰 舞蹈對話
  ➜ 鬥牛 + 森巴 舞蹈對話
  （配樂：融合前兩種節奏）

第二部分（25 秒）：全員合舞
  ➜ 四人排成一列或圓形
  ➜ 融合多種舞步（Jive 快拍節奏為主）
  ➜ 配樂：Jive 節奏（170 BPM）

第三部分（10 秒）：定格
  ➜ 全員同時停在一個誇張 Pose
  ➜ 最後掌聲 + 尖叫

背景音效：漸強的樂曲 + 掌聲
```

**指導**：
- 融合的核心：不是一個人跳一個人跳，而是**互動**（對視、拍手配合）
- Jive 節奏製造「最後的狂歡」感
- 定格 Pose 要全員一致，做 5 秒完全靜止
- 掌聲漸強，最後達到高潮

---

### 【結尾口號】（約 10 秒）

```
（掌聲中，四人站成一排，面向觀眾）

👧全員（帶著節奏，像口號一樣）：
「誰最厲害不重要！」
（做出「不重要」的手勢）

「一起跳舞最美妙～！」
（做出歡樂的雙手舉高動作）

👧四人（同時報名字，速度快，帶來親密感）：
「我們是～心允、霓安、樂晴、以樂！」

（全員敬禮謝幕）

👧全員（最後齊聲，甜蜜的語調）：
「謝謝大家！」
```

**指導**：
- 開始的掌聲要漸弱，讓台詞清晰
- 「不重要」的手勢製造幽默
- 報名字時要有「介紹自己」的親密感（對著特定家長區域看）
- 敬禮要整齊、同步
- 最後的「謝謝大家」要帶著真誠的微笑

## ✅ 排練提示卡片

### 角色特色提示

| 角色 | 氣質 | 關鍵台詞 | 肢體特徵 |
|------|------|---------|---------|
| **倫巴** | 優雅、感性 | 「拉丁的靈魂～」 | 拖字尾、慢動作、手放心口 |
| **恰恰** | 快速、有力 | 「傻蛋！」、「我阿嬤都比你快！」 | 語速快、點步、手插腰 |
| **鬥牛** | 驕傲、詭異 | 「西班牙最帥！」、「大家掌聲都一樣耶！」 | 挺胸、誇張 Pose、突兀說話 |
| **森巴** | 活潑、聰慧 | 「隨時能上台才專業！」、「一起跳舞最美妙！」 | 搖擺走路、扭腰、決定性台詞 |

### 時間檢查點

- ⏱ **30 秒**：倫巴 vs 恰恰 對峙結束
- ⏱ **65 秒**：鬥牛「ㄘㄟˊ」反應
- ⏱ **100 秒**：「那就比給觀眾看啦」（進舞蹈）
- ⏱ **150 秒**：第一段 Solo 結束
- ⏱ **165 秒**：「大家掌聲都一樣耶」反轉
- ⏱ **220 秒**：敬禮謝幕

### 排練節奏

**第 1-2 次排練**：
- 讀台詞，別打亂節奏，重點是記熟
- 每人練自己的部分

**第 3-4 次排練**：
- 加上肢體動作，感受節奏加速
- 練習台詞的語速變化

**第 5-6 次排練**：
- 全員齊舞，練銜接點
- 調整掌聲、音效時機

**第 7 次排練**：
- 完整走位，計時驗證
- 調整舞蹈時長（如超時就縮舞蹈，不要加台詞）

## 📋 檢查清單（Pre-Performance）

開演前 2 小時檢查：

- [ ] 四人都背完台詞了？有沒有卡詞？
- [ ] 音樂、掌聲、尖叫音效都到位？
- [ ] 舞蹈順序清楚嗎？（Rumba → Cha-Cha → Paso Doble → Samba）
- [ ] 第二段融合舞蹈排好了？
- [ ] 舞台空間夠嗎？（需要至少 3m x 3m）
- [ ] 道具（如有）都在？（倫巴的扇子？）
- [ ] 服裝確認過了？（各舞種都不一樣）
- [ ] 麥克風測試過？（如果用麥克風的話）
- [ ] 燈光設定好？（如有追光）
- [ ] 排練前 15 分鐘做快速走位 + 音樂搭配

## 🎬 vs AI 版本的改進之處

| 維度 | AI 版本問題 | 本版本改進 |
|------|-----------|---------|
| **節奏感** | 感覺 2 倍速，喘不過氣 | 加入「停頓點」+ 群雄亂舞遞進感 |
| **笑點清晰度** | 有些笑點被砍到不明顯 | 笑點臺詞完整保留（阿嬤梗、法國梗、ㄘㄟˊ） |
| **舞蹈銜接** | 說完就跳，很突兀 | 用「那就比給觀眾看啦」做決定觸發 |
| **情緒線** | 平的，沒有反轉高點 | 倫巴的「每種舞都很特別」製造感人反轉 |
| **多人平衡** | 有人話太多，有人幾乎沒台詞 | 台詞分配均勻，每人都有 solo moment |
| **小朋友友善度** | 句子有點長，容易忘 | 短句為主（多數 ≤15 字） |

## 💬 給 Luo 的話

這版本核心思路是：**不是砍字，而是重新設計節奏層次**。

🎯 我保留了：
- 阿嬤梗、法國梗、森巴睡覺、ㄘㄟˊ 反應
- 倫巴的「靈魂」、恰恰的「力量」、鬥牛的「帥」、森巴的「熱情」
- 小朋友已經排練過的角色感

🎯 我改了什麼：
- 開場極簡（3 句）→ 快速進入
- 倫巴的獨白簡化但保留精髓
- 鬥牛 + 森巴 場景合併 → 製造「群雄亂舞」的加速感
- 用「那就比給觀眾看啦」自然銜接舞蹈（不突兀）
- 加上倫巴的反轉台詞「每種舞都很特別」→ 製造感人的轉折

⏱ 時間應該是 **3 分 30-40 秒**，舞蹈充分（120 秒），台詞流暢（85 秒）。

有問題的話，我們邊排練邊調整！
