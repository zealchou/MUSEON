## 核心概念

SSA（Systemic Startup Accelerator）系統創業加速器包含 **27 條染色體**，代表顧問式診斷的完整維度。這 27 條染色體結構化為三個認知層級：

1. **第一章 — 理念層**（8 條，SSA-I.01~08）：品牌價值觀的基礎
2. **第二章 — 設計層**（10 條，SSA-I.09~18）：價值傳遞的系統架構
3. **第三章 — 分配層**（9 條，SSA-I.19~27）：價值共創與分配機制

每條染色體對應 SSA-I 課程的特定模組，兼涵五層學習（L1 概念→L2 語言→L3 結構→L4 行動→L5 控制）。

## 使用指南

### I. 顧問快速路由——客戶診斷定位

**場景**：初次與客戶接觸，3~5 分鐘快速診斷需求

1. **第一層篩選**（問 5 個問題）：
   - 「貴公司的品牌理念清楚嗎？」→ 如否，進入 #01~08（理念層）
   - 「你們有明確的客戶旅程設計嗎？」→ 如否，進入 #09~18（設計層）
   - 「你們怎麼留住核心社群？」→ 如否，進入 #19~27（分配層）

2. **精確定位**（依客戶回答）：
   - 若理念模糊 → 推薦 SSA-I.01、SSA-I.08（#01、#08）
   - 若漏斗效率低 → 推薦 SSA-I.10~12（#10~12）
   - 若缺乏長期共創機制 → 推薦 SSA-I.19~27（#19~27）

3. **定製課程組合**：
   - 傳統企業（品牌重建）：#01~08 + #09~12（6 週）
   - SaaS 增長型（漏斗優化）：#09~18（8 週）
   - 社群經營型（生態自驅）：#13~27（12 週）

### II. 教練課程設計——螺旋式進階

**原則**：每條染色體在 L1~L5 五層中都能展開

**3 個月基礎課程路徑**：
```
週 1-2：理念澄清（#01~08，L1-L2）
  └─ 工作坊：Golden Circle、理念語言、品牌一致性

週 3-4：產品設計（#10，L3-L4）
  └─ 實作：漏斗分層、各層價值定位

週 5-6：行銷語言（#11、#14，L3-L4）
  └─ 工作坊：三層漏斗文案、轉換語句

週 7-8：社群留存（#12、#22、#25，L3-L4）
  └─ 實作：3 個月社群計畫、粘性設計

週 9-10：整合與迭代（#13、#18，L4-L5）
  └─ 驗証：30 日行動計畫、KPI 跟蹤

週 11-12：共創與進化（#19~27，L4-L5）
  └─ 設計：長期社群進化路徑、6 個月計畫
```

**每週迴圈**：
- 週一：講座（L1 概念）
- 週二～四：工作坊（L2~L4 實作）
- 週五：檢討回顧（L5 自我管理）

### III. 診斷完整性檢查清單

在客戶檔案中標註：

```
[ ] 理念層是否健全？（#01~08 覆蓋度）
[ ] 設計層是否完整？（#09~18 覆蓋度）
[ ] 分配層是否可持續？（#19~27 覆蓋度）
[ ] 五層深度是否足夠？（L1-L5 均衡度）
[ ] 與客戶的行業特性是否匹配？（定製度）
```
