第三章 ROUNDTABLE 圓桌詰問

博弈矩陣與交鋒分析

⚔ 軍師視角（孫子兵法）

Scott 的「大師境界」路線 = 以逸待勞。韓大哥到處承諾 = 兵力分散，三條戰線齊開卻無一鞏固。Dorin 孤軍奮戰 = 將在外軍令不受，無後援難持久。

核心策略：不戰而屈人之兵——用「愛民如己」的實際行動讓韓大哥的空頭支票自動曝光，不需主動攻擊，時間與行動差距會自動說話。

🌀 縱橫家視角（鬼谷子）

韓大哥的弱點已知（說到沒做到），不需要主動攻擊，只需要用行動形成對比。Dorin 鐵樁腳多但風向一轉就沒了——Scott 需要創造風向（品牌宣言 + 大師理念清晰化），而非等風向出現。

最危險的情境：Scott 內核不清晰 + Dorin 氣勢維持 → 可能被反超。時間不等人，內核定調必須在第一週完成。

🔮 教練視角（DHARMA）

Scott 最大的敵人不是韓大哥也不是 Dorin，是自己的內在不穩定。微軟事件的心結必須在一週內放下——不是忘記，是接受並轉化。

大師境界 = 處變不驚 = 看到問題背後的問題——這本身就是品牌宣言的核心。「大師」不是頭銜，是每一次面對挑戰時的行為模式。

🕵 情報分析視角（Shadow）

韓大哥可能的動作：最後兩週密集拜票、發佈創新政見、拉攏中立派。機率：中等，但已無足夠信用支撐。

Dorin 可能的動作：維持現有鐵樁腳、保守穩健、等風向。弱點在於被動，無法主動造勢。

Scott 的最大盲點：覺得「時間會證明一切」但沒時間了。每一天不行動 = 把機會讓給 Dorin。
