## 二、高風險詞彙詳解

### 🔴 風險 1：「三層各自抽成」+ 「同一客戶最多被三層各抽一次」

**現狀位置**：
- `commission-and-bonus-rules.md` L90-100
- `commission-and-bonus-rules-v3.md` L109-119

**現有文本**：
```markdown
三層為**獨立等級，各自抽成**，不疊加。同一客戶的月費分潤，三層各自依自己的級距抽取。

**同一客戶最多被三層各抽一次**，上限為三層分潤加總。
```

**風險分析**：
- 「三層各抽」乍看像多層次獎金機制
- 用詞「抽」字帶有財務操縱感
- 容易被解讀為「同一客戶被收割多次」

**修改建議**：
```markdown
MUSEON 採用垂直員工薪酬結構。同一客戶的月費分潤由以下三個角色獨立計算報酬：

1. **業務人員**（直接服務客戶）— 根據個人級距獲得提成
2. **業務主管**（管理業務人員）— 根據個人級距獲得團隊管理報酬
3. **經銷商**（認證經營者）— 根據個人級距獲得運營報酬

三層報酬基於各自的績效級距和服務價值獨立計算，不存在「上層從下層獲利」的機制。
同一客戶的月費（NT$6,000）被三層按比例分配，總和不超過月費本身。
```

---

### 🔴 風險 2：「招募並培訓業務人員」

**現狀位置**：
- `commission-and-bonus-rules.md` L32
- `commission-and-bonus-rules-v3.md` L104

**現有文本**：
```markdown
- **經銷商**：完成培訓 + 能獨立輔導客戶，可招募並培訓業務人員
```

**風險分析**：
- 「招募」容易被聯想為傳銷式的「拉人」
- 沒有強調「招募」的人是 MUSEON 員工（受僱關係），不是獨立經銷商

**修改建議**：
```markdown
- **經銷商**：完成培訓 + 能獨立輔導客戶，有權聘用並培訓業務人員
  （所有直屬員工均簽訂 MUSEON 勞動合約，受公司薪酬制度管轄，非獨立經銷商網路）
```

或更清晰的版本：
```markdown
- **經銷商**：經認證的資深業務執行官，可以組建直屬業務團隊並負責其培訓
  （團隊成員為 MUSEON 正式員工，非代理商或獨立承包人）
```

---

### 🟡 風險 3：「旗下業務簽的客戶」

**現狀位置**：
- `commission-and-bonus-rules.md` L97-98
- `commission-and-bonus-rules-v3.md` L113-116

**現有文本**：
```markdown
| 業務主管 | 旗下業務簽的客戶 | 依主管自己級距抽 |
| 經銷商 | 旗下所有客戶 | 依經銷商自己級距抽 |
```

**風險分析**：
- 「旗下」是傳銷常用術語，與「下線」含義接近
- 應該用「直屬」或「直報」來強調正式員工關係

**修改建議**：
```markdown
| 業務主管 | 直屬業務員簽署的客戶 | 依主管自己級距計算團隊管理報酬 |
| 經銷商 | 所轄業務體系內的所有客戶 | 依經銷商自己級距計算運營報酬 |
```

---

### 🟡 風險 4：「分潤」用詞歧義

**現狀位置**：整份獎金規則文件

**風險分析**：
- 「分潤」在商業中有兩種含義：
  1. 合法的「績效分配」（公司員工的薪酬一部分）
  2. 傳銷的「多層獎金」（上層從下層獲利）
- 文脈中是第 1 種，但用詞歧義大

**修改建議**：
在文件開頭或關鍵段落澄清：
```markdown
### 名詞定義

**月費分潤**：MUSEON 員工基於績效級距從月費收入中獲得的報酬。
- 非外包收入，而是員工薪酬組成
- 基於個人級距獨立計算，不涉及「上層從下層獲利」
- 類似傳統企業的「管理層分紅」或「銷售提成」
```

## 四、制度層面（無風險）

以下機制確認為合法且無傳銷特徵：

✅ **月費分潤不是「多層獎金」**
- 每層獨立計算，基於各自績效，不疊加
- 客戶付月費一次，被三層各分配一次（不是重複收費）
- 類似傳統公司的「主管分紅」或「區域經理報酬」

✅ **升降級完全基於績效，無「拉人數」門檻**
- v2.0：案件數（客戶簽約）
- v3.0：積分（新案 + 客戶留存）
- 沒有「推薦 N 人升級」的要求

✅ **防反套利機制完善**
- 同統編 90 天內只算 1 次新案（防拆單）
- 引薦需主管確認（防假引薦）
- 追回客服不得為原負責業務（防故意放流失）
- 簽約後 7 天內退約不計案件（防品質差客戶）

✅ **沒有「進貨壓力」**
- 沒有囤貨制、進貨金額、最低消費要求
- 業務人員服務客戶，不需要「買入產品」

## 六、對外溝通樣板

### 如果被問「會不會是傳銷？」

**標準回答**：
```
MUSEON 是典型的「直僱員工、績效分級」的企業薪酬模式，與傳銷有以下本質區別：

1. 【受僱關係】
   ✗ 傳銷：獨立經銷商（簽合約給公司，自負盈虧）
   ✓ MUSEON：所有業務人員簽訂正式勞動合約，受公司薪酬制度管轄

2. 【收入來源】
   ✗ 傳銷：主要來自「發展下線」（拉人獲利）
   ✓ MUSEON：主要來自「客戶簽約」（服務客戶獲利）

3. 【升級門檻】
   ✗ 傳銷：需要「推薦 N 人加入」
   ✓ MUSEON：完全基於績效（簽約案件或積分），無「拉人」要求

4. 【防反套利】
   ✗ 傳銷：無防機制
   ✓ MUSEON：同統編 90 天內只算 1 次、引薦需主管確認、追回客服防護等

5. 【進貨壓力】
   ✗ 傳銷：業務員需「進貨」才能上市
   ✓ MUSEON：無任何進貨、囤貨、最低消費要求

簡言之，MUSEON 是「員工制銷售組織」（如 IBM、甲骨文的銷售部），不是「經銷商網路」。
```

## 七、附錄：修改樣板

### 修改樣板 A：「三層各抽」敘述

**改前**：
```
三層為獨立等級，各自抽成，不疊加。同一客戶的月費分潤，三層各自依自己的級距抽取。
同一客戶最多被三層各抽一次，上限為三層分潤加總。
```

**改後**：
```
MUSEON 採用垂直員工薪酬結構。業務人員、業務主管、經銷商三個角色在服務同一客戶時，各自根據自身績效級距和職位責任獨立計算報酬：

- **業務人員**獲得簽約獎金與月費分潤（服務客戶的直接報酬）
- **業務主管**獲得團隊管理報酬（管理直屬業務員的報酬）
- **經銷商**獲得運營報酬（整個業務體系的運營報酬）

三層報酬均從客戶的設定費與月費中按比例分配，總和等於公司預設的報酬成本，不存在「上層從下層獲利」或「多層獎金疊加」的機制。
```

### 修改樣板 B：「招募」敘述

**改前**：
```
經銷商：完成培訓 + 能獨立輔導客戶，可招募並培訓業務人員
```

**改後**：
```
經銷商：經認證的資深業務執行官，有權聘用並培訓業務人員。所有直屬員工均簽訂 MUSEON 正式勞動合約，受公司薪酬制度管轄，非獨立經銷商或代理商。
```
