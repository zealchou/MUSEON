系統範圍（MVP 第一階段）

包含範圍

  • 帳號系統：註冊、登入、身分驗證（JWT）
  • Billing Profile：使用者付款資料管理
  • License（序號）：平台建立、狀態管理、用戶查詢
  • Subscription：訂閱週期、狀態機（ACTIVE / GRACE / SUSPENDED）
  • Payment：人工收款紀錄、確認、開票基礎
  • Device Pairing：一次性配對碼、設備綁定、Token 發行
  • Entitlement：短效授權、Ed25519 簽章回應
  • Pre-Job Check：接新任務前授權檢查
  • Audit Log：關鍵操作完整稽核（16 種 action）
  • Alert Center：風控告警、事件通知
  • Notification：SendGrid email 整合、Bull queue

不包含範圍

  • 自動金流串接（綠界 / 藍新 / Stripe — Phase 15）
  • 退款流程（Phase 15）
  • 推薦分潤（Phase 16）
  • 企業級 IAM / SSO
  • 複雜報表與 BI 儀表板
  • 深度反逆向 / 反 Hook

技術棧

後端（Backend）

  • NestJS — 模組化架構
  • TypeORM — ORM 與 Migration
  • PostgreSQL — 關聯式資料庫
  • Bull Queue — 背景任務隊列
  • SendGrid — Email 服務
  • Ed25519 — 數位簽章
  • JWT — 身分驗證

前端（Frontend）

  • Vue 3 — Progressive Framework
  • Vite — 快速打包工具
  • shadcn-vue — 無頭 UI 元件
  • Tailwind CSS — 工具型 CSS
  • Pinia — 狀態管理
  • vue-i18n — 多語言

測試與部署

  • Cucumber — BDD 測試
  • Jest — 單元測試
  • Supertest — API 測試
  • PM2 — 流程管理
  • Docker — 容器化（預留）
