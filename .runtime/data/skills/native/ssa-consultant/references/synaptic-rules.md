## 染色體速查表

每條染色體的三軸摘要：D（洞見）、S（算子/路徑）、P（監控指標）。

### 系統建構群

**SSA01 系統主權與脫鉤**
- D：主權在於老闆與執行層的脫鉤，讓系統自運轉
- S：盤點重複性工作 → 轉譯 SOP → 委派 → 監控異常
- P：人治依賴度。高度依賴單一自然人 = 主權喪失

**SSA03 系統穩態與三腳支撐**
- D：系統包含多個子系統，缺一不可
- S：產品漏斗 × 社群漏斗 × 行銷漏斗，至少三種獲客管道
- P：子系統失靈偵測。某環節卡住則觸發弱點分析

**SSA09 自我治理與合規守則**
- D：法治勝於人治，規矩一致性是系統權威來源
- S：定義紅線 → 零容忍執行 → 反饋修正
- P：內部熵值。人為例外過多 = 治理失敗

**SSA12 演化迭代與弱點分析**
- D：持續改進是系統之魂，挑戰高目標逼出弱點
- S：朱蘭品質管理：分析弱點 → 針對優化 → 提升整體
- P：演化停滯。相似問題重複發生 = 需深度優化

### 顧問銷售群

**SSA02 顧問式反射覺察**
- D：對話是改寫身份的修行，鏡射客戶真實狀態
- S：建立關係 → 尋找主因 → 深入了解 → 找到痛點。追問擊穿 FP 防衛
- P：偽裝需求偵測。表面回答 → 強制共情程序

**SSA17 關係動力與商務關係盤**
- D：萬物皆關係，從五運了解對方狀態並尋找共鳴
- S：分析權利/責任/利益，建立信任與親和力
- P：信任溫度。防禦心理出現 → 停止邏輯推動，切換共情

**SSA19 意圖聚焦與目標落差**
- D：明確目標與現況的落差是行動唯一引擎
- S：期望目標 - 當前現況 = 真實落差。讓客戶看見不行動的代價
- P：決心熵值。落差感不足 → 重新痛點挖掘

**SSA22 感召力與英雄故事**
- D：分享經歷建立親和力，引發追隨動機
- S：分享脆弱 → 分享選擇 → 分享代價 → 展現力量
- P：共鳴度。故事無法引發共鳴 → 調整敘事權重

### 品牌經營群

**SSA05 真實口碑與石卵驗證**
- D：真實大於包裝，優質產品是口碑的底牌
- S：獲取真實見證 → 分類（財富/關係/事業）→ 拉回前端。產品是卵，禁止宣稱為石
- P：廣告/真實落差。體感滿意度下降 → 停止投放，回歸產品力

**SSA08 品牌人格與五感識別**
- D：品牌是拒絕的累積，在視覺/行為/理念保持一致性
- S：在五感節點植入品牌基因，確保不在場也被記得
- P：風格漂移。視覺或語氣不符人格 → 執行 VIS/BIS/IIS 校準

**SSA25 傳承閉環與教練培育**
- D：透過整合確保知識轉化為肌肉記憶
- S：我是誰 → 信仰核心 → 觀察洞察 → 成功個案
- P：傳承失真。輸出無法體現品牌立場 → 重新對焦

### 營收成長群

**SSA04 價值分配與能量流**
- D：分潤才有動力，利潤是回流至體驗/員工/夥伴的能量
- S：漲價 → 提高毛利 → 分配給鐵粉/經銷夥伴 → 激發推薦
- P：動力枯竭。夥伴銷售意願下降 → 檢查毛利分配

**SSA10 邏輯重組與價值疊加**
- D：透過產品組合與多重價值疊加，讓價格感模糊、價值感爆棚
- S：主產品 × 附加服務 × 情感價值
- P：市場競爭係數。陷入價格戰 → 啟動邏輯重組

**SSA14 鐵粉裂變與臨界質量**
- D：鐵粉不僅自己買還會帶人來，是營收自動化的核心
- S：建立粉絲社群 → 提供獨特價值 → 設計推薦獎勵
- P：鐵粉占比。陌生客比例 > 80% = 系統不穩

**SSA23 奇正相生與兵家詭道**
- D：免費模式吸引新客（奇），優質服務留住老客（正）
- S：免費試用/服務（奇）→ 導引至核心高價產品（正）
- P：轉換斷層。奇兵引人但正兵留不住 → 流程優化

### 資源槓桿群

**SSA07 資產累積與見證庫**
- D：可證明的價值是未來談判的最高籌碼
- S：整理客戶資料 → 分類消費帶 → 精準再行銷。人流轉現金流
- P：資產荒廢度。數據未標籤化 → 累積力喪失

**SSA11 情境賽道與橫向適配**
- D：同情境不同賽道，尋找不具競爭關係的夥伴
- S：TA 情境掃描 → 尋找互補產品 → 流量互換
- P：合作摩擦力。產生競爭衝突 → 賽道微調

**SSA13 外部資源槓桿路由**
- D：用最低成本透過資源交換創造最高價值
- S：識別對手最渴望的資源 → 用剩餘資源去換。建立商務資源庫
- P：槓桿平衡。確保交換符合 Dharma 價值觀

**SSA24 非對稱槓桿與 X 模型**
- D：落差是唯一的創造力，尋找不必解決困難也能達成目標的路徑
- S：識別核心卡點 → 注入外部資源/槓桿 → 實現 1:100 增長
- P：投入產出比。努力與結果比 < 1:10 → 重尋非對稱支點

### 全人演化群

**SSA15 突觸強化與 80/20 策略**
- D：核心 20% VIP 是公司的突觸，給予極端優先能量補給
- S：識別前 20% 高價值客戶 → 客製化方案 → 深度情感連結
- P：資源誤配。大量精力花在低價值客戶 → 權重修正

**SSA18 系統超越與全人進化**
- D：系統不只是商業更是生命，實現全人動態平衡
- S：八方位體檢（天/風/水/山/地/雷/火/澤）
- P：全人卡點。事業成功但身體卡住 = 演化失衡

**SSA20 戰略決策與破局思維**
- D：想變強的起點是害怕，識別恐懼背後的真實動機
- S：挑戰現有思維框架 → 尋找新商業模式 → 質變目標
- P：思維僵化。在庫存/價格等低維打轉 → 啟動大腦重組

**SSA27 系統呼吸與 PCT 閉環**
- D：讓系統自動回正，維持一張一弛的節律
- S：每週檢視 → 反饋回路 → 自動回正
- P：過熱熵。連續運轉無呼吸 → 強制啟動晨間呼吸+願景設定

### 戰略整合群

**SSA06 結構化商模 12 力**
- D：12 力如人體經絡，是秩序基礎，透過 Canvas 全域診斷
- S：12 力盤點 → 識別最弱環節 → 飽和優化。奧卡姆剃刀剪除低效
- P：結構一致性。團隊行為偏離 → 商模雷達圖重新對焦

**SSA16 行動閉環與以終為始**
- D：從顧客最終口碑分享倒推回最初吸引點
- S：口碑預設 → 轉換設計 → 單次消費提升 → 重複消費設計
- P：流程斷層。客戶在某環節失聯 → 體驗路徑潤滑

**SSA21 趨勢預測與市場識機**
- D：建立售後服務系統捕捉客戶需求變化作為預測依據
- S：分析回購行為 → 預測下一波市場需求點
- P：感知延遲。市場變化三個月後才反應 = 感知鈍化

**SSA26 全域索引與三盤定位**
- D：分析問題出在個人、關係還是事業盤
- S：判定問題根源盤 → 跨盤調用資源 → 執行整合解方
- P：定位偏差。在事業盤處理個人盤問題 → 邏輯導航修正

## 突觸聯動規則

常見情境的染色體自動聯動組合：

| 情境 | 聯動染色體組 |
|---|---|
| 客戶抗拒成交 | SSA02（訪談）+ SSA17（關係溫度）+ SSA19（落差） |
| 老闆焦慮過勞 | SSA01（脫鉤）+ SSA18（八方位）+ SSA27（呼吸） |
| 營收面臨瓶頸 | SSA10（組合）+ SSA12（弱點優化）+ SSA24（X模型） |
| 品牌形象模糊 | SSA08（識別）+ SSA22（故事）+ SSA25（結晶） |
| 客戶說太貴了 | SSA10（價值疊加）+ SSA04（價值分配）+ SSA23（奇正） |
| 獲客管道單一 | SSA03（三腳支撐）+ SSA11（橫向適配）+ SSA14（鐵粉） |
| 團隊執行力弱 | SSA09（合規）+ SSA01（脫鉤）+ SSA04（分潤動力） |
| 客戶轉介紹不足 | SSA14（鐵粉裂變）+ SSA05（口碑）+ SSA16（閉環） |
| 老闆不敢賣 | SSA20（破局）+ SSA22（英雄故事）+ SSA25（結晶） |
| 系統感知鈍化 | SSA21（趨勢）+ SSA07（資產累積）+ SSA12（弱點） |
| 全人能量失衡 | SSA18（八方位）+ SSA27（PCT）+ SSA15（80/20） |

## 跨外掛聯動對照

當 SSA 染色體指向其他外掛的能力時：

| SSA 染色體 | 聯動外掛 | 聯動原因 |
|---|---|---|
| SSA06（12力） | business-12 | 直接對應商模十二力框架，可深度展開 12 力診斷 |
| SSA24（X模型） | xmodel | 直接對應 X-Model 非對稱槓桿推演 |
| SSA13（外部槓桿） | xmodel | 對應 8 槓桿維度掃描與交換模型 |
| SSA10（價值疊加） | business-12 | 對應 BF11 談判力、BF01 產品力的深度診斷 |
| SSA14（鐵粉裂變） | business-12 | 對應 BF09 社群力、BF04 共享力的深度診斷 |
| SSA08（品牌識別） | business-12 | 對應 BF07 品牌力、BF10 視覺力的深度診斷 |
